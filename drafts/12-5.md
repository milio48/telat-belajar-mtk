# Bab 5: Eksponen dan Logaritma

## 1. Akar Konsep
Fungsi eksponen menggambarkan pertumbuhan atau peluruhan yang berlipat ganda secara berulang. Fungsi logaritma adalah kebalikannya — menentukan berapa kali pelipatan diperlukan untuk mencapai suatu nilai tertentu. Fungsi eksponen ibarat pertumbuhan populasi bakteri yang membelah diri setiap jam: jumlahnya berlipat sangat cepat. Fungsi logaritma menjawab pertanyaan "berapa jam yang dibutuhkan agar bakteri mencapai jumlah tertentu?"

> **Catatan:** Pertidaksamaan eksponen dan logaritma memerlukan perhatian khusus pada **basis**[^1]. Arah tanda ketidaksamaan berubah jika basisnya berada di antara 0 dan 1.

## 3. Mekanika Dasar
Untuk persamaan eksponen dengan basis sama:
Jika $a^{f(x)} = a^{g(x)}$ (dengan syarat $a > 0$ dan $a \ne 1$), maka:
$f(x) = g(x)$

Untuk persamaan logaritma:
$^{a}\log(f(x)) = {}^{a}\log(g(x))$ menghasilkan $f(x) = g(x)$, dengan syarat $f(x) > 0$ dan $g(x) > 0$.

Contoh: Selesaikan $2^{3x} = 2^{12}$.
Karena basis sama (basis 2), maka $3x = 12$, sehingga $x = 4$.

## 4. Logika Komputasi
```javascript
function hitungWaktuPelipatan(jumlahAwal, target) {
  // Model: N = jumlahAwal * 2^t
  // t = log2(target / jumlahAwal)
  if (jumlahAwal <= 0) return "Jumlah awal harus positif";

  const faktorLipat = target / jumlahAwal;
  const waktu = Math.log2(faktorLipat);

  return waktu;
}

const awal = 4;   // 4 koloni bakteri
const target = 64; // target 64 koloni

console.log(hitungWaktuPelipatan(awal, target)); // Output: 4
```

## 6. Uji Ketangkasan

**Soal 1:**
Carilah nilai $x$ dari persamaan $5^{(2x - 1)} = 125$.

**Penyelesaian:**
1. Ubah ruas kanan menjadi basis 5: $125 = 5^3$.
2. Persamaan menjadi $5^{(2x - 1)} = 5^3$.
3. Samakan pangkat: $2x - 1 = 3$, maka $2x = 4$, sehingga $x = 2$.

---

**Soal 2:**
Sebuah populasi virus berjumlah 500 sel. Populasi berlipat tiga setiap 2 jam. Berapa jumlah virus setelah 6 jam?

**Penyelesaian:**
1. Formulasikan model: $N = 500 \times 3^{t/2}$, di mana $t$ = waktu dalam jam.
2. Substitusi $t = 6$: $N = 500 \times 3^{6/2} = 500 \times 3^3$.
3. Hitung: $N = 500 \times 27 = 13.500$ sel virus.

---

[^1]: **Basis (logaritma/eksponen):** Bilangan pokok yang dipangkatkan. Dalam $2^3 = 8$, angka 2 adalah basis. Dalam $^{2}\log 8 = 3$, angka 2 juga adalah basis. Basis harus positif dan tidak sama dengan 1.
