# Bab 2: Pengukuran Lanjutan

## 1. Akar Konsep (Definisi & Deskripsi)
Kecepatan ("Speed") adalah rasio hibrida gabungan antara "Jarak yang dilalui" bertarung melawan "Durasi Waktu tempuhnya". Mobil dianggap cepat bila jarak yang sangat jauh ditempuh dengan melahap waktu yang sebentar saja (sedikit). 

## 2. Pijakan Bumi (Contoh Dunia Nyata)
- **Aturan Lalu Lintas Polisi / Joki Drag**: Di jalan tol tertera batas max Kecepatan: 100 Km/Jam. Artinya secara matematis mobil itu pantang memakan aspal sejauh 100 kilometer dalam waktu ngebut di bawah durasi 1 jam putaran jarum.
- **Astronomi**: Mengukur jarak bintang ke bumi bukan lagi pakai meteran tanah, tapi unit turunan yang dinamai "Tahun Cahaya" (Laju cahaya menembus angkasa 1 tahun kecepatannya).

## 3. Mekanika Dasar (Praktek Perhitungan)
**Rumus Segitiga (Jokowi = Jarak / Kecepatan x Waktu):**
- $\text{Kecepatan} = \frac{\text{Jarak}}{\text{Waktu}}$
- $\text{Jarak} = \text{Kecepatan} \times \text{Waktu}$
- $\text{Waktu} = \frac{\text{Jarak}}{\text{Kecepatan}}$

*Contoh*: 
Jarak kota A-B = 120 km. 
Motor Ayah melaju Konstan 40 km/jam.
Waktu tiba Ayah = $120 \div 40 = 3$ jam di atas sadel motor.

## 4. Logika Komputasi (Implementasi Kode)
Fitur kalkulasi GPS layaknya Google Maps menggunakan parameter macet/kecepatan rata2 ini loh! 
```javascript
function hitungEstimasiTiba(jarakTotalKm, kecepatanRataMobil) {
  if (kecepatanRataMobil <= 0) return "Kendaraan Diam";
  
  let prediksiJam = jarakTotalKm / kecepatanRataMobil;
  return `Anda akan sapai di rute tujuan dalam ${prediksiJam} Jam lagi.`;
}

console.log(hitungEstimasiTiba(120, 40)); 
// Output: "Anda akan sapai di rute tujuan dalam 3 Jam lagi."
```

## 5. Navigasi Global (Glosarium Inggris)
- Kecepatan / Kelajuan: *Speed / Velocity*
- Jarak Tempuh: *Distance*
- Rata-rata: *Average*

## 6. Uji Ketangkasan (Soal & Penyelesaian)
**Soal 1:**
Kereta Api express berangkat menempuh kota sejauh 250 Km. Dia tiba di stasiun akhir tepat 5 jam durasi perjalanan konstan tanpa jeda perhentian. Secara rata-rata hitungan Murni, berapa kecepatan rel keretanya per-Jam?
**Penyelesaian 1:**
Rumus: $\text{Kecepatan} = \frac{\text{Jarak}}{\text{Waktu}}$.
$Kecepatan = 250 \div 5$
$= 50$ km/jam.
Kecepatannya rata adalah 50 Kilometer Per Jam konstan.
