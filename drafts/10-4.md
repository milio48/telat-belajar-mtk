# Bab 4: Pertidaksamaan

## 1. Akar Konsep
Pertidaksamaan adalah pernyataan matematis yang membandingkan dua nilai tidak sama melalui lambang kurang dari ($<$), lebih dari ($>$), atau kombinasinya bersama tanda sama dengan. Pertidaksamaan ibarat batas kecepatan pada jalan tol (misal minimum 60 km/jam, dan maksimal 100 km/jam); kita diizinkan melaju di ragam angka kecepatan yang berbeda asalkan berada dalam zona irisan rentang batas aturan tersebut.

## 2. Pijakan Bumi
- **Insinyur Mesin:** Menentukan spesifikasi suhu operasional komponen mesin. Komponen akan meleleh jika suhu melampaui $150^\circ\text{C}$ ($t > 150$).
- **Logistik Kurir:** Mencari batas berat paket yang bisa dibawa oleh satu truk ringan. Total kardus ditambah berat supir tidak boleh melewati 2000 kilogram ($x + y \le 2000$).

## 3. Mekanika Dasar
Pertidaksamaan linier: $ax + b < c$.
Sifat dasar: Jika dikali/dibagi bilangan negatif, tanda berbalik lambang.

Contoh: Selesaikan $-2x + 4 \ge 10$
Kurangi kedua sisi dengan 4: $-2x \ge 6$
Bagi dengan bilangan negatif -2 (tanda berbalik): $x \le -3$

## 4. Logika Komputasi
```javascript
function ujiPertidaksamaanRentang(batasBawah, batasAtas, nilaiUji) {
  // Mengecek apakah sebuah nilai masuk dalam interval yang ditetapkan
  if (nilaiUji >= batasBawah && nilaiUji <= batasAtas) {
    return true; // Valid berada di rentang yang diizinkan
  } else {
    return false; // Berada di luar rentang (pelanggaran margin)
  }
}

const suhuMesin = 160;
const statusMesinAman = ujiPertidaksamaanRentang(60, 150, suhuMesin);
console.log(statusMesinAman); // Output: false
```

## 6. Uji Ketangkasan
**Soal:**
Untuk lulus seleksi beasiswa, rata-rata nilai matematika dan fisika seorang siswa tidak boleh kurang dari 85. Apabila nilai fisikanya adalah 82, berapakah nilai minimal matematika yang harus ia dapatkan?

**Penyelesaian:**
1. Formulasikan model dari soal tersebut: $(m + 82) / 2 \ge 85$.
2. Kalikan persamaan dengan konstanta pembuat rata-rata: $m + 82 \ge 170$.
3. Kurangi kedua sisi dengan 82: $m \ge 88$. Nilai mutlak minimal untuk matematika adalah 88.
