# Bab 7: Limit Fungsi

## 1. Akar Konsep
Limit fungsi adalah nilai yang didekati oleh keluaran suatu fungsi ketika masukan (variabel) mendekati suatu nilai tertentu. Limit tidak selalu sama dengan nilai fungsi di titik tersebut — yang penting adalah perilaku fungsi **saat mendekati** titik itu. Limit ibarat berjalan menuju tepi tebing: meskipun kamu tidak boleh melangkah ke ujungnya, kamu bisa menaksir kedalamannya dengan berjalan sedekat mungkin.

> **Catatan:** Bentuk tak tentu[^1] seperti $\frac{0}{0}$ memerlukan penyederhanaan aljabar (faktorisasi, perkalian sekawan, dll.) sebelum limit bisa dihitung.

## 2. Pijakan Bumi
- **Teknisi Otomotif:** Mengukur kecepatan sesaat kendaraan pada titik waktu tertentu sebagai limit dari kecepatan rata-rata dalam interval waktu yang semakin kecil.
- **Ahli Keuangan:** Menghitung bunga kontinu (compounding) sebagai limit dari bunga majemuk saat frekuensi penggandaan mendekati tak hingga.

## 3. Mekanika Dasar
Notasi limit: $\lim_{x \to c} f(x) = L$

Contoh bentuk tak tentu: $\lim_{x \to 2} \frac{x^2 - 4}{x - 2}$

1. Substitusi $x = 2$ menghasilkan $\frac{0}{0}$ (bentuk tak tentu).
2. Faktorkan pembilang: $\frac{(x - 2)(x + 2)}{(x - 2)}$.
3. Sederhanakan: $\lim_{x \to 2} (x + 2) = 2 + 2 = 4$.

## 4. Logika Komputasi
```javascript
function estimasiLimit(titikTarget) {
  // Simulasi limit f(x) = (x^2 - 4) / (x - 2) mendekati titik target
  // Setelah difaktorkan: f(x) = x + 2
  const hasilLimit = titikTarget + 2;
  return hasilLimit;
}

console.log(estimasiLimit(2)); // Output: 4
```

## 6. Uji Ketangkasan

**Soal 1:**
Hitunglah $\lim_{x \to 3} \frac{x^2 - 9}{x - 3}$.

**Penyelesaian:**
1. Substitusi $x = 3$ menghasilkan $\frac{0}{0}$ (bentuk tak tentu).
2. Faktorkan pembilang: $x^2 - 9 = (x - 3)(x + 3)$.
3. Sederhanakan: $\frac{(x-3)(x+3)}{(x-3)} = x + 3$. Maka $\lim_{x \to 3} (x + 3) = 6$.

---

**Soal 2:**
Hitunglah $\lim_{x \to 0} \frac{x^2 + 5x}{x}$.

**Penyelesaian:**
1. Substitusi $x = 0$ menghasilkan $\frac{0}{0}$ (bentuk tak tentu).
2. Faktorkan pembilang: $\frac{x(x + 5)}{x}$.
3. Sederhanakan: $\lim_{x \to 0} (x + 5) = 0 + 5 = 5$.

---

[^1]: **Bentuk Tak Tentu:** Ekspresi matematika yang tidak bisa dievaluasi langsung, seperti $\frac{0}{0}$, $\frac{\infty}{\infty}$, atau $0 \times \infty$. Penyelesaiannya memerlukan manipulasi aljabar atau aturan L'Hôpital.
