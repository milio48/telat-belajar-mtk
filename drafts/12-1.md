# Bab 1: Integral

## 1. Akar Konsep
Integral adalah konsep kalkulus yang bertindak sebagai manipulasi fungsional kebalikan atau invers mutlak dari pengerjaan modifikasi turunan kurva diferensial, sekaligus sebagai metode fundamental untuk merangkum total akumulasi perhitungan luasan porsi kumulatif di bawah irisan margin grafik fungsional. Integral ibarat menghitung total volume jumlah curah tampungan tumpahan hujan di dalam silinder tabung air. Turunan (diferensial) adalah nilai margin kecepatan menetesnya air per detik ke dalam silinder, sementara konsep integral adalah menjumlahkan seluruh akumulasi setiap letak tetes air tersebut dari rentang waktu nol sampai hujan berhenti murni demi menembak tebakan komposit volume tampungan fungsional mutlaknya di dalam galon uji ukur.

## 2. Pijakan Bumi
- **Teknisi Bendungan Raksasa:** Menghitung total porsi akumulasi komposit mutlak dari kalkulasi debit volume tampungan luapan air bertekanan bahaya dalam rentang margin durasi observasi jam bendungan agar tak merusak konstruksi batas aman tembok.
- **Ahli Pencitraan Medis (MRI Scan):** Mengintegralkan perhitungan susunan tumpukan komposit gambar parameter iris tipis rute rekaman pindai margin dua dimensi fungsional anatomi lapis kepala untuk merekonstruksi organ dalam wujud murni otak medis volume tiga dimensi yang presisi di komputer fiktif radiologi.

## 3. Mekanika Dasar
Konsep mutlak pemetaan integral tak tentu (anti turunan aljabar porsi rasio $ax^n$):
$\int ax^n dx = \frac{a}{n+1} x^{n+1} + C$

Contoh untuk resolusi fungsional komputasi mekanika murni $\int 4x^3 dx$:
1. Tambahkan proporsi skalar murni letak parameter indeks di pangkat dengan bilangan ukuran satu ($3 + 1 = 4$).
2. Bagi rasionya terhadap koefisien fungsional konstanta fiktif persamaan pada bagian rute awal ($4 / 4 = 1$).
3. Bentuk murni modifikasi nilai kalkulasi rasio akhir integral persamaannya berubah jadi susunan proporsi komposit mutlak: $x^4 + C$.

## 4. Logika Komputasi
```javascript
function fungsiLogikaIntegralDasar(koefisienDasar, rasioPangkatAwal) {
  // Hanya valid untuk bentuk fungsi umum integral ax^n
  const modifikasiProporsiPangkat = rasioPangkatAwal + 1;
  const konstantaHasilAkhir = koefisienDasar / modifikasiProporsiPangkat;
  
  return {
    konstanta: konstantaHasilAkhir,
    pangkat: modifikasiProporsiPangkat,
    teksOutput: `${konstantaHasilAkhir}x^${modifikasiProporsiPangkat} + C`
  };
}

// Menjalankan wujud komputasi dari integral 6x^2
const marginHasilKalkulasi = fungsiLogikaIntegralDasar(6, 2);
console.log(marginHasilKalkulasi.teksOutput); // Output: "2x^3 + C"
```

## 6. Uji Ketangkasan
**Soal:**
Kalkulasikanlah besaran mutlak hasil akhir fungsional pengerjaan modifikasi untuk persamaan fiktif aljabar komposit rasio limit integral berikut secara seksama dengan kaidah anti turunan dasar murni mekanika rasio aljabar susun polinomial: $\int (3x^2 - 4x + 5) dx$.

**Penyelesaian:**
1. Bedah mutlak kalkulasi porsi rute fungsional ini dan kerjakan secara per ruas suku letak komposisi aljabar (integralkan pecahan komposisi fiktif mutlak masing-masing rasionya secara bertahap miring).
2. Terapkan fungsional rute kalkulus penambahan derajat mutlak koefisien persamaan ke bilangan pangkat fungsi letak rasionya lalu fungsikan letak hitungan modifikasi komposit ini sebagai rute rasio margin pembagi utama koefisien fiktif:
 - Bagian pertama: $(3 / (2+1)) \cdot x^{2+1} = 1x^3 = x^3$.
 - Bagian kedua: $(-4 / (1+1)) \cdot x^{1+1} = -2x^2$.
 - Bagian rasio konstan bebas murni komposit ketiga: $5$ berubah letak memuat variabel hitung $x$ letak porsi aljabar komposit jadi persamaan fiktif nilai rentang rasio murni $5x$.
3. Gabungkan keseluruhan porsi struktur hitung turunannya untuk menghasilkan rute letak turunan fungsional anti letak balikan diferensial rasionya lalu bubuhkan wujud parameter rasio tambahan porsi konstanta mutlak sembarang murni aljabar limitasi simbol C untuk menampung besaran hilang: Hasil akhir mutlak susunan aljabarnya menjadi bentuk porsi fungsional komposit wujud $x^3 - 2x^2 + 5x + C$.
