# Bab 5: Logika Matematika

## 1. Akar Konsep
Logika matematika adalah cabang matematika yang mengeksplorasi aturan dasar mengenai kebenaran atau kesalahan dari sebuah pernyataan komposit tanpa ambiguitas. Logika ibarat sirkuit pada saklar instalasi lampu. Ketika lampu dirangkai seri (seperti operasi konjungsi/DAN), semua saklar wajib ditutup agar lampu menyala; jika dirangkai paralel (operasi disjungsi/ATAU), lampu tetap hidup asalkan minimal ada satu saklar yang tersambung.

## 2. Pijakan Bumi
- **Programmer Ahli:** Merangkai "Gerbang Logika" pada pembuatan perangkat lunak dengan aturan if-else untuk mengeksekusi aksi khusus sesuai syarat yang diberikan pengguna (AND, OR, NOT).
- **Hukum & Pengacara:** Menggunakan teknik penarikan kesimpulan modus ponens dan tollens dari pasal hukum untuk mendeduksi perbuatan ilegal dengan dalil yang pasti dan tak terbantahkan.

## 3. Mekanika Dasar
Konjungsi ($p \wedge q$) bernilai benar hanya jika $p$ dan $q$ benar.
Disjungsi ($p \vee q$) bernilai salah hanya jika $p$ dan $q$ salah.
Implikasi ($p \Rightarrow q$) bernilai salah hanya jika $p$ benar, tetapi $q$ salah.

Tabel kebenaran Implikasi:
$T \Rightarrow T = T$
$T \Rightarrow F = F$
$F \Rightarrow T = T$
$F \Rightarrow F = T$

## 4. Logika Komputasi
```javascript
function ujiLogikaSirkuit(skalarA, skalarB) {
  // Operasi logika DAN (AND) di JavaScript menggunakan &&
  const konjungsi = skalarA && skalarB;
  
  // Operasi logika ATAU (OR) menggunakan ||
  const disjungsi = skalarA || skalarB;
  
  return { statusAnd: konjungsi, statusOr: disjungsi };
}

const lampuA = true;
const lampuB = false;
const hasilSirkuit = ujiLogikaSirkuit(lampuA, lampuB);
console.log(hasilSirkuit); // Output: { statusAnd: false, statusOr: true }
```

## 5. Navigasi Global
| Istilah Indonesia | Istilah Inggris |
| --- | --- |
| Konjungsi (Dan) | Conjunction (AND) |
| Disjungsi (Atau) | Disjunction (OR) |
| Implikasi (Jika..Maka) | Implication (If..Then) |

## 6. Uji Ketangkasan
**Soal:**
Terdapat sebuah pernyataan implikasi dari pendaftaran bank: "Jika kamu karyawan tetap, maka kredit rumahmu akan disetujui". Ternyata kenyataannya Budi adalah karyawan tetap, namun kredit rumahnya tidak disetujui. Apakah pernyataan bank tersebut benar atau salah menurut logika implikasi?

**Penyelesaian:**
1. Tentukan predikat (p): "Kamu karyawan tetap" (nilai Budi = Benar / True).
2. Tentukan konklusi (q): "Kredit disetujui" (nilai Budi = Salah / False).
3. Evaluasi $True \Rightarrow False$, menurut tabel kebenaran implikasi, hasilnya adalah False. Jadi, janji awal dari bank bernilai cacat logika atau bohong secara utuh.
