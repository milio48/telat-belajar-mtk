# Bab 1: Bilangan Cacah dan Operasi Hitung

## 1. Akar Konsep (Definisi & Deskripsi)

Bayangkan angka sebagai tumpukan batu bata. Jika kamu hanya memiliki beberapa balok tunggal, kamu mudah menghitungnya dengan menyebut $"1, 2, 3"$. Namun, bayangkan kamu mempunyai tumpukan yang sangat besar, menghitungnya satu per satu tentu akan memakan banyak waktu.

Karena itulah kita menciptakan **Sistem Nilai Tempat**. Ini mirip dengan cara kasir minimarket mengatur kembalian di lacinya. Ketika sekumpulan $10$ satuan terkumpul, kita membungkusnya menjadi satu ikatan baru yang kita sebut "Puluhan". Ketika kita memiliki $10$ ikatan puluhan, kita memasukkannya ke dalam sebuah karung dan menyebutnya "Ratusan".

Setiap angka yang kita tulis posisinya sangat menentukan nilai. Angka $5$ yang berada di paling kanan mewakili $5$ keping satuan. Tetapi, jika angka $5$ itu digeser sedikit ke kiri (seperti $50$), ia kini menjadi $5$ gulung puluhan. Semakin angka bergeser ke ruas kiri, derajat "kebesaran" tumpukan tersebut bernilai sepuluh kali lipat.

## 2. Pijakan Bumi (Contoh Dunia Nyata)

Nilai tempat ini digunakan agar peradaban manusia tidak kewalahan menghitung:
* **Penghitung Uang Kertas:** Di bank, pegawai di sana akan merapikan $10$ lembar sepuluh ribuan menjadi satu bendel senilai seratus ribu, lalu menyusun tumpukan tersebut dalam brankas.
* **Gudang Toko Kelontong:** Alih-alih menghitung satu per satu permen, mereka memesan $1$ kardus (ratusan), mendapatkan bonus $2$ stoples kecil (puluhan), lalu membuka eceran $3$ biji di atas meja kasir. (Mewakili nilai $123$).

## 3. Mekanika Dasar (Praktek Perhitungan)

Berlatih membedah struktur angka:
Angka **$254$** dibaca "Dua Ratus Lima Puluh Empat", yang berarti:
* $2$ mewakili kelompok **Ratusan** ($200$ buah)
* $5$ mewakili kelompok **Puluhan** ($50$ buah)
* $4$ mewakili kelompok **Satuan** ($4$ buah tunggal)

**Membandingkan Bilangan Besar:**
Jika kita membandingkan $321$ dengan $340$, mana yang lebih besar?
1. Lihat dari kelompok yang terbesar (ratusan). Keduanya sama-sama $300$. Jadi ini seri, lanjutkan pandangan ke pilar sebelahnya.
2. Bandingkan blok puluhannya. Angka $4$ lebih besar dari angka $2$ (karena jumlah isinya $40$ lebih banyak dari $20$).
3. Sehingga, $340 > 321$.

**Mengurutkan:**
Jika ada deret tumpukan, kita selalu urutkan dari mencari nilai paling tinggi di blok letaknya dari barisan kiri ke kanan.

## 4. Logika Komputasi (Implementasi Kode)

Mesin digital membaca bilangan dengan memisahkan nilai tempat, persis seperti yang manusia kalkulasikan, dalam menentukan urutan antrian atau sistem daftar harga.

```javascript
// Representasi Angka dalam sistem desimal
let jumlahPelanggan = 682;

// Jika kita asumsikan membuat kode pemecah digit
// (Contoh untuk melihat logika di balik strukturnya)
let nilaiRatusan = Math.floor(jumlahPelanggan / 100) * 100; // Dapat angka 600
let sisaDariRatusan = jumlahPelanggan % 100; // Menyisakan 82
let nilaiPuluhan = Math.floor(sisaDariRatusan / 10) * 10; // Dapat angka 80
let nilaiSatuan = sisaDariRatusan % 10; // Dapat angka 2

console.log("Rincian Nilai Tempat:");
console.log("Ratusan:", nilaiRatusan);
console.log("Puluhan:", nilaiPuluhan);
console.log("Satuan:", nilaiSatuan);

// Program Membandingkan Angka
let pengunjungSenin = 150;
let pengunjungSelasa = 135;

if (pengunjungSenin > pengunjungSelasa) {
  console.log("Senin lebih ramai, kita butuh lebih banyak kursi.");
}
```

## 6. Uji Ketangkasan (Soal & Penyelesaian)

**Soal 1:**
Dalam pertandingan mengumpulkan kaleng bekas, Kelompok Elang berhasil mengumpulkan $427$ kaleng, sedangkan Kelompok Harimau punya $472$ kaleng. Kelompok mana yang menang, dan tolong bedah dari aspek nilai tempat!

**Penyelesaian:**
1. Bedah Kelompok Elang ($427$): Ratusan = $4$, Puluhan = $2$, Satuan = $7$.
2. Bedah Kelompok Harimau ($472$): Ratusan = $4$, Puluhan = $7$, Satuan = $2$.
3. Membandingkan: Mulai dari Ratusan, nilainya seri ($4 = 4$). Lanjut ke Puluhan, ternyata angka $7$ lebih besar daripada $2$. 
4. Kesimpulan: **Kelompok Harimau** menang karena memegang lebih banyak ikatan puluhan kaleng.

**Soal 2:**
Ibu Budi ingin menyusun nomor seri di lemari arsipnya. Ia memiliki nomor: $105, 125, 115$. Urutkan dari laci yang paling atas (nomor terkecil), dengan alur berpikirmu!

**Penyelesaian:**
1. Semua bilangan memiliki nilai seratus ($100$) di awal, yang artinya seimbang.
2. Kita periksa angka tengahnya (Puluhan). Pada $105$ puluhannya adalah $0$. Pada $115$, puluhannya $1$. Pada $125$, puluhannya $2$.
3. Susun berdasarkan puluhannya, $0$ paling ringan, diikuti $1$, lalu $2$. 
4. Hasil urutan adalah: **$105, 115, 125$**.
