# Bab 1: Bilangan Cacah dan Operasi Hitung

## 1. Akar Konsep (Definisi & Deskripsi)
Misalkan kamu mengurus ribuan kelereng. Menghitungnya satu per satu tentu akan membingungkan. Sistem Nilai Tempat adalah cara kita mengelompokkan barang ke dalam wadah yang lebih besar: satuan (kelereng lepas), puluhan (1 kantong kecil isi 10), dan ratusan (1 kardus isi 10 kantong kecil). Posisi angka menentukan "ukuran wadah"-nya.

## 2. Pijakan Bumi (Contoh Dunia Nyata)
- **Teler Bank**: Mengikat lembaran uang dalam bendel puluhan atau memasukannya dalam kotak ratusan ribu untuk memudahkan perhitungan saldo nasabah.
- **Petugas Inventaris Gudang**: Menyimpan barang dalam kemasan dus (ratusan unit) dan kemasan kotak kecil (puluhan unit) agar tidak repot menghitung eceran.

## 3. Mekanika Dasar (Praktek Perhitungan)
**Nilai Tempat:**
Pada angka $345$:
- Angka $3$ menempati **ratusan** (bernilai 300)
- Angka $4$ menempati **puluhan** (bernilai 40)
- Angka $5$ menempati **satuan** (bernilai 5)

**Membandingkan Bilangan:**
Bandingkan ratusannya dulu, lalu puluhan, baru satuan.
$345$ lebih besar dari $321$ (karena puluhan $40 > 20$). Ditulis: $345 > 321$

**Penjumlahan Bersusun:**
  $123$
  $245$
  --- +
  $368$

## 4. Logika Komputasi (Implementasi Kode)
Dalam pemrograman, kita sering kali perlu memecah angka berdasarkan posisi nilai tempatnya untuk dianalisis.

```javascript
function uraiNilaiTempat(angka) {
  const ratusan = Math.floor(angka / 100) * 100;
  const sisaRatusan = angka % 100;
  const puluhan = Math.floor(sisaRatusan / 10) * 10;
  const satuan = sisaRatusan % 10;
  
  return `${angka} = ${ratusan} + ${puluhan} + ${satuan}`;
}

console.log(uraiNilaiTempat(345)); // Output: "345 = 300 + 40 + 5"
```

## 5. Navigasi Global (Glosarium Inggris)
- Nilai Tempat: *Place Value*
- Ratusan: *Hundreds*
- Puluhan: *Tens*
- Satuan: *Ones / Units*
- Membandingkan: *Comparing*

## 6. Uji Ketangkasan (Soal & Penyelesaian)
**Soal 1:**
Adik punya kelereng sebanyak $253$ butir, dan kakak punya $235$ butir. Siapa yang punya kelereng lebih banyak?
**Penyelesaian 1:**
Bandingkan dari ratusan: sama-sama $2$.
Bandingkan puluhan: adik $5$, kakak $3$. Karena $5 > 3$, adik punya lebih banyak kelereng ($253 > 235$).
