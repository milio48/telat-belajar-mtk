# Bab 1: Bilangan Cacah dan Operasi Hitung

## 1. Akar Konsep
Nilai tempat adalah posisi suatu angka dalam bilangan yang menentukan nilainya. Ratusan, puluhan, dan satuan ibarat laci pada kabinet berkas yang mengelompokkan dokumen berdasarkan seberapa besar volumenya.

## 2. Pijakan Bumi
- Petugas Sensus: Mengelompokkan dan merekap jumlah warga dalam entitas keluarga, RT, dan skala desa.
- Kasir Bank: Menyusun kelompok uang kertas di laci kas berdasarkan nilai satuannya agar mempermudah perhitungan.

## 3. Mekanika Dasar
Nilai bilangan cacah dapat diuraikan berdasarkan posisi urutannya:
$Bilangan = (a \times 100) + (b \times 10) + (c \times 1)$
Contoh penjabaran bilangan 234:
$234 = 200 + 30 + 4$

## 4. Logika Komputasi
```javascript
function uraiNilaiTempat(nilaiBilangan) {
  let jumlahRatusan = Math.floor(nilaiBilangan / 100);
  let sisaBilangan = nilaiBilangan % 100;
  let jumlahPuluhan = Math.floor(sisaBilangan / 10);
  let jumlahSatuan = sisaBilangan % 10;
  
  return { jumlahRatusan, jumlahPuluhan, jumlahSatuan };
}
```

## 6. Uji Ketangkasan
**Soal:** Sebuah perpustakaan sekolah menerima 125 buku cerita dan 53 buku pelajaran. Berapa seluruh jumlah buku yang diterima?
**Penyelesaian:**
1. Urai susunan nilai tempat dari bilangan yang ada: 125 (100 + 20 + 5) dan 53 (50 + 3).
2. Lakukan operasi penjumlahan secara urut pada kelompok satuan puluhan (20 + 50 = 70) dan satuan paling belakang (5 + 3 = 8).
3. Satukan seluruh total perhitungan penjumlahannya: 100 + 70 + 8 = 178 buku.
