# Bab 1: Statistika

## 1. Akar Konsep
Statistika adalah ilmu yang mempelajari cara merencanakan, mengumpulkan, menganalisis, lalu menginterpretasikan data untuk mengambil keputusan. Statistika ibarat koki yang mencicipi sup. Koki tersebut tidak perlu meminum satu panci penuh sup untuk tahu rasanya pas atau tidak; ia cukup mengambil satu sendok kecil (sampel) yang diaduk rata untuk menyimpulkan rasa keseluruhan panci (populasi).

## 2. Pijakan Bumi
- **Data Analyst E-Commerce:** Mengumpulkan profil usia, lokasi, dan daya beli pengunjung web untuk memprediksi produk apa yang akan laris dijual bulan depan.
- **Ahli Demografi Negara:** Menghitung ukuran pemusatan berupa rata-rata (mean) pendapatan warga di sebuah kota untuk menentukan tingkat Upah Minimum Regional (UMR) yang layak.

## 3. Mekanika Dasar
Konsep dasar rata-rata (Mean): $\bar{x} = \frac{\sum x_i}{n}$
Di mana $\sum x_i$ adalah jumlah seluruh nilai data, dan $n$ adalah banyaknya data.

Contoh: Terdapat hasil nilai ujian 5 murid: 70, 80, 85, 90, 75.
Total nilai = $70 + 80 + 85 + 90 + 75 = 400$
Banyak data = 5
Rata-rata = $400 / 5 = 80$.

## 4. Logika Komputasi
```javascript
function hitungRataRata(kumpulanData) {
  // Mengecek apakah kumpulan data kosong
  if (kumpulanData.length === 0) return 0;
  
  // Menggunakan reduce untuk menjumlahkan seluruh elemen array
  const totalJumlah = kumpulanData.reduce((akumulator, nilai) => akumulator + nilai, 0);
  
  // Membagi total kemunculan dengan kuantitas array
  const nilaiRata = totalJumlah / kumpulanData.length;
  
  return nilaiRata;
}

const nilaiSiswa = [70, 80, 85, 90, 75];
const rataRataKelas = hitungRataRata(nilaiSiswa);
console.log(rataRataKelas); // Output: 80
```

## 6. Uji Ketangkasan
**Soal:**
Toko kue "Ceria" mencatat penjualan bolu panggang selama satu minggu penuh, dengan rekapitulasi data sebagai berikut: 15, 12, 18, 14, 20, 16, 10 loyang. Berapakah rata-rata bolu yang terjual per hari dalam satu minggu itu?

**Penyelesaian:**
1. Kumpulkan seluruh data penjualan bolu setiap harinya dan hitung jumlah total dalam sepekan. Total kue $= 15 + 12 + 18 + 14 + 20 + 16 + 10 = 105$.
2. Identifikasi jumlah penyebut data (satu minggu memiliki 7 hari pencatatan), maka $n = 7$.
3. Bagi total penjualan dengan penyebut data: $105 / 7 = 15$. Kesimpulannya, rata-rata harian adalah 15 loyang bolu terjual setiap harinya.
