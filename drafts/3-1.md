# Bab 1: Bilangan, Uang, dan Operasi Hitung

## 1. Akar Konsep (Definisi & Deskripsi)
Garis bilangan adalah "jalan panjang" tempat semua angka tinggal secara berurutan. Semakin ke kanan angkanya semakin besar. Seperti alamat rumah yang berurutan. Sedangkan uang adalah kesepakatan nilai, alat tukar yang diwakili oleh angka-angka besar seperti ribuan dan puluhan ribu.

## 2. Pijakan Bumi (Contoh Dunia Nyata)
- **Kurir Paket (Garis Bilangan)**: Harus mengurutkan alamat rumah (nomor 4, nomor 6, nomor 8) agar tidak bolak-balik.
- **Wirausahawan (Uang)**: Menghitung modal, laba, pecahan kembalian uang dari pembeli secara akurat supaya tidak merugi.

## 3. Mekanika Dasar (Praktek Perhitungan)
**Operasi Hitung (Sampai 3 Angka):**
Penjumlahan bersusun dengan teknik menyimpan ("carry"):
  $158$
  $294$
  --- +
  $452$  *(8+4=12, tulis 2 simpan 1. 1+5+9=15, tulis 5 simpan 1. 1+1+2=4)*

**Menghitung Uang:**
Kamu beli buku Rp3.000 dan pensil Rp1.500.
Total harga: $3000 + 1500 = 4500$.
Membayar dengan Rp5.000. Kembalian: $5000 - 4500 = 500$ (lima ratus rupiah).

## 4. Logika Komputasi (Implementasi Kode)
Mesin kasir di minimarket memproses perhitungan uang kembalian secara otomatis.

```javascript
// Mesin Kasir Sederhana
function hitungKembalian(totalBelanja, uangDibayar) {
  let kembalian = uangDibayar - totalBelanja;
  if(kembalian < 0) {
    return "Uang Anda kurang!";
  } else {
    return `Kembalian Anda: Rp${kembalian}`;
  }
}

console.log(hitungKembalian(4500, 5000)); // Output: "Kembalian Anda: Rp500"
```

## 5. Navigasi Global (Glosarium Inggris)
- Garis Bilangan: *Number Line*
- Penjumlahan Ratusan: *Three-digit Addition*
- Menyimpan (dalam penjumlahan): *Carry Over*
- Uang: *Money / Currency*
- Harga / Kembalian: *Price / Change*

## 6. Uji Ketangkasan (Soal & Penyelesaian)
**Soal 1:**
Jarak rumah Andi ke sekolah adalah 350 meter. Jarak sekolah ke perpustakaan adalah 175 meter. Berapa total jarak jika Andi berangkat dari rumah ke perpustakaan melewati sekolah?
**Penyelesaian 1:**
$350 + 175 = 525$. Jaraknya adalah 525 meter.

**Soal 2:**
Ibu membeli gula seharga Rp6.500. Ibu membayar dengan uang Rp10.000. Berapa uang kembalian ibu?
**Penyelesaian 2:**
$10000 - 6500 = 3500$. Uang kembalian yang diterima adalah Rp3.500.
