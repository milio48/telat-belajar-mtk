# Bab 1: Bentuk Pangkat, Akar, dan Logaritma

## 1. Akar Konsep
Eksponen (pangkat) adalah perkalian berulang dari suatu bilangan, akar adalah kebalikan dari pangkat yang mencari nilai asal suatu bilangan sebelum dipangkatkan, sedangkan logaritma adalah operasi matematika untuk menentukan besar pangkat yang diperlukan untuk mendapatkan suatu nilai tertentu. Eksponen ibarat melipat sebuah kertas berkali-kali untuk membuatnya tebal secara eksponensial. Akar ibarat menghitung berapa lembar ketebalan aslinya jika lipatannya dibuka. Logaritma ibarat mencari tahu berapa kali kita harus melipat kertas tersebut agar mencapai ketebalan tertentu.

## 2. Pijakan Bumi
- **Ilmu Astronomi:** Menggunakan notasi ilmiah (pangkat) untuk menuliskan jarak antar bintang yang sangat jauh secara ringkas.
- **Ahli Akuntansi:** Menggunakan logaritma dalam perhitungan bunga majemuk investasi jangka panjang untuk menentukan waktu pengembalian modal.

## 3. Mekanika Dasar
Sifat eksponen: $a^m \times a^n = a^{m+n}$
Sifat bentuk akar: $\sqrt{a \times b} = \sqrt{a} \times \sqrt{b}$
Sifat logaritma: $^a\log(b \times c) = ^a\log b + ^a\log c$

Contoh perhitungan dengan sifat logaritma:
$^2\log(8) + ^2\log(4) = ^2\log(8 \times 4) = ^2\log(32) = 5$

## 4. Logika Komputasi
```javascript
function hitungLogaritmaBasis2(nilai) {
  // Menghitung logaritma basis 2 menggunakan logaritma natural (Math.log2 atau Math.log(nilai)/Math.log(2))
  const hasilLogaritma = Math.log2(nilai);
  return hasilLogaritma;
}

const nilaiAwal = 32;
const pangkatBasis = hitungLogaritmaBasis2(nilaiAwal);
console.log(pangkatBasis); // Output: 5
```

## 6. Uji Ketangkasan
**Soal:**
Seorang ilmuwan meneliti populasi bakteri. Bakteri tersebut membelah diri menjadi dua setiap jam. Jika pada awalnya ada 100 bakteri, berapa jam yang dibutuhkan agar populasi bakteri menjadi 3200?

**Penyelesaian:**
1. Formulasikan masalah dalam bentuk pangkat: $100 \times 2^t = 3200$.
2. Sederhanakan persamaan untuk mendapatkan bentuk eksponensial murni: $2^t = 3200 / 100$, sehingga $2^t = 32$.
3. Gunakan konsep logaritma atau sifat pangkat untuk mencari $t$: $2^t = 2^5$, maka $t = 5$. Waktu yang dibutuhkan adalah 5 jam.
