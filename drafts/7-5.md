# Bab 5: Penyajian Data

## 1. Akar Konsep
Penyajian data adalah proses pengorganisasian serta penampilan informasi kuantitatif yang masih acak ke dalam sebuah wujud format yang lebih terstruktural. Metode penyajian ini dapat menggunakan sarana tabel rekapitulasi numerik maupun ragam variasi gambar grafis visual (diagram garis, diagram batang, hingga diagram lingkaran) sesuai keperluan jenis data mentahnya. Hal ini akan memudahkan pembaca guna membedah dan mengambil kesimpulan spesifik yang rasional. Penyajian ini ibarat buku cerita bergambar yang menyederhanakan data teks rumit menjadi ilustrasi terang.

## 2. Pijakan Bumi
* **Pembuat Prakiraan Cuaca:** Stasiun pengamat suhu harian menyajikan trend fluktuasi curah hujan per bulan melalui skema diagram batang yang mudah dibaca publik warganet.
* **Manajer Pasar Swalayan:** Seorang pengurus gerai melihat evaluasi presentase laku barang makanan ringan dengan membaca rasio potongan kue di diagram lingkaran.

## 3. Mekanika Dasar
Menentukan besar porsi derajat sudut pada visual diagram sudut lingkaran total berdasarkan perhitungan proporsi tertentu:
$$ \text{Sudut} = \frac{\text{Jumlah Data Bagian}}{\text{Total Data}} \times 360^{\circ} $$
Contoh jika 10 dari 40 siswa menyukai warna biru, besar derajatnya adalah:
$$ \text{Sudut} = \frac{10}{40} \times 360^{\circ} = 90^{\circ} $$

## 4. Logika Komputasi
```javascript
function hitungBesarDerajat(dataSebagian, dataTotal) {
  let persentasePorsi = dataSebagian / dataTotal;
  let besarDerajatSudut = persentasePorsi * 360;
  return besarDerajatSudut;
}

let penyukaBiru = 10;
let keseluruhanSiswa = 40;
let hasilDerajat = hitungBesarDerajat(penyukaBiru, keseluruhanSiswa);
console.log("Besaran proporsi juring: " + hasilDerajat + " derajat");
```

1. Lakukan pemisahan kolom pada setiap buah: masing-masing 5 apel, 8 jeruk, 7 pisang.
2. Buat skema rumus kolom jumlahan yakni gabungan keseluruhan objeknya: 5 + 8 + 7.
3. Teruskan operasi penjumlahannya hingga ketemu satu nilai konklusif akhir sebesar 20 buah hitungnya.
