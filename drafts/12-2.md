# Bab 2: Program Linear

## 1. Akar Konsep
Program linear adalah sebuah metode khusus aljabar komputasi pemetaan matriks model terapan logistik persamaan mutlak pencarian solusi hampiran yang menarget mutlak letak untuk memperoleh poin margin kombinasi nilai ukur paling fungsional minimum maupun ukuran margin porsi hasil puncak nilai maksimum aljabar (optimal) yang ditarik mutlak dari parameter kendala rute variabel komposit sistem letak batas pertidaksamaan linear fiktif. Program linear murni ibarat mekanisme fungsional manajer penyeleksi pelamar fiktif dalam perusahaan rute margin. Pihak perusahaan murni memiliki fungsi ukur nilai modal komposit yang minim, serta tuntutan batas ukur tenggat proyek fiktif komposit yang amat sempit margin pergerakannya, lalu alat aljabar fungsional manajer porsi wajib ini mencari rasio hitungan letak rute penugasan margin komposit tim dari pegawai kombinasi rasio efisien nilai mana yang mencetak margin fiktif laba fungsional produksi kerja dengan porsi nilai tertinggi maksimum.

## 2. Pijakan Bumi
- **Spesialis Perencana Transportasi Distribusi Logistik:** Memetakan skema model rute komposit porsi persamaan irisan limit fungsi distribusi rute jalan truk limit muatan kargo nilai gudang menuju aneka letak rute tujuan gerai rasionya dengan sasaran nilai komposit meminimalkan rute angka besaran tanggungan mutlak dana uang beli bensin kurir rute kurir.
- **Diet Nutrisien Gizi:** Merangkai logistik komposit mutlak kombinasi dari ragam campuran rute pangan diet fungsional rumah sakit mutlak rute irisan agar kebutuhan porsi nutrisi jenis vitamin dan nilai rasio kalori minimum fiktif dipenuhi, dengan menekan sebisa ukuran titik rasionya batas ukur biaya anggaran porsi beli belanja bulanan aljabar porsi belanjanya sampai nilai komposit minim mutlak batas titik minimum murni porsi anggarannya.

## 3. Mekanika Dasar
Pola metode penentuan nilai grafik aljabar tipe optimal maksmum ujung:
Fungsi komposit tujuan letak sasaran irisan keuntungan (Z): $Z = ax + by$
Sistem kendala fungsional rute batas rasio irisan kurva murni pertidaksamaan:
- $cx + dy \le m$ (rasio rute kendala material limit satu)
- $ex + fy \le n$ (rasio mutlak kendala kerja rute rute dua)
- $x \ge 0$, $y \ge 0$ (batas margin syarat posisi di rentang letak bilangan cacah positif)

Setelah membuat arsiran nilai rute irisan area margin di koordinat letak sumbu silang ganda, uji rute titik potong aljabar posisi komposit ke setiap sudut pinggir garis wilayah irisan tersebut untuk mendapatkan rute angka mana yang memunculkan nilai poin fungsional maksimum perhitungan rasio mutlak $Z$.

## 4. Logika Komputasi
```javascript
function fungsiHitungLabaMaksimalProgramLinear() {
  // Simulasi mencari perpotongan garis optimal dari himpunan irisan penyelesaian rute murni
  const susunanFiktifNilaiKombinasiX = [0, 4, 2]; // Contoh titik acak irisan
  const susunanTitikPersamaanKombinasiY = [5, 0, 3]; // Hasil nilai y yang berpadanan fungsional posisinya tersebut mutlak.
  
  let catatanRasioLabaPuncak = 0;
  
  // Mencari porsi laba dengan iterasi uji titik uji Z = 1000x + 1500y mutlak limit
  for (let rentangKalkulasiIndex = 0; rentangKalkulasiIndex < 3; rentangKalkulasiIndex++) {
    const angkaPerhitunganZ = (1000 * susunanFiktifNilaiKombinasiX[rentangKalkulasiIndex]) + (1500 * susunanTitikPersamaanKombinasiY[rentangKalkulasiIndex]);
    
    // Periksa status laba margin
    if (angkaPerhitunganZ > catatanRasioLabaPuncak) {
      catatanRasioLabaPuncak = angkaPerhitunganZ;
    }
  }
  
  return catatanRasioLabaPuncak;
}

const puncakHasilUjiPendapatan = fungsiHitungLabaMaksimalProgramLinear();
console.log(puncakHasilUjiPendapatan); // Output margin rasionya: 7500
```

## 6. Uji Ketangkasan
**Soal:**
Seorang pengusaha tata kelola perabotan ingin memproduksi rute logistik wujud komposit mebel kayu yang terdiri fungsional rasio dari bangku dan lemari mutlak letak irisan aljabarnya batas rute hitungnya murni rasionya per unit lemari. Profit fungsional wujud modifikasi dari unit kursi menembus rasio Rp 50.000 dan untuk profit produk item mutlak lemari sebesar fiktif nilai rentang Rp 80.000 (Fungsi tujuannya margin Z tersebut yaitu wujud $Z = 50.000x + 80.000y$). Bila hasil gambar kurir irisan himpunan arsiran nilai margin rasionya murni memberi titik uji sudut komposit kelajuan fungsional di garis titik perpotongan batas mutlak fungsional rute poin A(10, 0), B(8, 6), dan rute titik letak di posisi miring potong arsiran letaknya di koordinat rasio C(0, 12), carilah posisi batas rasio pendapatan omzet mutlak batas komposit puncaknya!

**Penyelesaian:**
1. Rangkum dan uji nilai rasionya tersebut secara masuk substitusi murni iteratif satu per satu porsi irisan uji titik kombinasi komposit.
2. Substitusikan kalkulasi titik potong murni irisan rute letak A(10, 0) ke dalam fungsi komposit fungsi margin Z tersebut: $(50.000 \times 10) + (80.000 \times 0) = \text{Rp } 500.000$.
3. Uji limit titik persamaan pinggir batas irisan mutlak B(8, 6): $(50.000 \times 8) + (80.000 \times 6) = 400.000 + 480.000 = \text{Rp } 880.000$.
4. Uji batas rute komputasi limit rute arsiran posisi limit letak murni rute fiktif pinggir rute pojok irisan terakhir letak C(0, 12): $(50.000 \times 0) + (80.000 \times 12) = \text{Rp } 960.000$.
5. Perhatikan rasio mutlak rekapitulasi poin limitasi margin tertinggi di rute kalkulasi arsiran nilai akhir Z uji komposit kombinasinya letak fiktif mutlak. Jadi fungsional nilai rasio komposit letak poin puncak dari kalkulasi omset letak laba perusahan rentang batas hitung laba rute arsiran maksimal rasio tertingginya yaitu jatuh murni pada nilai uang ukur rentangan rupiah modifikasi komputasi sejumlah rasio uang Rp 960.000 rupiah murni rentang labanya.
