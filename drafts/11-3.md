# Bab 3: Trigonometri Lanjutan

## 1. Akar Konsep
Trigonometri lanjutan mengeksplorasi modifikasi dari rasio sudut dasar dengan melakukan penjumlahan, selisih antar sudut, dan analisis perhitungan lipat ganda (sudut ganda). Konsep ini ibarat menganalisis dua putaran ombak air di laut. Ketika dua gelombang ombak yang ukurannya berbeda saling bertabrakan, tingginya bisa saling memperkuat (menjumlah) atau saling meredam (selisih ganda). Trigonometri lanjutan menghitung secara persis hasil riak gabungan dari "tabrakan" tersebut.

## 2. Pijakan Bumi
- **Insinyur Suara Analog:** Mencampur manipulasi dua frekuensi gelombang gelombang suara beda amplitudo (mixing) menggunakan identitas selisih sinus dan penjumlahan cosinus untuk mencegah pecahnya kejernihan suara.
- **Ahli Grafik Komputer (Animasi):** Menggunakan fungsi matriks modifikasi rotasi dari sudut manipulasi ganda (double-angle) untuk menggambar dan menghaluskan animasi perputaran dari lengan robot.

## 3. Mekanika Dasar
Rumus jumlah dan selisih sinus-cosinus:
$\sin(\alpha \pm \beta) = \sin \alpha \cos \beta \pm \cos \alpha \sin \beta$
$\cos(\alpha \pm \beta) = \cos \alpha \cos \beta \mp \sin \alpha \sin \beta$

Sudut ganda:
$\sin 2\alpha = 2 \sin \alpha \cos \alpha$

Contoh untuk menghitung $\cos(15^\circ)$:
$\cos(15^\circ) = \cos(45^\circ - 30^\circ)$
$\cos(15^\circ) = \cos(45^\circ)\cos(30^\circ) + \sin(45^\circ)\sin(30^\circ)$
$\cos(15^\circ) = (\frac{1}{2}\sqrt{2})(\frac{1}{2}\sqrt{3}) + (\frac{1}{2}\sqrt{2})(\frac{1}{2})$
$\cos(15^\circ) = \frac{1}{4}\sqrt{6} + \frac{1}{4}\sqrt{2}$

## 4. Logika Komputasi
```javascript
function hitungCosSelisihDuaSudut(sudutSatuD, sudutDuaD) {
  // Parsing dari bentuk degree ke dalam ukuran radian murni
  const radSatu = sudutSatuD * (Math.PI / 180);
  const radDua = sudutDuaD * (Math.PI / 180);
  
  // Mengaplikasikan identitas selisih Cos(A-B) = CosA CosB + SinA SinB
  const turunanAwal = Math.cos(radSatu) * Math.cos(radDua);
  const turunanAkhir = Math.sin(radSatu) * Math.sin(radDua);
  
  return turunanAwal + turunanAkhir;
}

// Simulasi nilai yang serupa dengan cosinus 15 derajat
const hasilMekanika = hitungCosSelisihDuaSudut(45, 30);
console.log(hasilMekanika); // Output: ~0.9659
```

## 6. Uji Ketangkasan
**Soal:**
Apabila diketahui angka dari besaran porsi $\sin \alpha = 5/13$ (dimana $\alpha$ adalah sudut siku positif lancip), tentukanlah nilai rincian dari fungsi manipulasi sudut ganda khusus $\sin(2\alpha)$.

**Penyelesaian:**
1. Karena ini adalah segitiga lancip dengan rupa pola $(5, X, 13)$, kita perlu mencari arah samping $(X)$ menggunakan dalil aljabar Pythagoras murni: $X = \sqrt{13^2 - 5^2} = \sqrt{169 - 25} = 12$. Maka $\cos \alpha = 12/13$.
2. Pergunakan formula murni rumus fungsi untuk modifikasi ganjil trigonometri tipe sudut ganda: $\sin(2\alpha) = 2 \sin \alpha \cos \alpha$.
3. Masukkan angka dari pecahan nilai: $2 \times (5/13) \times (12/13) = 120 / 169$. Jawabannya adalah bilangan proporsional seratus dua puluh per seratus enam puluh sembilan.
