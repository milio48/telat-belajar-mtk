# Bab 4: Lingkaran

## 1. Akar Konsep
Lingkaran dalam pandangan aljabar adalah tempat kedudukan dari himpunan titik-titik pada bidang datar yang jaraknya selalu sama absolut terhadap suatu titik pusat yang spesifik. Lingkaran ibarat mengikat seekor anjing peliharaan ke sebuah patok besi pusat menggunakan tali karet sejauh dua meter. Area batasan paling jauh tempat sang anjing bisa belari memutar bebas dengan tarikan kencang tali ukur membentuk kerangka lingkaran sejati.

## 2. Pijakan Bumi
- **Radar Navigasi Menara:** Membuat perangkat lunak batas kendali bandara yang membuat peringatan matematis ketika ada jejak objek pesawat masuk lebih dekat ke batas titik koordinasi persamaan menara radius tersebut.
- **Ahli Perakit Poros Mekanik:** Menghitung ukuran kemiringan pada alat mesin pencacah singgung (garis singgung lingkaran) melalui sudut kalkulasi yang hanya mengenai sabuk secara utuh dan tepat tanpa adanya tabrakan material memotong.

## 3. Mekanika Dasar
Persamaan lingkaran standar dengan pusat $(0,0)$ adalah:
$x^2 + y^2 = r^2$

Persamaan dengan titik pusat acak sembarang di $(a,b)$:
$(x - a)^2 + (y - b)^2 = r^2$

Contoh untuk pusat $(2, 3)$ dengan jari jari $4$:
Persamaan ringkasnya: $(x - 2)^2 + (y - 3)^2 = 16$
Terjemahan panjang persamaannya: $(x^2 - 4x + 4) + (y^2 - 6y + 9) = 16$, sehingga menjurus pada $x^2 + y^2 - 4x - 6y - 3 = 0$.

## 4. Logika Komputasi
```javascript
function ujiTitikLingkaran(pusatX, pusatY, radiusAsli, targetX, targetY) {
  // Mengecek apakah suatu titik terletak di dalam, tepat pada batas alas, atau luar
  const jarakSisiX = Math.pow((targetX - pusatX), 2);
  const jarakSisiY = Math.pow((targetY - pusatY), 2);
  const lokasiAsliTitik = jarakSisiX + jarakSisiY;
  const radiusKuadratLingkaran = Math.pow(radiusAsli, 2);
  
  if (lokasiAsliTitik < radiusKuadratLingkaran) return "Di dalam";
  if (lokasiAsliTitik === radiusKuadratLingkaran) return "Melekat (Pada garis)";
  return "Di luar radius batasan";
}

const statusRadar = ujiTitikLingkaran(2, 3, 4, 2, 5); // x:2,y:5 dengan pusat lingkaran (2,3),r=4
console.log(statusRadar); // Output: Di dalam
```

## 6. Uji Ketangkasan
**Soal:**
Tentukan rumusan baku persamaan sebuah lingkaran mutlak jika lingkaran tersebut melabuhkan wujud titik pusat rotasinya bersemayam di koordinat persis $A(-3, 2)$ serta jari-jarinya terukur mantap 5 satuan unit meter.

**Penyelesaian:**
1. Rangkum elemen parameter mutlak dari soal: $a = -3, b = 2$, dan bagian terluar kerangjang $r = 5$.
2. Padankan data parameter tersebut ke dalam skema wujud persamaan kurva lingkaran yang tidak berpusat di titik semesta awal nol: $(x - a)^2 + (y - b)^2 = r^2$.
3. Masukkan perhitungan parameter: $(x - (-3))^2 + (y - 2)^2 = 5^2$. Hasil draf akhirnya menjadi bentuk formula $(x + 3)^2 + (y - 2)^2 = 25$.
