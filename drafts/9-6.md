# Bab 6: Relasi dan Fungsi

## 1. Akar Konsep
Relasi adalah hubungan keterikatan bebas antara dua himpunan bilangan himpunan kelompok independen. Fungsi (atau pemetaan) adalah bentuk relasi khusus di mana setiap anggota satuan asal (domain) memiliki satu pasangan pasti tak ganda dengan anggota di ruang hasil tangkapan. Relasi ibarat kontak nomor teman di telepon bebas arah, fungsi ibarat nomor induksi sidik jari telunjuk staf kantor yang menempel ke presensi kerja spesifik setiap harinya tiada ganda.

## 2. Pijakan Bumi
* **Administrator Basis Data**: Menggunakan fungsi logika mesin penarik tarif retribusi yang mengubah waktu parkir (jam) menjadi jumlah pasti tagihan portal pembayaran.
* **Pakar Keuangan Saham**: Membuat relasi dan grafik perubahan posisi uang komoditi atas hasil laporan panen secara per bulan.

## 3. Mekanika Dasar
Struktur penulisan logika pemetaan dasar f terhadap x tertulis:
$$ f(x) = ax + b $$

Contoh substitusi hitungan f(x):
Diberikan model $f(x) = 2x + 1$. Tentukan total hitungan tangkap titik relasi $f(3)$. 
Maka hitungannya $f(3) = 2(3) + 1 = 6 + 1 = 7$.

## 4. Logika Komputasi
```javascript
function hitungHasilPemetaan(skalaFungsi) {
  const konstantaSatu = 2;
  const konstantaDua = 1;
  const totalBeban = (konstantaSatu * skalaFungsi) + konstantaDua;
  return totalBeban;
}

const pemetaanAkhir = hitungHasilPemetaan(3);
```

## 6. Uji Ketangkasan
**Soal:** Diketahui ada mesin pemetaan sandi logika fungsi dengan ekspresi $g(x) = 5x - 2$. Tentukan nilai yang tepat angka keluaran mesin jikalau $x = 4$.

**Penyelesaian:**
1. Ambil relasi rumus aturan dasarnya: $g(x) = 5x - 2$.
2. Ubah isi dan ganti huruf identitas $x$ langsung menggunakan angka input 4.
3. Hitung runtutan langkah kalkulasi: $g(4) = 5(4) - 2 = 20 - 2 = 18$.
