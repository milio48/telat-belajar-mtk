# Bab 1: Pengenalan dan Operasi Bilangan

## 1. Akar Konsep (Definisi & Deskripsi)

Bayangkan kamu memiliki sebuah kotak. Jika kamu ingin menghitung berapa banyak kelereng yang ada di dalam kotak tersebut, kamu pasti akan mulai berhitung dari "Satu, dua, tiga, ...". Angka-angka yang kita gunakan untuk menghitung benda nyata ini disebut **Bilangan Asli**. 

Namun, bagaimana jika kotak tersebut kosong? Kita membutuhkan sebuah simbol untuk mewakili "ketiadaan". Angka tersebut adalah "Nol" (0). Jika kita menggabungkan angka nol ke dalam barisan bilangan asli, kumpulan angka tersebut ($0, 1, 2, 3, \dots$) dinamakan **Bilangan Cacah**.

Jadi, bilangan asli dimulai dari $1$, sedangkan bilangan cacah dimulai dari $0$.

Dalam matematika, kita juga dapat menggabungkan atau mengurangi isi dari dua kelompok benda. Ini disebut operasi penjumlahan dan pengurangan. Menariknya, jika kamu mempunyai 3 kelereng di tangan kiri dan 2 kelereng di tangan kanan, jumlah keseluruhannya akan sama saat kamu menukarkan posisinya (2 di tangan kiri, 3 di tangan kanan). Ini dinamakan sifat pertukaran.

## 2. Pijakan Bumi (Contoh Dunia Nyata)

Konsep hitung dasar ini digunakan oleh banyak profesi:
* **Kasir Minimarket**: Mereka harus menjumlahkan total harga barang yang dibeli dan mengurangkan jumlah uang yang dibayarkan untuk menghitung kembalian.
* **Petugas Gudang**: Mencatat jumlah stok barang masuk (penjumlahan) dan barang yang keluar untuk dikirim (pengurangan).
* **Penjual Buah**: Memastikan persediaan buah selalu cukup dengan menghitung keranjang yang penuh dan kosong.

## 3. Mekanika Dasar (Praktek Perhitungan)

Mari kita berlatih menggunakan angka sederhana dan menerapkan sifat-sifat operasinya:

**Penjumlahan (Menggabungkan dua kelompok benda):**
Misalnya kamu memiliki $12$ buku cerita dan mendapatkan hadiah $5$ buku lagi.
$12 + 5 = 17$

**Sifat Pertukaran (Komutatif):**
Menjumlahkan dua angka bisa ditukar posisinya tanpa mengubah hasil.
$8 + 4 = 12$
$4 + 8 = 12$
Sehingga, $8 + 4 = 4 + 8$.

**Sifat Pengelompokan (Asosiatif):**
Saat menjumlahkan tiga angka, kamu bisa memilih untuk menjumlahkan kelompok yang mana saja lebih dulu.
$(2 + 3) + 5 = 5 + 5 = 10$
$2 + (3 + 5) = 2 + 8 = 10$

**Pengurangan (Mengambil sebagian benda):**
Jika kamu memiliki $20$ buah permen dan membagikan $6$ buah kepada temanmu, sisa permen dihitung sebagai:
$20 - 6 = 14$

## 4. Logika Komputasi (Implementasi Kode)

Di dalam program komputer, penjumlahan dan pengurangan diwakili menggunakan operator `+` dan `-`. Mesin dapat menghitungnya dalam hitungan milidetik.

```javascript
// Deklarasi variabel jumlah barang awal
let stokPensil = 15;
let kirimanBaru = 5;

// Melakukan perhitungan penjumlahan
let totalPensil = stokPensil + kirimanBaru;
console.log("Total Pensil:", totalPensil); // Output: 20

// Mencontohkan sifat pertukaran
let hitungA = 10 + 5;
let hitungB = 5 + 10;
let sifatPertukaranBenar = (hitungA === hitungB);
console.log("Apakah sifat pertukaran berlaku?", sifatPertukaranBenar); // Output: true

// Melakukan perhitungan pengurangan
let pensilTerjual = 4;
let sisaPensil = totalPensil - pensilTerjual;
console.log("Sisa Pensil di Toko:", sisaPensil); // Output: 16
```

## 5. Navigasi Global (Glosarium Inggris)

* **Natural Numbers:** Bilangan asli ($1, 2, 3, \dots$)
* **Whole Numbers:** Bilangan cacah ($0, 1, 2, 3, \dots$)
* **Addition:** Penjumlahan
* **Subtraction:** Pengurangan
* **Commutative Property:** Sifat pertukaran
* **Associative Property:** Sifat pengelompokan

## 6. Uji Ketangkasan (Soal & Penyelesaian)

**Soal 1:**
Andi memiliki $14$ batu bata. Budi memberinya $5$ batu bata lagi. Kemudian, Andi menggunakan $8$ batu bata untuk menyusun pondasi rumah mainannya. Berapa sisa batu bata Andi sekarang?

**Penyelesaian:**
1. Alur penjumlahan berdasarkan batu bata yang didapat: $14 + 5 = 19$.
2. Alur pengurangan saat digunakan: $19 - 8 = 11$.
3. Jadi, sisa batu bata Andi adalah $11$ buah.

**Soal 2:**
Tentukan nilai dari bagian kosong ini dengan memanfaatkan sifat pertukaran penjumlahannya: $15 + 7 = \dots + 15$.

**Penyelesaian:**
Menurut sifat pertukaran (komutatif), posisi angka dapat bergeser tanpa mengubah nilai jumlah akhirnya. Oleh karena itu, $15 + 7 = 7 + 15$. Jawaban untuk titik-titik adalah $7$.
