# Bab 1: Pengenalan dan Operasi Bilangan (Sampai 2 Angka)

## 1. Akar Konsep (Definisi & Deskripsi)
Bayangkan kamu memiliki sebuah kotak kosong. Saat kotak itu benar-benar kosong, jumlah isinya adalah **Nol (0)**. Bilangan yang dimulai dari nol disebut **Bilangan Cacah**. Jika kamu memasukkan satu kelereng ke dalam kotak, lalu dua, lalu tiga, dan seterusnya, hitungan yang dimulai dari benda nyata pertama (1, 2, 3...) disebut **Bilangan Asli**. 

## 2. Pijakan Bumi (Contoh Dunia Nyata)
- **Kasir Minimarket**: Kasir harus menghitung jumlah barang yang dibeli dan memastikan totalnya benar.
- **Peternak**: Menghitung jumlah ayam di kandangnya setiap pagi. Jika ada yang hilang atau bertambah, peternak harus tahu persis perbedaannya.

## 3. Mekanika Dasar (Praktek Perhitungan)
**Membilang dan Menulis:**
1, 2, 3, 4, 5... (membilang satu per satu)
Jika kita punya 12 apel, kita menulisnya dengan lambang: $12$.

**Penjumlahan (Sampai 20):**
Kamu punya 8 jeruk, lalu temanmu memberi 5 jeruk lagi.
$8 + 5 = 13$

**Pengurangan (Sampai 20):**
Dari 13 jeruk, kamu memakan 4.
$13 - 4 = 9$

**Sifat Pertukaran (Komutatif):**
Menjumlahkan 5 + 3 sama hasilnya dengan 3 + 5.
$5 + 3 = 8$
$3 + 5 = 8$

## 4. Logika Komputasi (Implementasi Kode)
Dalam bahasa pemrograman, kita bisa membuat mesin penjumlah dan pengurang sederhana.

```javascript
// Mesin Penjumlah
function tambah(a, b) {
  return a + b;
}

// Mesin Pengurang
function kurang(a, b) {
  return a - b;
}

console.log(tambah(5, 3)); // Output: 8
console.log(kurang(13, 4)); // Output: 9
```

## 5. Navigasi Global (Glosarium Inggris)
- Bilangan Asli: *Natural Numbers*
- Bilangan Cacah: *Whole Numbers*
- Penjumlahan: *Addition*
- Pengurangan: *Subtraction*
- Sifat Pertukaran: *Commutative Property*

## 6. Uji Ketangkasan (Soal & Penyelesaian)
**Soal 1:**
Budi memiliki 15 kelereng. Ia memberikan 6 kelereng kepada Andi. Berapa sisa kelereng Budi?
**Penyelesaian 1:**
$15 - 6 = 9$. Sisa kelereng Budi adalah 9.

**Soal 2:**
Di taman ada 7 burung merpati dan 8 burung gereja. Berapa total burung di taman? Buktikan bahwa urutan penjumlahan tidak mengubah hasil!
**Penyelesaian 2:**
Burung merpati + burung gereja: $7 + 8 = 15$.
Burung gereja + burung merpati: $8 + 7 = 15$.
Sifat pertukaran terbukti benar, totalnya tetap 15 burung.
