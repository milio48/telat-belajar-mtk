# Bab 1: Operasi Bilangan Bulat, Pangkat, dan Akar

## Akar Konsep
Bilangan bulat mencakup seluruh nilai utuh yaitu bilangan positif, negatif, dan nol. Pangkat adalah cara singkat untuk menulis perkalian bilangan yang sama secara berulang. Akar merupakan operasi kebalikan dari pangkat untuk mencari bilangan asal. Pangkat dan akar ibarat melipat sebuah kertas menjadi dua bagian secara berulang untuk mengecilkan luasnya, lalu membentangkannya kembali ke ukuran aslinya.

## Pijakan Bumi
- **Akuntan Publik:** Menggunakan operasi bilangan positif dan negatif untuk menghitung posisi laba atau rugi neraca perusahaan.
- **Ahli Kriptografi:** Menerapkan operasi bilangan berpangkat dan akar untuk mengamankan data enkripsi kata sandi digital.

## Mekanika Dasar
Operasi hitung campuran memiliki prioritas: tanda kurung, pangkat dan akar, perkalian dan pembagian, kemudian penjumlahan dan pengurangan.
Contoh:
$2^3 + \sqrt{16} - (-5)$
$= (2 \times 2 \times 2) + 4 + 5$
$= 8 + 4 + 5 = 17$

## Logika Komputasi
```javascript
function hitungCampuran(angkaDasar, angkaPangkat, angkaAkar, angkaNegatif) {
    const hasilPangkat = Math.pow(angkaDasar, angkaPangkat);
    const hasilAkar = Math.sqrt(angkaAkar);
    const operasiAkhir = hasilPangkat + hasilAkar - angkaNegatif;
    
    return operasiAkhir;
}

const totalHitung = hitungCampuran(2, 3, 16, -5);
console.log(totalHitung); // Output: 17
```

## Uji Ketangkasan
**Soal:** Sebuah toko swalayan awalnya merugi sebesar Rp 15.000. Untuk menutupi kerugian, pengelola menyuntikkan dana sebesar $5^3 \times 100$ Rupiah. Toko tersebut kemudian mendapatkan potongan pajak senilai akar dari $1.000.000$ Rupiah, yang dikembalikan secara tunai. Berapa total saldo keuangan toko saat ini?

**Penyelesaian:**
1. Hitung nilai pangkat terlebih dahulu: $5^3 = 5 \times 5 \times 5 = 125$. Nilai suntikan dana adalah $125 \times 100 = 12.500$.
2. Hitung nilai akar dari potongan pajak: $\sqrt{1.000.000} = 1000$.
3. Hitung total akhir menggunakan operasi bilangan campuran: Rugi (negatif) + Dana + Pajak Kembalian. Operasinya adalah $-15.000 + 12.500 + 1000 = -1500$.
Saldo keuangan toko masih merugi sebesar Rp 1.500.
