# Bab 4: Bangun Datar, Keliling, dan Luas

## 1. Akar Konsep (Definisi & Deskripsi)
- **Keliling**: Batas luar pagar suatu kebun, yakni menjumlahkan batas per satu setiap ujung tepinya.
- **Luas**: Seberapa besar area/permukaan lantai atau tikar di kebun tersebut. Berapa bayak kotak kecil pembatas yang bisa dimasukkan.

## 2. Pijakan Bumi (Contoh Dunia Nyata)
- **Tukang Pagar (Keliling)**: Membeli material kawat yang dihitung dengan Keliling.
- **Pengecat Ruangan (Luas)**: Pengecat dinding tembok perlu mengitung luas (area kotak dinding) sebelum memutuskan beli kaleng cat volume brp.

## 3. Mekanika Dasar (Praktek Perhitungan)
**Keliling:**
Keliling persegi = menjumlahkan 4 sisinya.
$K = \text{sisi} \times 4$
Keliling persegi panjang = $2 \times (\text{panjang} + \text{lebar})$

**Luas:**
Luas Persegi (membentuk area ubin):
$L = \text{sisi} \times \text{sisi}$ 
Luas Persegi panjang:
$L = \text{panjang} \times \text{lebar}$

Misal: Persegi panjang dengan panjang 5 dan lebar 3.
Luas = $5 \times 3 = 15$ satuan persegi.

## 4. Logika Komputasi (Implementasi Kode)
Dalam grafik rendering di browser (Canvas/HTML elements) luas itu di kalkulasi pada ukuran lebar pixels.

```javascript
class PersegiPanjang {
  constructor(panjang, lebar) {
    this.panjang = panjang;
    this.lebar = lebar;
  }
  
  hitungLuas() {
    return this.panjang * this.lebar;
  }
  
  hitungKeliling() {
    return 2 * (this.panjang + this.lebar);
  }
}

const lapangan = new PersegiPanjang(10, 5);
console.log(`Luas Area: ${lapangan.hitungLuas()}`); // Output: "Luas Area: 50"
console.log(`Kawat Keliling: ${lapangan.hitungKeliling()}`); // Output: "Kawat Keliling: 30"
```

## 5. Navigasi Global (Glosarium Inggris)
- Keliling: *Perimeter*
- Luas: *Area*
- Panjang: *Length*
- Lebar: *Width*
- Sudut Lancip / Tumpul / Siku-siku: *Acute / Obtuse / Right Angle*

## 6. Uji Ketangkasan (Soal & Penyelesaian)
**Soal 1:**
Sebuah petak sawah berbentuk persegi punya panjang sisi 10 meter. Berapa keliling dan luasnya?
**Penyelesaian 1:**
Keliling: $10 \times 4 = 40$ meter.
Luas: $10 \times 10 = 100$ meter persegi ($m^2$).

**Soal 2:**
Kebun bentuk persegi panjang, panjang 15m, lebarnya 5m. Petani akan memasang pagar listrik di kelilingnya. Berapa panjang kawat yang dibutuhkan?
**Penyelesaian 2:**
Pagar berarti Keliling: $K = 2 \times (15 + 5) = 2 \times 20 = 40$ meter. Butuh kawat 40m.
