# Bab 3: Geometri (Luas dan Volume Lanjutan)

## 1. Akar Konsep
Pengukuran bangun datar dan bangun ruang lanjutan melibatkan perhitungan dimensi bidang bersudut dan melengkung, seperti segi banyak, lingkaran, prisma, serta tabung. Menghitung ruang ini ibarat mengetahui seberapa banyak ukuran bentang bahan pelat besi yang dibutuhkan untuk membuat sebuah pipa saluran air (luas permukaan luar) atau mengetahui seberapa banyak air yang dapat dimuat di dalamnya (volume).

## 2. Pijakan Bumi
- **Pembuat kemasan bahan:** Menentukan luasan kertas karton baku yang diperlukan untuk membuat sebuah kotak kardus coklat yang berbentuk prisma segitiga.
- **Insinyur perminyakan:** Menghitung daya tampung muat kapasitas minyak bumi mentah di dalam sebuah tangki penampungan ladang pusat berbentuk tabung berukuran raksasa.

## 3. Mekanika Dasar
Rumus luas lingkaran:
$L = \pi \times r^2$

Rumus volume tabung:
$V = \pi \times r^2 \times t$

Keterangan:
$\pi$ = $3.14$ (Konstanta pi rasio lingkaran)
$r$ = Jari-jari lingkaran bidang alas
$t$ = Tinggi silinder tabung

Contoh perhitungan luas sebuah lingkaran:
Jika panjang garis tengah jari-jari $r = 10$ meter, maka luasan lingkarannya:
$L = 3.14 \times 10^2 = 3.14 \times 100 = 314$ meter persegi.

## 4. Logika Komputasi
```javascript
function hitungVolumeTabung(jariJari, tinggi) {
  const konstantaPi = 3.14;
  const luasBidangAlas = konstantaPi * jariJari * jariJari;
  const nilaiVolumeTotal = luasBidangAlas * tinggi;
  
  return nilaiVolumeTotal;
}

const radiusTangkiData = 10;
const tinggiTangkiData = 5;
console.log(hitungVolumeTabung(radiusTangkiData, tinggiTangkiData)); // Output: 1570
```

## 6. Uji Ketangkasan
**Soal:**
Sebuah drum wadah berbentuk tabung yang kokoh memiliki jari-jari bentuk alas $10$ meter serta ketinggian selubung $2$ meter tegak lurus. Berapakah ukuran nilai hitung volume udara kosong yang bermuatan di dalam drum logam tersebut saat wadahnya tertutup rapat sepenuhnya? (Gunakan standar $\pi = 3.14$)

**Penyelesaian:**
1. Identifikasi informasi data teknis bilangan masukan: konstanta rasio $\pi = 3.14$, ruas jari-jari $r = 10$, ukuran tinggi $t = 2$.
2. Hitung ukuran dari bidang alas lingkaran dengan penerapan bentuk rumus persamaan bangun $L = \pi \times r^2$, dengan komputasi operasi $3.14 \times (10 \times 10) = 314$ meter persegi.
3. Kalikan luas alas pada perhitungan di poin atas terhadap variabel dimensi ketinggian selubung sisi dengan mengikuti standar rumus volume silinder $V = L \times t$, sehingga mendapat $314 \times 2 = 628$ meter kubik.
