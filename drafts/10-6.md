# Bab 6: Trigonometri Dasar

## 1. Akar Konsep
Trigonometri dasar adalah studi mengenai relasi ukuran sisi dan celah sudut dari bangun segitiga, khususnya segitiga siku-siku. Rasio ini menghasilkan pola sinus, cosinus, dan tangen. Trigonometri ibarat mekanisme senter proyeksi bayangan. Semakin miring sudut cahaya disorot ke sebuah tiang, semakin panjang bayangan yang terbentuk di lantai, yang mana panjang ini bisa dihitung menggunakan perbandingan proporsi trigonometri.

## 2. Pijakan Bumi
- **Arsitek Bangunan:** Menghitung kemiringan dan panjang rangka kayu penyangga atap rumah agar sesuai dengan ukuran tinggi dindingnya menggunakan perbandingan cosinus atau sinus.
- **Navigator Kapal:** Mengukur jarak perahu yang sedang berlabuh dengan posisi tebing mercusuar tanpa perlu mengukur secara fisik manual di perairan laut, melainkan menggunakan theodolite untuk memakai sudut elevasi tangen tiang.

## 3. Mekanika Dasar
Konsep pada segitiga siku-siku (sudut $\theta$ terhadap sisi miring $r$, sisi depan $y$, sisi samping $x$):
$\sin \theta = \frac{\text{Depan}}{\text{Miring}} = \frac{y}{r}$
$\cos \theta = \frac{\text{Samping}}{\text{Miring}} = \frac{x}{r}$
$\tan \theta = \frac{\text{Depan}}{\text{Samping}} = \frac{y}{x}$

Contoh: Segitiga dengan sisi tegak $3$, sisi mendatar $4$, panjang miring $5$.
$\sin \theta = \frac{3}{5} = 0.6$
$\cos \theta = \frac{4}{5} = 0.8$

## 4. Logika Komputasi
```javascript
function hitungTinggiGedung(jarakPengamat, sudutElevasiDerajat) {
  // Dalam JavaScript tangen membutuhkan input dalam radian
  const phiKonstanta = Math.PI;
  const sudutRadian = sudutElevasiDerajat * (phiKonstanta / 180);
  
  // Rumus: tan = depan / samping -> depan = tan * samping
  const rasioTangen = Math.tan(sudutRadian);
  const tinggiGedung = rasioTangen * jarakPengamat;
  
  return tinggiGedung;
}

const estimasiTinggi = hitungTinggiGedung(100, 45); // jarak 100m, sudut 45 derajat
console.log(estimasiTinggi); // Output: 100
```

## 6. Uji Ketangkasan
**Soal:**
Seorang pengamat berdiri sejauh 20 meter dari sebuah pohon cemara yang tumbuh sangat tegak. Dari pandangan matanya (posisi bawah), ia melihat ujung atas pohon itu dengan sudut pantauan elevasi sebesar $60^\circ$. Hitunglah tinggi pohon cemara tersebut (Abaikan tinggi badan pengamat dan anggap nilai dari $\tan 60^\circ$ adalan $\sqrt{3}$).

**Penyelesaian:**
1. Gambarkan model segitiga posisinya. Tinggi pohon adalah sisi yang menghadap sudut observasi "Depan", dan jarak pengamat adalah lintasan alas tapak "Samping".
2. Fungsi trigonometris yang pas adalah Tangen: $\tan 60^\circ = \frac{\text{Tinggi Pohon}}{\text{Jarak Orang}}$.
3. Lakukan substitusi variabel: $\sqrt{3} = \frac{\text{Tinggi Pohon}}{20}$. Berarti Tinggi Pohon $= 20 \sqrt{3}$ meter.
