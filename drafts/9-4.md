# Bab 4: Pola Bilangan

## 1. Akar Konsep
Pola bilangan adalah susunan atau urutan angka-angka yang membentuk aturan logika secara berulang. Pola ini dapat direpresentasikan dengan angka murni maupun konfigurasi susunan visual suatu objek nyata. Pola bilangan ibarat tangga nada dalam musik yang tersusun rapi atau pola tumpukan kaleng susu di rak swalayan.

## 2. Pijakan Bumi
* **Peneliti Data**: Mempelajari pola urutan angka pertumbuhan suatu penyakit secara progresif agar masa transisi vaksin bisa disiapkan.
* **Ahli Perkotaan**: Menyusun nomor pemukiman dan pembagian zona letak perumahan secara ganjil-genap yang terstruktur.

## 3. Mekanika Dasar
Pola bilangan aritmetika beraturan yang memiliki jarak perbedaan nilai konstan mempunyai struktur formula deret:
$$ U_n = a + (n - 1)b $$

Contoh susunan sederhana (selisih antar angka konstan bernilai 2): 
Barisan: 2, 4, 6, 8. Suku awal $a=2$, nilai beda $b=2$. Suku ketiga ($U_3$) adalah $2 + (3 - 1)2 = 6$.

## 4. Logika Komputasi
```javascript
function cariPolaUrutan(titikMulai, nilaiBeda, indexUrutan) {
  const deretJarak = indexUrutan - 1;
  const nilaiAkhir = titikMulai + (deretJarak * nilaiBeda);
  return nilaiAkhir;
}

const deretHasil = cariPolaUrutan(2, 2, 3);
```

## 6. Uji Ketangkasan
**Soal:** Terdapat sebuah aturan susun kelompok meja dengan jumlah secara berurutan: 5, 8, 11, ... Berapa jumlah meja pada urutan kelompok yang ke-10?

**Penyelesaian:**
1. Tentukan kondisi permulaan: $a=5$ dan beda/langkah dari 5 ke 8 adalah $b=3$.
2. Gunakan relasi aturannya: $U_n = a + (n - 1)b$.
3. Ganti nilainya untuk urutan ke-10: $U_{10} = 5 + (10 - 1)3 = 5 + 27 = 32$.
