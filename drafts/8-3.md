# Bab 3: Teorema Pythagoras

## 1. Akar Konsep
Teorema Pythagoras menyatakan bahwa pada semua bangun ruang datar bertipe struktur segitiga siku-siku, jumlah penghitungan kuadrat dari parameter panjang kedua sisi tegak sejajar selalu sama dengan besaran luas pemangkatan kuadrat sisi miring garis miringnya. Teorema ini menjamin hukum relasional numerik baku dari bagian rusuk sisi di batas miring murni tersebut. Struktur relasional geometri ini ibarat meletakkan papan tangga kemiringan sandar pada dinding tembok penyangga untuk mengambil rasio pencapaian pijakan vertikal elevasi paling optimal.

## 2. Pijakan Bumi
* **Tukang Konstruksi Kayu:** Mempraktikkan pengoperasionalan kuadrat perangkaian batangan panel meja perakitan mebel agar menghasilkan sudutan pinggiran produk yang sepenuhnya tegak rata seimbang.
* **Petugas Penyelamat Kebakaran:** Menggunakan kalkulasi rasio panjang diagonal bidang alat luncur penyelamatan gawat darurat guna menyesuaikan lokasi aman penempatan pondasi truk dasar kendaraan mereka.

## 3. Mekanika Dasar
Rumus utama persilangan batas bangun segitiga siku-siku Teorema Pythagoras:
$$ c^2 = a^2 + b^2 $$
Di mana letak rusuk siku miring adalah simbol parameter $c$, sedangkan ukuran perbatasan dinding sisinya adalah lambang satuan besaran $a$ dan variabel ukur nilai tinggi batas $b$.
Contoh: Diketahui skala pemetaan rasio alas sisi $a = 6$ senti dan tinggi rasio dinding tegak $b = 8$ centimeter.
$c^2 = 6^2 + 8^2$
$c^2 = 36 + 64 = 100$
Panjang kemiringan relasi: $c = \sqrt{100} = 10$.

## 4. Logika Komputasi
```javascript
function hitungHipotenusaSegitiga(sisiDatarAlas, sisiDatarTegak) {
  const areaPersegiAlas = sisiDatarAlas * sisiDatarAlas;
  const areaPersegiTegak = sisiDatarTegak * sisiDatarTegak;
  
  const jumlahBesaranKeduanya = areaPersegiAlas + areaPersegiTegak;
  const rasioDiagonalPenuh = Math.sqrt(jumlahBesaranKeduanya);
  
  return rasioDiagonalPenuh;
}

console.log(hitungHipotenusaSegitiga(6, 8));
```
