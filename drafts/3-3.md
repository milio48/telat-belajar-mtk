# Bab 3: Pengukuran Baku

## 1. Akar Konsep (Definisi & Deskripsi)
Pengukuran baku artinya mengubah ukuran menjadi satuan universal. Setiap satuan punya "Level", mirip dengan anak tangga. Ketika kamu berurusan dengan jarak jauh (contoh jalan tol), kilometer lebih pas dipakai dari centimter. Sama dari gram ke kilogram di satuan berat.

## 2. Pijakan Bumi (Contoh Dunia Nyata)
- **Pelari Maraton (Panjang/Jarak)**: Berlatih dengan mengukur berapa banyak KM lari hari itu, jika kurang sedikit (sisa dihitung pakai M bukan KM).
- **Resep Masakan Koki (Berat)**: Semua ukuran gula dan bahan kimia tepung dicatat gram dan kilogramnya.

## 3. Mekanika Dasar (Praktek Perhitungan)
**Hubungan Panjang (Km ke m):**
- 1 Kilometer (km) = 1.000 Meter (m)
$2 \text{ km} = 2 \times 1.000 \text{ m} = 2.000 \text{ m}$

**Hubungan Berat (Kg ke g):**
- 1 Kilogram (kg) = 1.000 Gram (g)
Jika tepung ada $3$ kg, maka itu sama dengan $3.000$ g.

**Hubungan Waktu:**
- 1 Jam = 60 Menit
- 1 Menit = 60 Detik

## 4. Logika Komputasi (Implementasi Kode)
Di komputer, konversi satuan ini sangat mudah karena kita menggunakan fungsi operasi hitungan baku.

```javascript
// Konverter Jarak: Km ke Meter
function kmTometer(km) {
  return km * 1000;
}

// Konverter Berat: Gram ke KG (Kebalikannya, tinggal Dibagi 1000)
function gramTokg(gr) {
  return gr / 1000;
}

console.log(`${kmTometer(2.5)} meter`); // Output: "2500 meter"
console.log(`${gramTokg(1500)} kg`);   // Output: "1.5 kg"
```

## 5. Navigasi Global (Glosarium Inggris)
- Jarak (Panjang): *Distance*
- Meter/Kilometer: *Meter / Kilometer*
- Konversi Satuan: *Unit Conversion*

## 6. Uji Ketangkasan (Soal & Penyelesaian)
**Soal 1:**
Jarak rumah Anto ke sekolah adalah 3 km. Berapa meter jaraknya?
**Penyelesaian 1:**
1 km = 1.000 m. Jarak 3 km = $3 \times 1.000 = 3.000$ meter.

**Soal 2:**
Ibu mempunyai beras seberat 2.500 gram. Berapa kg dan gram berat beras ibu?
**Penyelesaian 2:**
Oleh karena 1.000 gram itu sama dengan 1 kg. Maka, pecahkan berat beras: $2.500 = 2.000 + 500$
Jadi, ibu mempunyai $2 \text{ kg} \text{ dan } 500 \text{ gram}$ beras.
