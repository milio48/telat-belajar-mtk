# Bab 3: Pengukuran Baku

## 1. Akar Konsep
Pengukuran baku melibatkan penggunaan alat ukur dan satuan standar antarbangsa untuk memastikan besaran yang dihasilkan konsisten tanpa memandang siapa yang mengukurnya. Setiap satuan memiliki faktor konversi tertentu yang menjelaskan hubungan dan hierarki di antara jenis pengukuran, seperti alat ukur jarak untuk melihat seberapa panjang objek. Pengukuran baku diibaratkan seperti timbangan di pasar yang memiliki anak beban logam berukuran sama secara universal.

## 2. Pijakan Bumi
- **Arsitek bangunan:** Mengukur dimensi kavling tanah, panjang beton, dan lebar lahan dengan alat ukur presisi meteran gulung untuk membuat rencana perancangan.
- **Perawat kesehatan:** Mengukur berat badan penderita asma menggunakan timbangan digital saat melakukan kontrol berobat berkala.

## 3. Mekanika Dasar
Formula konversi antar satuan ukur pada umumnya berjenjang. Untuk satuan massa dengan awalan prefiks kelipatan sepuluh: $1 \text{ kg} = 1000 \text{ gram}$. 
Contoh: Ani memiliki beras di karung seberat 2 kilogram. Berarti beras tepung itu secara hitungan sama dengan $2 \times 1000 = 2000 \text{ gram}$.

## 4. Logika Komputasi
```javascript
function konversiKiloKeGram(nilaiGramAwal) {
  const konstantaBagi = 1000;
  let totalDikonversi = nilaiGramAwal * konstantaBagi;
  
  return totalDikonversi;
}

console.log(konversiKiloKeGram(2));
```

## 6. Uji Ketangkasan
**Soal:** Ayah membeli terigu seberat 3 kg di toko untuk membuat roti. Sesampainya di dapur rumah, ayah harus menimbang kembali dalam satuan gram. Berapa gram berat terigu tersebut kini?
**Penyelesaian:**
1. Identifikasi bentuk berat awal terigu yang dikonfirmasi: 3 kg. 
2. Ingat kembali rumus pengubahannya. Faktor pengalinya dari satuan kilogram menuju gram adalah dikali satu ribu.
3. Hitung hasil perkalian dengan cepat: $3 \text{ kg} \times 1000 = 3000 \text{ gram}$. Jadi, terigu tersebut seberat 3000 gram.
