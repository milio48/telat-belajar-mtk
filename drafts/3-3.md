# Bab 3: Pengukuran Baku

## 1. Akar Konsep (Definisi & Deskripsi)

Sebelumnya kita telah mengenal pengukuran standar yang disepakati internasional, seperti meter, kilogram, dan jam. Saat kita dewasa, dunia menuntut presisi yang lebih luas. 

Sebuah truk pengantar pasir tentu sangat repot jika harus ditimbang dalam porsi "gram" karena akan menghasilkan urutan angka yang terlalu panjang. Sebaliknya, obat nyamuk perlu ditimbang di angka lebih kecil karena "kilogram" terlalu besar dan kasar sehingga membahayakan reaksi kimia. Oleh sebab itu, sistem metrik kita mendesain "**Tangga Satuan Baku**".

Setiap kali kita merubah alat ke spesifikasi unit yang lebih kecil/halus, kita memperbanyak jumlah nilainya sepuluh kali lipat. Sebaliknya, saat kita merubah ke patokan beban skala berat yang lebih raksasa, kita mengikis nilainya dibagi sepuluh.

## 2. Pijakan Bumi (Contoh Dunia Nyata)

Memilih unit ukuran baku yang efisien dan mengkonversinya amat penting pada skenario teknis:
* **Logistik Kapal Barang:** Mereka tak mau pusing menggunakan satuan Kilogram untuk mencatat besi alat berat, mereka beralih memakai patokan besar Ton / Kuintal.
* **Laboratorium Medis:** Darah dihitung bukan per Liter air minum, melainkan per Mililiter ($ml$) karena volumenya sedikit demi sedikit diekstrak di alat sampel.
* **Pelari Marathon:** Melacak daya tahannya bukan dari detak detiknya, tapi menotal durasinya hingga bergeser menyentuh berapa Jam perjalanan. 

## 3. Mekanika Dasar (Praktek Perhitungan)

**Hubungan Antarsatuan Panjang (Tangga Turun):**
Meter ($m$) -> Desimeter ($dm$) -> Sentimeter ($cm$) -> Milimeter ($mm$)
Setiap turun satu tangga, angka dikali $\times 10$.
$1~m = 10~dm = 100~cm = 1.000~mm$. (Milimeter adalah garis sangat kecil di penggaris).

**Hubungan Antarsatuan Berat:**
Kilogram ($kg$) -> Hektogram/Ons ($hg$) -> Dekagram ($dag$) -> Gram ($g$)
Sama, setiap turun dikali $10$.
Banyak di pasar menggunakan sebutan "Ons".  $1~kg = 10~hg$ (Ons).
Atau beban angkut raksasa: $1$ Ton $= 1.000~kg$.

**Hubungan Waktu Baku:**
Sistem putaran tidak memakai prinsip 10, melainkan kelipatan putaran bumi/mesin mekanik 60:
* $1$ Jam $= 60$ Menit
* $1$ Menit $= 60$ Detik
* (Jadi, $1$ Jam $= 60 \times 60 = 3.600$ Detik)

## 4. Logika Komputasi (Implementasi Kode)

Aplikasi Navigasi Peta (GPS) selalu membacakan arah di sistem metrik agar lebih dipahami pengguna ketika kecepatan mobil tinggi atau pelan.

```javascript
// Konverter Jarak Kendaraan dari Meter ke Kilometer 
let jarakTersisaMeter = 4500;

// Karena Meter naik ke Kilometer (3 tangga), kita bagi dengan 1000
let konversiKilometer = jarakTersisaMeter / 1000;

if (konversiKilometer > 3) {
  console.log("Jarak masih: " + konversiKilometer + " km. Mari dengar lagu di radio.");
} else {
  console.log("Hampir sampai!");
}

// Konversi Waktu Menunggu Oven
let waktuOvenMenit = 120;
// Menit ke Jam (naik tanga waktu, maka dibagi 60)
let waktuJam = waktuOvenMenit / 60;
console.log("Lama memanggang setara dengan:", waktuJam, "Jam");
```

## 6. Uji Ketangkasan (Soal & Penyelesaian)

**Soal 1:**
Tukang cat memiliki sebuah tangga lipat. Panjang tangga tersebut terukur sejauh $3$ meter terbentang. Ubahlah panjang batang terbentang tangga lipat kayu itu ke dalam rincian jarak Sentimeter ($cm$).

**Penyelesaian:**
1. Ingat letak urutan tangganya: Meter berada di atas Sentimeter.
2. Dari Meter ke Desimeter (turun 1, maka $\times 10$). Dari Desimeter ke Sentimeter (turun 1 lagi, dikali $\times 10$). Artinya pengali utamanya adalah $\times 100$.
3. Perhitungannya: $3 \times 100 = 300$.
4. Jadi dimensi totalnya adalah **$300~cm$**.

**Soal 2:**
Ibu memasak gulai daging persiapan acara keluarga yang banyak. Di pasar ia memesan ke tukang potong daging sebanyak $4$ Kilogram. Berapa sajakah berat setara daging Ibu dalam nilai konversi Gram?

**Penyelesaian:**
1. Dari $kg$ ke gram, harus menuruni tangga panjang ($kg \rightarrow hg \rightarrow dag \rightarrow g$). Ada $3$ loncatan tangga.
2. $3$ anak tangga artinya kita menyertakan efek perkalian $\times 1.000$ lipat partikel nilai kekecilan gramnya.
3. Pengerjaan: $4 \times 1.000 = 4.000$.
4. Ibu secara utuh telah memesan irisan padat total hingga memuat **$4.000~g$** daging.
