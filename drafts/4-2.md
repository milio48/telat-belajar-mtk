# Bab 2: Kelipatan, Faktor, dan Bilangan Romawi

## Akar Konsep
Kelipatan adalah deret angka perolehan hasil perkalian antara suatu bilangan dengan rangkaian bilangan asli secara berurutan. Faktor prima merupakan himpunan bilangan prima yang bisa membagi rata bilangan asli dengan sisa perolehan pembagi bernilai nol. Menentukan Kelipatan Persekutuan Terkecil ibarat pencarian waktu siklus pertemuan dua buah bus angkutan pada halte keberangkatan dari titik interval jadwal terminal berbeda.

## Pijakan Bumi
- **Dokter jaga rumah sakit:** Menentukan rentang jam batas pembagian jadwal regu kelompok unit bedah berdasarkan konsep persekutuan kelipatan per perawat.
- **Drafter bangunan gedung:** Menempatkan letak ukir pilar bangunan sejajar yang bersudut presisi menggunakan patokan faktorisasi FPB angka struktur ornamen ukur presisi balok bangunan.

## Mekanika Dasar
Rumus mencari Kelipatan Persekutuan Terkecil (KPK) dari $a$ dan $b$:
$\text{KPK}(a,b) = \frac{|a \times b|}{\text{FPB}(a,b)}$

Contoh perhitungan dengan angka 4 dan 6:
Faktor Prima $4 = 2^2$
Faktor Prima $6 = 2 \times 3$
$\text{KPK} = 2^2 \times 3 = 4 \times 3 = 12$

## Logika Komputasi
```javascript
function cariKelipatanTerkecil(angkaPertama, angkaKedua) {
  let kelipatanMaksimal = angkaPertama * angkaKedua;
  let ukuranKelipatan = 1;
  while (ukuranKelipatan <= kelipatanMaksimal) {
    if (ukuranKelipatan % angkaPertama === 0 && ukuranKelipatan % angkaKedua === 0) {
      return ukuranKelipatan;
    }
    ukuranKelipatan++;
  }
}

console.log(cariKelipatanTerkecil(4, 6));
```

## Uji Ketangkasan
Lampu rambu persimpangan pertama di arah timur menyala bergilir pada tiap detik angka kelipatan tiga. Lampu rambu arah selatan juga beroperasi merotasi terang sinarnya tepat setiap empat detik. Pada waktu ke berapakah dua jenis rambu arah mata angin akan menunjukkan indikator penyinaran menyala tajam bertepatan selaras?
1. Susun barisan hasil dari angka kelipatan per tiganya (3, 6, 9, 12, 15).
2. Tulis letak deretan nilai angka bilangan interval lipat empat rentang ukurnya (4, 8, 12, 16).
3. Tandai dan temukan persimpangan letak kemunculan deret angka bernilai besaran hitung berbobot ekuivalen pada letak ke-12. Jadi pancaran terang bersama akan meronta terang tepat ketika menyinggung interval titik detak waktu hitung nilai indeks bilangan angka yang bernilai dua belas.
