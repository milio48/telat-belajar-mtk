# Bab 2: Kelipatan, Faktor, dan Bilangan Romawi

## 1. Akar Konsep (Definisi & Deskripsi)
- **Kelipatan**: Jika kamu meloncat 2 langkah, loncatanmu jatuh di: 2..4..6..8 dsb. Itulah kelipatan 2.
- **Faktor**: Adalag pembagi adil untuk suatu bilangan. Angka berapa saja yang kalau dibagi angka itu hasilnya pas (tanpa sisa).
- **Bilangan Prima**: Angka penyendiri. Angka kuat yang faktor pembaginya cuma 1 atau dirinya sendiri (tidak bisa dibagi rata oleh angka lain). $2, 3, 5, 7, 11$, dll.

## 2. Pijakan Bumi (Contoh Dunia Nyata)
- **Sistem Cuti Karyawan (KPK)**: Karyawan A cuti loncat setiap 4 hari. Karyawan B cuti loncat tiap 6 hari. Kapan mereka bisa cuti/libur main barengan? (Mencari Kelipatan Persekutuan Terkecil).
- **Bagi Raport Oleh Guru (FPB)**: Dari jumlah buku dan permen tersedia, berapa hadiah maksimal (kelompok anak) yang bisa dibagi secara imbang sama rata ke dalam tas paket bantuan tersebut? (Mencari Faktor Persekutuan Terbesar).

## 3. Mekanika Dasar (Praktek Perhitungan)
**Faktorisasi Prima KPK dan FPB**
(Gunakan pohon akar pembagian faktorisasi dasar dengan membagikan angka dengan faktor prima)

*Mencari FPB & KPK dari 12 & 18:*
$12 = 2 \times 2 \times 3$ $(2^2 \times 3)$
$18 = 2 \times 3 \times 3$ $(2 \times 3^2)$

- **FPB**: Ambil angka yang ADA DI KEDUANYA (persekutuan), lalu pilih pangkat *terkecil*.
$2 \times 3 = 6$
- **KPK**: Ambil SEMUA angka, jika ada beda kembar pangkat, pakai pangkat *terbesar*.
$2^2 \times 3^2 = 4 \times 9 = 36$

**Bilangan Romawi Dasar**
$I = 1$
$V = 5$
$X = 10$
$L = 50$
$C = 100$

## 4. Logika Komputasi (Implementasi Kode)
Bahasa pemprograman bisa cepat menghitung modulo (sisa pembagian) untuk mengecek FPB KPK menggunakan "Fungsi Rekursif".

```javascript
// Algoritma Euclidean untuk mencari FPB
function cariFPB(a, b) {
  if (b === 0) {
    return a;
  }
  return cariFPB(b, a % b); // rekursi pembagian sisa (modulo)
}

function cariKPK(a, b) {
    return (a * b) / cariFPB(a, b);
}

console.log(`FPB: ${cariFPB(12, 18)}`); // Output: 6
console.log(`KPK: ${cariKPK(12, 18)}`); // Output: 36
```

## 5. Navigasi Global (Glosarium Inggris)
- Kelipatan: *Multiples*
- Faktor: *Factors*
- Bilangan Prima: *Prime Numbers*
- FPB (Faktor Persekutuan Terbesar): *Greatest Common Divisor (GCD)*
- KPK (Kelipatan Persekutuan Terkecil): *Least Common Multiple (LCM)*

## 6. Uji Ketangkasan (Soal & Penyelesaian)
**Soal 1:**
Lonceng A berbunyi setiap 3 jam. Lonceng B bunyi setiap 4 jam. Jika jam 12.00 mereka bunyi bersamaan, kapan waktu keduanya akan berbunyi bareng kembali?
**Penyelesaian 1:**
KPK dari $3$ dan $4$ adalah $12$. (Mereka akan ketemu di jam ke-12). Pukul berbunyi lagi: $12.00 + 12 \text{ jam} = 24.00$ (Bisa dibilang tepat tengah malam putaran berikutnya).

**Soal 2:**
Konversikan angka bilangan cacah 23 ke bentuk Romawi!
**Penyelesaian 2:**
Tulis per-angka puluhan, lalu satuan. $20 = XX$, dan disusul $3 = III$.
Maka bentuk menjadi: $XXIII$.
