# Bab 7: Bidang Kartesius

## 1. Akar Konsep
Bidang Kartesius adalah sistem koordinat dua dimensi yang dibentuk oleh dua garis bilangan tegak lurus: sumbu X (mendatar) dan sumbu Y (tegak). Titik perpotongan kedua sumbu disebut **titik asal** atau **origin** $(0, 0)$. Setiap titik pada bidang ini dinyatakan sebagai pasangan terurut $(x, y)$. Bidang Kartesius ibarat sistem alamat kursi di bioskop — nomor baris menunjukkan posisi horizontal dan nomor kolom menunjukkan posisi vertikal.

> **Catatan:** Bidang Kartesius terbagi menjadi empat **kuadran**[^1]. Kuadran I $(+, +)$, Kuadran II $(-, +)$, Kuadran III $(-, -)$, dan Kuadran IV $(+, -)$.

## 2. Pijakan Bumi
* **Pengembang Aplikasi Peta:** Menerapkan sistem koordinat GPS untuk menetapkan lokasi tujuan pengguna pada aplikasi navigasi.
* **Perancang Video Game:** Mengatur posisi dan pergerakan karakter pada layar menggunakan koordinat $(x, y)$ secara presisi.

## 3. Mekanika Dasar
Penulisan posisi titik pada bidang Kartesius:
$$ P(x, y) $$

Keterangan:
- $x$ = jarak horizontal dari titik asal (positif ke kanan, negatif ke kiri).
- $y$ = jarak vertikal dari titik asal (positif ke atas, negatif ke bawah).

Contoh: Titik $A(3, 4)$ berarti posisinya 3 satuan ke kanan dan 4 satuan ke atas dari titik asal $(0, 0)$.

## 4. Logika Komputasi
```javascript
function geserTitik(posisiX, posisiY, geserX, geserY) {
  const titikBaruX = posisiX + geserX;
  const titikBaruY = posisiY + geserY;

  return { x: titikBaruX, y: titikBaruY };
}

console.log(geserTitik(3, 4, 2, -1));
// Output: { x: 5, y: 3 }
```

## 6. Uji Ketangkasan

**Soal 1:**
Sebuah robot mainan berada di titik $K(2, 1)$. Robot bergeser 3 satuan ke kanan (arah sumbu X positif) dan 5 satuan ke bawah (arah sumbu Y negatif). Tentukan koordinat posisi akhir robot.

**Penyelesaian:**
1. Hitung koordinat X baru: $2 + 3 = 5$.
2. Hitung koordinat Y baru: $1 - 5 = -4$.
3. Posisi akhir robot adalah $K'(5, -4)$.

---

**Soal 2:**
Titik $P(4, -2)$ dicerminkan terhadap sumbu X. Tentukan koordinat titik hasil pencerminan $P'$.

**Penyelesaian:**
1. Pada pencerminan terhadap sumbu X, nilai $x$ tetap dan nilai $y$ berubah tanda[^2].
2. Koordinat awal: $P(4, -2)$.
3. Koordinat hasil pencerminan: $P'(4, 2)$.

---

[^1]: **Kuadran:** Empat bagian bidang yang terbentuk dari perpotongan sumbu X dan sumbu Y. Penomoran dimulai dari kanan atas (Kuadran I) dan berputar berlawanan arah jarum jam.

[^2]: **Pencerminan terhadap sumbu X:** Transformasi geometri di mana titik $(x, y)$ menjadi $(x, -y)$. Secara visual, bayangan titik tersebut "terlipat" melewati garis sumbu X.
