# Bab 2: Sistem Persamaan Linear Dua Variabel

## 1. Akar Konsep
Sistem Persamaan Linear Dua Variabel (SPLDV) adalah himpunan matematik yang memuat komponen beberapa persamaan linear dengan jumlah dua variabel tidak diketahui yang diikat untuk dicarikan serangkaian himpunan persimpangannya. Penyelesaian sistem merupakan himpunan nilai yang membuat semua persamaan menjadi proposisi benar. Konsep SPLDV ibarat mencari suatu lokasi titik persilangan temu antar dua ruas rel jalan kereta yang bertabrakan pada sebuah peta rancangan lahan.

## 2. Pijakan Bumi
* **Manajer Logistik:** Memanfaatkan pemodelan matematis ini untuk mengoptimalkan komposisi pembelian unit kargo armada transportasi berdasarkan kendala batas kapasitas serta batas anggaran keuangan harian perusahaan.
* **Ahli Gizi Medis:** Menerapkan penemuan persilangan komposisi massa campuran komponen bahan medis mentah yang paling tepat agar mencapai kadar rasio nutrien senyawa pasien target terpadu.

## 3. Mekanika Dasar
Bentuk formulasi sistem persamaan linear dua variabel:
$$ a_1x + b_1y = c_1 $$
$$ a_2x + b_2y = c_2 $$
Contoh perhitungan dari persamaan $x + y = 6$ serta $x - y = 2$:
Jalankan eliminasi penjumlahan: $(x + y) + (x - y) = 6 + 2 \implies 2x = 8 \implies x = 4$.
Substitusi nilai konversi variabel pertama ke susunan formulasi awal: $4 + y = 6 \implies y = 2$.
Nilai pertemuan akar himpunan adalah berlokasi di titik kordinat (4, 2).

## 4. Logika Komputasi
```javascript
function selesaikanEliminasiMatriks(koefisienTotal, koefisienSelisih) {
  // Hanya berlaku khusus kasus SPLDV berformat x+y dan x-y
  const temukanNilaiX = (koefisienTotal + koefisienSelisih) / 2;
  const temukanNilaiY = koefisienTotal - temukanNilaiX;
  
  return { variabelX: temukanNilaiX, variabelY: temukanNilaiY };
}

console.log(selesaikanEliminasiMatriks(6, 2));
```
