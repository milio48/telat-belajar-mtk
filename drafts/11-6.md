# Bab 6: Fungsi Komposisi dan Fungsi Invers

## 1. Akar Konsep
Fungsi komposisi adalah proses penggabungan dari tahapan alur fungsional pemetaan elemen matematika lebih dari satu susunan rujukan ganda. Fungsi invers adalah aksi mutlak sebaliknya yang ditujukan murni untuk merunut secara kebalikan jejak asal data operasi di dalam sebuah struktur mesin komposisi pemetaan aljabar. Fungsi komposisi ibarat tahapan rantai pekerja beruntun dalam perakitan bodi mobil. Fungsi pertama bertugas menghaluskan murni potongan lempeng besi dasar mobil, lantas lembaran rangka metal baja lempeng yang murni telah dihaluskan ini beranjak diserahkan segera secara serempak menuju fungsi orang yang kedua langsung semata khusus untuk penyemprot pewarnaan cat mesin bodi komponen otomotif yang solid.

## 3. Mekanika Dasar
Konfigurasi skema turunan sifat penggabungan fungsional: $(f \circ g)(x) = f(g(x))$.
Notasi baku tata kelola formula jejak mutlak kebalikan dari mesin matematis inversi tipe linear: Jika $y = f(x)$, maka rumusan murni untuk mendapatkan rupa dari formulasi bentuk invers dasar tersebut adalah manifestasi dari $x = f^{-1}(y)$.

Contoh kalkulasi mekanik. Diketahui rumusan baku $f(x) = 2x + 1$ dan model murni susunan wujud $g(x) = x - 3$.
Berapa wujud konkret gabungannya dalam sistem susunan tata ruang operasi $(f \circ g)(x)$ komputasinya?
1. Susun komposit rupa dalam rumus aljabar yang tersusun $f(g(x)) = 2(x - 3) + 1$.
2. Urai kurungnya pada sisi aljabarnya: $= 2x - 6 + 1$.
3. Kompletkan selisih nilai konstanta di belakang $x$: $(f \circ g)(x) = 2x - 5$.

## 4. Logika Komputasi
```javascript
function fungsiLogikaPengganda(nilaiX) {
  return (2 * nilaiX) + 1;
}

function fungsiLogikaPengurangan(nilaiX) {
  return nilaiX - 3;
}

function eksekutorKomposisiGanda(angkaUji) {
  // Mengeksekusi fungsi kedua (g), lalu memasukannya ke fungsi pertama (f)
  const hasilG = fungsiLogikaPengurangan(angkaUji);
  const hasilKomposisiFinal = fungsiLogikaPengganda(hasilG);
  
  return hasilKomposisiFinal;
}

// Menjalankan wujud contoh di atas f(g(10)) 
const tesMesinKomposisi = eksekutorKomposisiGanda(10);
console.log(tesMesinKomposisi); // Output: 15
```

## 6. Uji Ketangkasan
**Soal:**
Terdapat rumus fungsi kalkulator yang memetakan pertidaksamaan pemetaan linear ganjil memudar yaitu dengan wujud manifestasi linear bentuk umum fungsional $f(x) = 3x - 5$. Cobalah kerjakan pencarian rute susun balik mutlak yang merepresentasikan pembalikan fungsi invers resminya.

**Penyelesaian:**
1. Formulasikan ulang skema fungsi tersebut ke tataran penulisan murni $y$: ubah formasi wujud fiktif murni fungsional persamaan rute logistik wujud awal formula menjadi komposit persamaan tipe koefisien $y = 3x - 5$.
2. Tukar elemen untuk menyisihkan $x$. Rute ini menuntut nilai porsi penggeseran letak $3x$ dan pergeseran bilangan variabel 5 menuju bagian luar kurung sisi murni lainnya di seberang kiri. Berdasarkan komputasi transisi aljabar persamaan linear letak, wujud modifikasinya berganti susun bentuk menjadi ganjil wujud bentuk formulasi rute komposit $3x = y + 5$.
3. Isolasi variabel komposit penempatan tipe sumbu target $x$: membagi turunan hasil rasio secara rata dengan besaran koefisien pendamping linear terluar yang telah ada yaitu angka koefisien skalar 3 secara rasio komposit turunan. $x = (y + 5) / 3$. Dengan begitu maka fungsional mutlak jejak tipe invers akhirnya adalah fungsional kembalian susun fiktif formal wujud balikan kemurnian $f^{-1}(x) = (x + 5) / 3$.
