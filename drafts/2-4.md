# Bab 4: Bangun Datar

## 1. Akar Konsep (Definisi & Deskripsi)
Setiap bentuk datar yang tertutup punya "Pagar" dan "Pojok".
- **Sisi (Pagar)**: Garis lurus yang membatasi bentuk tersebut dari dunia luar.
- **Sudut (Pojok)**: Titik tajam tempat dua dari garis-garis pagar tersebut saling bertabrakan dan menempel.

## 2. Pijakan Bumi (Contoh Dunia Nyata)
- **Pembuat Rambu Lalu Lintas**: Membuat rambu peringatan (Segitiga) atau papan informasi (Persegi Panjang), harus memotong sisi-sisinya agar rapi dan tidak melukai pekerja.
- **Peletakan Ubin**: Lantai rumah menggunakan ubin yang dibentuk persegi (4 sisi, 4 sudut) agar bisa disusun rapat tanpa celah.

## 3. Mekanika Dasar (Praktek Perhitungan)
**Mengidentifikasi Sisi dan Sudut:**
- **Segitiga**: Punya $3$ sisi lurus. Punya $3$ sudut ujung yang lancip.
- **Persegi / Segi Empat**: Punya $4$ sisi lurus. Punya $4$ sudut.
- **Segi Lima (Pentagon)**: Punya $5$ sisi, $5$ sudut.

## 4. Logika Komputasi (Implementasi Kode)
Sebuah program komputer bisa menyimpan representasi bentuk ini dengan mencatat jumlah titik sudut (vertex/nodes).

```javascript
const daftarBangunDatar = {
  "Segitiga": { sisi: 3, sudut: 3 },
  "Persegi": { sisi: 4, sudut: 4 },
  "Segi Lima": { sisi: 5, sudut: 5 }
};

function kenaliBangun(sisi) {
  for (let nama in daftarBangunDatar) {
    if (daftarBangunDatar[nama].sisi === sisi) {
      return `Bangun ini adalah ${nama}`;
    }
  }
  return "Bentuk tidak diketahui";
}

console.log(kenaliBangun(4)); // Output: "Bangun ini adalah Persegi"
```

## 5. Navigasi Global (Glosarium Inggris)
- Sisi: *Side / Edge*
- Sudut: *Angle / Corner / Vertex*
- Segi Lima: *Pentagon*

## 6. Uji Ketangkasan (Soal & Penyelesaian)
**Soal 1:**
Gambarlah atau bayangkan sebuah buku tulis. Ada berapa banyak sisi dan sudut dari sampul buku itu?
**Penyelesaian 1:**
Buku tulis berbentuk segi empat. Karena itu pinggirannya (sisi) ada 4 buah, dan pojoknya (sudut) ada 4 buah.
