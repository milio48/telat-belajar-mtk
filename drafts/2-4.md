# Bab 4: Bangun Datar

## 1. Akar Konsep (Definisi & Deskripsi)

Bayangkan dirimu berjalan membentengi sebuah area petak sawah. Untuk menjaga petak sawah dari ancaman, kamu membangun pagar lurus secara bersinggung. Garis-garis pagar ini membelah satu dataran besar ruang rumput yang ada. Dalam dunia konsep, bidang inilah yang kita sebut dengan **Bangun Datar**. Ia tidak peduli tebal ketebalan tanahnya; yang dirisaukan bangun datar adalah keluasannya bidang tanahnya yang tersisa saat disorot garis ujung perbatasan pagar! 

Di dalam struktur Bangun Datar yang bersinggung-singgungan itu, terdapat dua unsur dasar terpenting:
1. **Sisi:** Garis pelindung pelindungnya yang mengelilingi daerah (Sama layaknya batangan pagar yang membentengi).
2. **Titik Sudut:** Tepat di pojok tempat pertemuan dinding dari pergeseran garis sisinya menyudut tajam atau tumpul ke sebuah patok pertemuan (Pojokan pagar tempat kayu paku dipakukan agar pagar berubah jalan di tikungan).

## 2. Pijakan Bumi (Contoh Dunia Nyata)

Geometri datar hadir untuk membentuk batasan zona agar lebih rapi saat memotong barang: 
* **Penyablon Baju:** Saat mencetak logo tulisan kotak bendera persis segi empat di tengah di dada depan. 
* **Pembuat Jendela Kaca Gedung Raksasa:** Jendela yang dipasang bisa terdiri dari bangun persegi berselang-seling pola yang ditutupi bingkai pinggir tepi penahannya angin.  
* **Pemandu Peta Darurat:** Melihat rambu jalur khusus berbentuk segitiga tajam yang membendung lalu lintas.

## 3. Mekanika Dasar (Praktek Perhitungan)

Kamu sekarang melangkah mengenal identitas keluarga bangun datar di sistem pengelompokannya, berdasarkan "Berapa banyak batangan pagar dan pos tikungannya".

**Keluarga Segitiga**
Kumpulan ini unik, dan selalu menuruti ciri minimalis mutlaknya:
* Minimal wajib memiliki persis $3$ Garis Sisi.
* Minimal wajib memiliki persis $3$ buah Titik Sudut/Pojokan.

**Keluarga Segi Empat (Persegi & Persegi Panjang)**
Kumpulan ini sangat dominan terlihat dirumah kita seperti ukuran layar ponsel dan dinding televisi:
* Tersusun pasti dari rangakaian persis $4$ Sisi Garis perbatasannya.
* Pasti tersusun berbelok-belok di rangakaian pertemuan $4$ Titik Sudut/Pojokan.

**Lingkaran (Kasus Spesial)**
Bangun ini sangat keras kepala dan unik karena memiliki sifat melingkar seperti cincin karet hulahup yang rata: 
* Hanya ada $1$ Sisi panjang lurus yang dibengkokkan secara lengkung tiada akhirnya memutar menutup.
* Memiliki NOL titi sudut ($0$). Ia tidak punya satupun pojokan karena pergerakannya lentur meluncur dan halus tanpa terputus patahan sama sekali.

## 4. Logika Komputasi (Implementasi Kode)

Bagi program animasi render permainan 2D dua-dimensi, logika penggambaran membutuhkan pendeteksian di setiap benturan jumlah garis di tikungannya agar grafis digambar seolah bergerak dan patah layaknya segitiga.

```javascript
// Database Objek Permainan
let databaseBentuk = [
  { benda: "Perisai Ksatria", sisi: 3, sudut: 3 },
  { benda: "Papan Lemari", sisi: 4, sudut: 4 },
  { benda: "Ban Kendaraan Truk", sisi: 1, sudut: 0 }
];

// Mesin Pemindai Karakterisasi Sudut (Pemilihan Otomatis Sifat)
for (let i = 0; i < databaseBentuk.length; i++) {
  let material = databaseBentuk[i];

  if (material.sudut === 3) {
    console.log(material.benda + " Didenfinisikan masuk kelompok SEGITIGA.");
  } else if (material.sudut === 4) {
    console.log(material.benda + " Didenfinisikan kelompok SEGI EMPAT stabil.");
  } else if (material.sudut === 0) {
    console.log(material.benda + " Terdeteksi sebagai LINGKARAN, sangat mulus.");
  }
}
```

## 5. Navigasi Global (Glosarium Inggris)

* **Flat Shape / Polygon:** Bangun Datar  (Poligon tertutup sisi)
* **Side:** Sisi (garis penyusun dataran)
* **Vertex / Corner:** Titik Sudut (Titik temu bengkokan)
* **Triangle:** Segitiga
* **Quadrilateral / Rectangle:** Segi Empat Persegi

## 6. Uji Ketangkasan (Soal & Penyelesaian)

**Soal 1:**
Andi diberikan $4$ helai sedotan oleh gurunya dan disuruh menyusun ujung dan ujung helai agar rapat terutup jadi sesuatu bentuk kurungan benda bangun datar tanpa sisa ada lebihan lubang yang terbuka. Bentuk keluarga bangun besar utama apa yang dihasilkan bangunan sedotannya Andi, dan hitung ada berapa ujung titik persendian antar batangnya?

**Penyelesaian:**
1. Karena ia diberi material sebanyak garis $4$ sedotan (ini merepresentasikan SISI garisnya berjumlah empat). 
2. Susunan tertutup ujung pangkal batang yang persis menggunakan $4$ sisi pembendung akan dengan mudah selalu berakhir membentuk keluarga kelompok **Segi Empat**.
3. Di persimpanggannya titik perpotongan, ada $4$ batang. maka setiap satu pasangan bertempu membentuk sudut. Alhasil jumlah di titik pertemuan (Titik Sudutnya) tentu ada sejumlah **$4$ buah  Titik Sudut**.
