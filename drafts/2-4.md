# Bab 4: Bangun Datar

## 1. Akar Konsep
Bangun datar adalah bidang tertutup dua dimensi rata yang terbentuk dari komposisi pertemuan garis bersudut. Karakteristik pinggiran garis sisinya pada bangun datar tersebut merepresentasikan batas fisik yang berwujud ibarat tepian presisi bidang selembar kertas yang halus.

## 2. Pijakan Bumi
- Pengrajin Ubin Keramik: Memotong kotak porselen mentah mengikuti panduan presisi ukuran bidang dasar bentuk persegi.
- Pembuat Potongan Rambu: Menyusun rambu tanda segitiga penunjuk arah pada lintasan persimpangan dan menjaga ketiga pinggiran besinya lurus.

## 3. Mekanika Dasar
Kita bisa mendefinisikan ragam entitas bangun datar tunggal berdasarkan pola kumpulan jumlah kepingan garis sisinya.
$\text{Segitiga} = 3 \text{ sisi}$
$\text{Segi Empat} = 4 \text{ sisi}$
Contoh spesifik pada penghitungan identitas jumlah lintasan pinggiran tepian selembar kertas standar:
$\text{Sisi Kertas Persegi Panjang} = 4$

## 4. Logika Komputasi
```javascript
function cekIdentitasBangun(totalSisi) {
  if (totalSisi === 3) {
    return "Segitiga";
  } else if (totalSisi === 4) {
    return "Segi Empat";
  }
  return "Tidak Diketahui";
}
```

## 6. Uji Ketangkasan
**Soal:** Sebuah layar monitor berbentuk persegi panjang tegak dan papan jalan dilarang masuk bentuknya membulat. Antara televisi dan plat rambu marka dilarang masuk jalan searah, mana yang sisinya punya wujud garis tepian lebih spesifik mirip empat potongan pinggiran kaku?
**Penyelesaian:**
1. Ketahui terlebih dahulu karakteristik dua buah dimensi objek fisik dalam identifikasi permasalahannya: layar televisi dan rambu plat arah.
2. Identifikasi formasi fisik layar televisi dan lihat jumlah sisi lurus bangunnya. Karena wujud layar datar itu dikepung pinggiran kaku batas simetri berbentuk persegi panjang, alhasil total jumlah sisinya eksak yaitu 4 garis sisi bersudut.
3. Rambu larangan tanpa siku adalah entitas bidang bentuk lingkaran bundar murni di mana garis lengkung melingkar absolutnya membuat formasi sudut lurus berjumlah nol. Maka opsi layarlah yang cocok berstatus segi empat spesifik geometri dua dimensi yang utuh kepingan batas pinggirannya.
