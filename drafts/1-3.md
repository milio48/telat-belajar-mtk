# Bab 3: Mengenal Bangun Datar dan Bangun Ruang Sederhana

## 1. Akar Konsep (Definisi & Deskripsi)
- **Bangun Datar**: Bayangkan bayangan sebuah benda yang jatuh di lantai; ia hanya memiliki panjang dan lebar, benar-benar pipih seperti stiker. Contoh: Segitiga, Segi empat, dan Lingkaran.
- **Bangun Ruang**: Ini adalah objek fisik yang bisa kamu genggam, memiliki volume atau "isi". Contoh: Kotak sepatu, bola, atau kaleng.

## 2. Pijakan Bumi (Contoh Dunia Nyata)
- **Desainer Grafis (Bangun Datar)**: Menggunakan lingkaran dan segitiga pipih di layar komputer untuk membuat sebuah logo.
- **Arsitek & Tukang Bangunan (Bangun Ruang)**: Membangun rumah berbentuk balok (kotak besar) dengan atap berbentuk prisma (segi tiga miring) yang nyata dan bisa dimasuki.

## 3. Mekanika Dasar (Praktek Perhitungan)
**Sifat Bangun Datar:**
- **Segitiga**: Memiliki 3 sisi bersambung.
- **Segi empat**: Memiliki 4 sisi bersambung.
- **Lingkaran**: Memiliki 1 sisi melengkung yang terus menyatu tanpa sudut.

**Mengelompokkan Bangun Ruang:**
- **Kubus / Balok**: Seperti kotak kado atau buku tebal.
- **Bola**: Bulat penuh seperti kelereng atau bola pimpong.
- **Tabung**: Seperti botol minum cincin atau kaleng susu.

## 4. Logika Komputasi (Implementasi Kode)
Dalam kode, kita dapat membuat daftar nama-nama bangun dan mengelompokkannya secara otomatis berdasarkan tipe garis/sisinya.

```javascript
function cekJenisBangunDatar(jumlahSisi) {
  if (jumlahSisi === 3) {
    return "Segitiga";
  } else if (jumlahSisi === 4) {
    return "Segi Empat";
  } else if (jumlahSisi === 0) {
    return "Lingkaran";
  } else {
    return "Bangun lainnya";
  }
}

console.log(cekJenisBangunDatar(3)); // Output: Segitiga
console.log(cekJenisBangunDatar(0)); // Output: Lingkaran
```

## 5. Navigasi Global (Glosarium Inggris)
- Bangun Datar: *2D Shapes (Two-Dimensional)*
- Bangun Ruang: *3D Shapes (Three-Dimensional)*
- Segitiga: *Triangle*
- Segi Empat: *Quadrilateral / Square*
- Lingkaran: *Circle*

## 6. Uji Ketangkasan (Soal & Penyelesaian)
**Soal 1:**
Benda apakah di rumahmu yang permukaannya berbentuk lingkaran dan pipih?
**Penyelesaian 1:**
Piring, koin uang logam, atau permukaan jam dinding. Semua itu adalah contoh implementasi bentuk lingkaran pipih (bangun datar) dalam keseharian.

**Soal 2:**
Ibu membeli sepatu baru. Kotak sepatunya termasuk bangun apa? Jika kotaknya digambar di atas selembar kertas pipih, bentuk apa yang akan tergambar?
**Penyelesaian 2:**
Kotak sepatu yang nyata adalah **Bangun Ruang** (Balok). Jika hanya digambar jejaknya di kertas pipih, bentuk bayangannya adalah **Bangun Datar** (Segi empat / Persegi Panjang).
