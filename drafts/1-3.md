# Bab 3: Mengenal Bangun Datar dan Bangun Ruang Sederhana

## 1. Akar Konsep
Bangun datar merupakan sebuah representasi objek geometri bernilai presisi dalam proyeksi ruang dimensi dua. Sebaliknya, bangun ruang membentuk komposisi figur geometris pejal dalam perhitungan batas ruang dimensi tiga secara utuh. Bangun ini ibarat rangka kertas yang membentuk pola potongan pipih, serta representasi sebuah kotak boks ruang bervolume spasial.

## 2. Pijakan Bumi
- Petugas rambu marka melukis papan peringatan jalan lalu-lintas yang menampilkan pola segitiga utuh.
- Kurir paket mengirimkan objek pesanan kemasan berbentuk kotak balok solid kepada alamat penerima.

## 3. Mekanika Dasar
Dimensi ruang dapat diidentifikasi hanya dari penghitungan variabel sisi pembatas geometri:
$$ N_{\text{sisi}} = n $$
Contoh: Bangun datar dasar jenis segitiga memiliki nilai sisi objek berjumlah $N_{\text{sisi}} = 3$. Bangun ruang balok terbentuk dengan konstruksi permukaan berjumlah komponen area $N_{\text{sisi}} = 6$.

## 4. Logika Komputasi
```javascript
function cekJenisGeometri(jumlahSisiDatar) {
  let deskripsiGeometri = "";
  if (jumlahSisiDatar === 3) {
    deskripsiGeometri = "Segitiga";
  } else if (jumlahSisiDatar === 4) {
    deskripsiGeometri = "Segi Empat";
  } else {
    deskripsiGeometri = "Bangun Tidak Dikenal";
  }
  return deskripsiGeometri;
}
console.log(cekJenisGeometri(4));
```

## 6. Uji Ketangkasan
Dalam tugas kesenian lipat kertas origami, ibu guru meminta para siswa sekolah untuk memotong jenis bangun dimensi dua yang tersusun atas tepat tiga potongan sisi perbatasan. Sebutkan nama spesifik klasifikasi bangun yang diinstruksikan oleh pendidik.

Penyelesaian:
1. Hitung jumlah syarat batasan pada instruksi soal adalah 3 buah sisi lurus komponen perbatasan.
2. Identifikasi bangun dimensi datar dengan atribut tepat berjumlah 3 garis komponen sisi keliling perbatasan.
3. Objek bangun geometris ini ditetapkan pada nomenklatur definisi segitiga dua dimensi ruang rata.
