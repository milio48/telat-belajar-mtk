# Bab 5: Bangun Datar dan Bangun Ruang

## Akar Konsep
Bangun datar adalah sebutan untuk bentuk bangun geometri dua dimensi yang memiliki luas hamparan dan letak batas keliling rentang besaran namun dimensi volume tak akan berbentuk dimensi kedalamannya. Bangun ruang berupa bentuk fisik bangun padat tiga dimensi parameter presisi perhitungannya mengandung batas ukuran isi ukuran volumenya. Bangun geometri ibarat cetakan potong kertas lembar lipatan bingkisan kardus persegi penyimpan paket balok pada toko kemasan belanja pengiriman. Letak jaring penyusunnya dapat direntang datar.

## Pijakan Bumi
- **Arsitek rumah tingkat:** Merancang rasio dimensi presisi luasan volume balok fondasi semen cor struktur ukuran perbandingan batas pinggir atap dimensi segitiga datar tumpuannya.
- **Drafter bangunan gedung:** Menempatkan letak ukir pilar bangunan silinder sejajar bersudut presisi atap kubah atas permukaan hamparan persegi pada pilar penyusun gedung strukturnya.

## Mekanika Dasar
Rumus menentukan luas jajar genjang ($L$) dari perkalian parameter ukuran letak garis sejajar mendatar alas tepi bawah ($a$) dan garis batas parameter per ukuran tinggi titik tarik puncaknya ($t$):
$L = a \times t$

Contoh perhitungan luas hampar bidang ukuran luasan nilai ukuran angka jajar genjang dimensi nilai pada dimensi sisi panjang $a = 5$ satuan ukur rasionya sisi rentangnya dengan nilai konversi ukuran $t = 3$:
$L = 5 \times 3 = 15$

## Logika Komputasi
```javascript
function hitungLuasBidang(alasDatarJajar, rentangTarikTinggi) {
  let takaranUkuranLuasan = alasDatarJajar * rentangTarikTinggi;
  return takaranUkuranLuasan;
}

console.log(hitungLuasBidang(5, 3));
```
