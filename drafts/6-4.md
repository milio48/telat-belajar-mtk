# Bab 4: Sistem Koordinat

## 1. Akar Konsep (Definisi & Deskripsi)
Anggaplah lantai kamarmu diberi ubin kotak-kotak catur tanpa batas dinding. "Titik 0 Pijakan" di bagian pusarnya kita beri jangkar palang dua garis tegak lurus mengiris (Sumbu Palang datar / Absis mendatar X) dan (Sumbu tiang Vertikal / Ordinat tegak lurus Y). Alamat titik mana kursor/karakter gamemu jalan, bertitik di 2 alamat: $(X \text{ sumbu datar}, Y \text{ sumbu lintang ketinggian})$.

## 2. Pijakan Bumi (Contoh Dunia Nyata)
- **Teknisi Radar Udara Pesawat**: Bandara membidik pesawat masuk layarnya tepat lewat pembagian sektor di koordinat radar $X$ lintang barat/timur $Y$ utara/selatanya pesawat asing melaju presisi di satelit.
- **Animator Game Engine (Unity/Unreal)**: Menaruh karakter pemainnya spawn meletakan pahlawan di posisi palang $(0,0)$ tengah layar monitor screen lalu menyetting sprite musuh peluru turun dari lintang atas sumbu Positif Ordinat $(X: 50, Y: 100)$. 

## 3. Mekanika Dasar (Praktek Perhitungan)
**Sumbu Koordinat $(X, Y)$ Kartesian René:** 
- Huruf $X$ di depan wajib dibaca duluan. Ini laju kereta lintasan datar mendatar di tanah lantai $(Kanan +, Kiri Negatif -)$
- Huruf $Y$ mendampingi di lapak kedua wajib terbaca selanjutnya ini indikator laju Lift tebing Naik/Turun Vertikal ketinggiannya ordinat $(Atas Positif +, bawah anjlok Minus -)$

**Misal ada titik $B (4, -2)$**. 
Artinya orang itu melangkah maju datar ke *Kanan Tanah* arah lurus $4$ ubin. Dan anjlok turun mengarah bawah tangga jurang (minus kebawah) $2$ lobang ubin tegak tiang.

## 4. Logika Komputasi (Implementasi Kode)
Setiap objek html / element kotak pixel canvas perhatikan, letak *absolut* style html code koordinat XY dasar di browser client web dom anda di programnya.
```javascript
class PetarungBidakan {
  constructor(awalSumbuDatarX, awalSumbuTegakY) {
    this.x = awalSumbuDatarX;
    this.y = awalSumbuTegakY;
  }
  
  // Game berjalan gerak Datar x Maju jalan kaki geser ke kanan ubin 
  bergerakMaju(speedUbinLangkah) {
      this.x += speedUbinLangkah;
      return `Koordinat Posisi Pemain Terkini Nempel pada Titik: X(${this.x}) dan Titik Y(${this.y})`;
  }
}
let mario = new PetarungBidakan(-2, 5);
console.log(mario.bergerakMaju(4)); 
// -2 maju langkah ditabrak 4 (+). jadinya Koordinat Posisi Pemain Berada di Titik X(2) , Titik Y(5).
```

## 5. Navigasi Global (Glosarium Inggris)
- Sumbu X: *X-Axis / Abscissa*
- Sumbu Y: *Y-Axis / Ordinate*
- Sistem Koordinat: *Cartesian Coordinate System*
- Titik Nol Tengah Asal: *Origin point*

## 6. Uji Ketangkasan (Soal & Penyelesaian)
**Soal 1:**
Sebutir debu peluru ditembak pelacak pada posisi awal tembakan di pangkal Titik ( $0, -3$ ). Artinya ia anjlok tiang minus 3 di garis vertikal sumbunya Y tegak lurus pualam layar. Tiba tiba debu rudal bermanuver melesat NAIK Lift mendadak tiang lurus atas ke arah menembus garis $+8$ Ubin vertikal. Berada di titik ordinat Y manakah ia mendarat hinggap sekarang (X nya Abaikan tetap saja letaknya statis diam di nol pangkal tak berbelok mendatar tanah)?
**Penyelesaian 1:**
Dari Y ($ -3 $) Minus tiga tebing anjlok tiang.
Ditambahkan (+) melontar pergerakan $ 8 $ ubin atas vertikal ordinat tinggi.
$-3 + 8 = 5$ 
Debu terparkir sekarang menempel diam titik pangkal baru $(X= 0, \text{ dan tegaknya Ordinat } Y \text{ nempel di atas level } 5 \text{ ubin})$
Posisi baru mutlak: $(0, 5)$.
