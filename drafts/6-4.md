# Bab 4: Sistem Koordinat

## 1. Akar Konsep
Sistem koordinat Kartesius adalah metode penentuan posisi titik menggunakan dua sumbu yang tegak lurus, yaitu sumbu mendatar (X) dan sumbu tegak (Y). Letak titik ini ibarat membaca peta digital yang menggunakan garis lintang dan bujur untuk menemukan lokasi spesifik.

## 2. Pijakan Bumi
- **Pengatur lalu lintas udara:** Memantau posisi pesawat pada radar dua dimensi untuk mengatur jalur penerbangan yang aman.
- **Pengembang perangkat lunak:** Menentukan letak karakter dalam permainan video secara presisi pada layar monitor.

## 3. Mekanika Dasar
Sistem penulisan koordinat ditandai dengan pasangan angka yang diapit tanda kurung:
$(x, y)$

Keterangan:
$x$ = Nilai pada sumbu mendatar (Absis)
$y$ = Nilai pada sumbu tegak (Ordinat)

Contoh posisi:
Titik koordinat $(3, 4)$ berarti posisinya berjarak 3 langkah ke arah kanan dari titik pusat sejajar sumbu X, dan 4 langkah ke arah atas sejajar sumbu Y.

## 4. Logika Komputasi
```javascript
function hitungJarakManhattan(posisiAwalX, posisiAwalY, posisiAkhirX, posisiAkhirY) {
  const selisihX = Math.abs(posisiAwalX - posisiAkhirX);
  const selisihY = Math.abs(posisiAwalY - posisiAkhirY);
  
  const jarakTotal = selisihX + selisihY;
  
  return jarakTotal;
}

console.log(hitungJarakManhattan(1, 2, 4, 6)); // Output: 7
```

## 6. Uji Ketangkasan
**Soal:**
Sebuah robot mainan berada di titik pangkal koordinat $(0, 0)$. Robot tersebut melaju 5 satuan ke arah sumbu X positif, kemudian bergerak 3 satuan ke arah sumbu Y negatif. Di titik koordinat manakah posisi robot tersebut sekarang?

**Penyelesaian:**
1. Titik mula-mula robot adalah di posisi absis $x = 0$ dan posisi ordinat $y = 0$.
2. Pergerakan pertama adalah ke arah sumbu X positif, sehingga nilai absis bertambah lima satuan: $0 + 5 = 5$.
3. Pergerakan kedua adalah ke arah sumbu Y negatif, sehingga nilai ordinat berkurang tiga satuan: $0 - 3 = -3$. Posisi robot sekarang adalah $(5, -3)$.
