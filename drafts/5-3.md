# Bab 3: Pecahan, Persen, dan Skala

## Akar Konsep
Pecahan, persen, dan bilangan desimal merupakan wujud matematis untuk menyatakan porsi dari sebuah nilai yang utuh. Skala adalah rasio perbandingan yang menunjukkan representasi pembesaran atau penyusutan ukuran objek benda. Skala ibarat ukuran wujud dari miniatur model mainan mobil yang dirancang persis dengan cetakan mobil dari pabrik aslinya.

## Pijakan Bumi
- **Manajer Pemasaran Ritel:** Menghitung potongan diskon barang dagangan menggunakan pecahan proporsi persen untuk merangsang penjualan.
- **Arsitek Sipil:** Menggambar rancangan konstruksi jalan layang kota secara akurat memakai hitungan skala peta pengecilan kertas.

## Mekanika Dasar
Pecahan murni dapat diubah menjadi bentuk desimal dan dilanjutkan menjadi persen berbasis bilangan ratusan.
Konversi pecahan $\frac{1}{4}$ menjadi persentase:
$\frac{1}{4} \times 100\% = \frac{100\%}{4} = 25\%$
Operasi dari perkalian pecahan dilakukan dengan mengalikan pembilang saling tindih dan penyebut saling kalikan:
$\frac{1}{3} \times \frac{2}{4} = \frac{1 \times 2}{3 \times 4} = \frac{2}{12} = \frac{1}{6}$

## Logika Komputasi
```javascript
function ubahKePersen(angkaPembilang, angkaPenyebut) {
    const nilaiDesimal = angkaPembilang / angkaPenyebut;
    const nilaiPersen = nilaiDesimal * 100;
    
    return nilaiPersen + "%";
}

const potonganDiskon = ubahKePersen(1, 4);
console.log(potonganDiskon); // Output: "25%"
```

## Uji Ketangkasan
**Soal 1:** Seorang kontraktor melihat gambar jalan tol pada peta dengan panjang $15$ cm. Skala peta tertulis $1 : 2000$. Berapakah panjang asli jalan tersebut dalam meter?

**Penyelesaian:**
1. Gunakan rumus skala[^1]: Panjang Asli $= \text{Panjang Peta} \times \text{Penyebut Skala}$.
2. Hitung dalam sentimeter: $15 \times 2000 = 30.000$ cm.
3. Konversi ke meter (dibagi 100): $30.000 \div 100 = 300$ meter. Panjang asli jalan adalah **300 meter**.

---

**Soal 2:** Sebuah toko memberikan diskon $20\%$ untuk sepatu seharga Rp250.000. Berapa harga yang harus dibayar setelah diskon?

**Penyelesaian:**
1. Hitung nilai diskon: $\frac{20}{100} \times 250.000 = 50.000$.
2. Kurangkan dari harga awal: $250.000 - 50.000 = 200.000$.
3. Harga setelah diskon adalah **Rp200.000**.

---

[^1]: **Skala Peta:** Perbandingan antara ukuran pada peta/gambar dengan ukuran sesungguhnya. Skala $1:2000$ berarti 1 cm pada peta mewakili 2000 cm (20 meter) di dunia nyata.
