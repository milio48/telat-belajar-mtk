# Bab 2: Pengukuran (Volume, Waktu, dan Debit)

## 1. Akar Konsep (Definisi & Deskripsi)
"Debit" aliran masuk cairan adalah kasta tertinggi dari sekedar konsep Kecepatan mobil di bab lalu. Kalau kecepatan mobil = Jarak dibagi Waktu tempuh; **Debit** mobil tanki tirta = Volume cairan tumpahan yang dimuntahkan membanjiri dibagi Durasi Waktu menumpahkannya per detik jamnya. Semacam daya dorong semprot selang airnya berkapasitas tembak seberapa besar memompa cairan tumpahnya ke wadah bak lain per menit detiknya!

## 2. Pijakan Bumi (Contoh Dunia Nyata)
- **Petugas PDAM Pompa Air**: Wajib tahu hitungan Debit aliran pipa pompa agar bak reservoir kota penuh terisi jam 4 pagi sebelum semua warga senegara buka keran sink wastafel menyedot airnya.
- **Petugas SPBU Bensin**: Moncong tangki pengisi tangki bensin motor, memiliki setelan ukur meter debit aliran curah bensin (Liter per Detik).

## 3. Mekanika Dasar (Praktek Perhitungan)
**Debit Aliran Cairan:**
- $\text{Debit} = \frac{\text{Volume}}{\text{Waktu}}$

**Konversi Waktu Presisi Menit Detik:**
Misal: $2,5 \text{ Jam} = 2 \text{ jam}$ penuh + $0,5 \text{ jam}$ ($\frac{1}{2}$ jam / 30 menit).
Jadi durasinya $2 \text{ jam } 30 \text{ menitata}$.

**Contoh Kasus Debit:**
Keran kencang sanggup memuntahkan $10$ liter tirta ke luar mulut pipanya dalam 2 detik. 
Debitnya = $10 \div 2 = 5$ Liter / per-detik moncong bukaan semburan konstan!

## 4. Logika Komputasi (Implementasi Kode)
Sebuah smart-meter PAM pamakaian air mendeteksi pemakaian bocor warga!
```javascript
function cekKebocoran(volumeAirKeluarsaatMalam, durasiNgalirDetik) {
  let debitLiterDetik = volumeAirKeluarsaatMalam / durasiNgalirDetik;
  
  if (debitLiterDetik > 2){
    return "ALARM BAHAYA PIPA BOCOR DERAS KERAS !";
  }
  return "Posisi Pipa Wajar normal Rembes Cuma Netes";
}

// Keluar 60 Liter ngalir diam diam selama rembes per 10 menit dimalam!
console.log(cekKebocoran(60, 10)); // return ALARM BAHAYA PIPA BOCOR karena bocor debitnya 6 liter per sekon menitis!
```

## 5. Navigasi Global (Glosarium Inggris)
- Debit Air Aliran: *Flow Rate / Volumetric Flow*
- Liter per Detik: *Liters per Second*
- Debit curah/Muntahan: *Discharge Rate*

## 6. Uji Ketangkasan (Soal & Penyelesaian)
**Soal 1:**
Sebuah keran ember menyala air pancurannya dan mengisi Penuh ember kapasitas 20 liter, waktu menunggunya cukup ditahan diam aja selama 5 menit. Berapakah debit pengisian curah keran tersebut per menitnya lurus ?
**Penyelesaian 1:**
Rumus: $\text{Volume} \div \text{Waktu}$.
$20 \text{ L} \div 5 \text{ Menit} = 4$.
Debit kecepatan daya sembur pompa kerannya adalah $4$ Liter dalam setiap perputaran waktu $1$ Menit!
