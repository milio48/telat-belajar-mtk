# Bab 2: Pengukuran (Volume, Waktu, dan Debit)

## 1. Akar Konsep
Debit adalah ukuran kecepatan aliran zat cair, yaitu seberapa banyak volume cairan yang melewati suatu tempat dalam satuan waktu tertentu. Debit ibarat mengukur seberapa cepat air keluar dari selang mesin pompa untuk mengisi sebuah tabung tangki air dalam hitungan menit.

## 2. Pijakan Bumi
- **Petugas PDAM:** Memantau kapasitas pasokan air yang mengalir ke perumahan warga setiap jam.
- **Perawat medis:** Mengatur kecepatan tetesan cairan infus pasien agar habis sesuai dengan tenggat waktu yang ditentukan dokter.

## 3. Mekanika Dasar
Rumus perhitungan debit:
$Q = \frac{V}{t}$

Keterangan:
$Q$ = Debit (volume per waktu)
$V$ = Volume
$t$ = Waktu

Contoh perhitungan:
Jika volume air sebanyak $10$ liter mengalir dalam waktu $2$ detik, maka debit aliran airnya:
$Q = \frac{10}{2} = 5$ liter/detik.

## 4. Logika Komputasi
```javascript
function hitungDebit(volume, waktu) {
  if (waktu === 0) {
    return 0; // Menghindari pembagian dengan nol
  }
  const nilaiDebit = volume / waktu;
  return nilaiDebit;
}

const volumeAir = 10; // liter
const waktuAlir = 2; // detik
console.log(hitungDebit(volumeAir, waktuAlir)); // Output: 5
```

## 6. Uji Ketangkasan
**Soal:**
Sebuah kolam renang memiliki muat volume air sebanyak $600$ liter. Kolam tersebut dikosongkan menggunakan mesin pemompa air dan habis terkuras dalam waktu $120$ menit. Berapakah nilai debit air yang keluar dari pompa tersebut dalam satuan liter/menit?

**Penyelesaian:**
1. Identifikasi nilai konstanta yang diketahui: volume ($V$) = $600$ liter, waktu ($t$) = $120$ menit.
2. Gunakan rumus persamaan debit: $Q = \frac{V}{t}$. Masukkan nilai menjadi $Q = \frac{600}{120}$.
3. Lakukan operasi pembagian untuk mendapatkan hasil nilai akhir sebesar $5$ liter/menit.
