# Bab 5: Pengumpulan, Pengolahan, dan Penyajian Data

## 1. Akar Konsep
Pengolahan data adalah proses mengumpulkan, menyusun, dan menganalisis kumpulan data mentah untuk menghasilkan informasi matematis. Kegiatan ini ibaratnya mengelompokkan secara teratur sekumpulan struk belanja agar bisa dihitung pengeluaran rata-rata bulanannya.

## 2. Pijakan Bumi
- **Tenaga medis:** Mencatat hasil suhu tubuh harian pasien pada tabel rawat inap untuk melihat perkembangan kondisi pasien.
- **Manajer penjualan:** Menentukan model produk sepatu yang paling banyak terjual untuk memperkirakan ukuran stok tambahan di gudang.

## 3. Mekanika Dasar
Rumus nilai rata-rata (Mean):
$\bar{x} = \frac{\Sigma x_i}{n}$

Keterangan:
$\bar{x}$ = Rata-rata dari nilai data.
$\Sigma x_i$ = Jumlah gabungan semua anggota nilai data penelitian.
$n$ = Banyaknya jumlah data penelitian.

Modus adalah nilai kelompok pengamatan yang paling sering muncul berulang pada data rekaman populasi.

Contoh perhitungan nilai rata-rata deret kumpulan ukuran $\{2, 4, 6\}$:
$\bar{x} = \frac{2 + 4 + 6}{3} = \frac{12}{3} = 4$.

## 4. Logika Komputasi
```javascript
function hitungRataRata(kumpulanData) {
  let totalData = 0;
  const jumlahElemen = kumpulanData.length;
  
  for (let i = 0; i < jumlahElemen; i++) {
    totalData += kumpulanData[i];
  }
  
  const nilaiRataRata = totalData / jumlahElemen;
  return nilaiRataRata;
}

const daftarNilaiTugas = [2, 4, 6];
console.log(hitungRataRata(daftarNilaiTugas)); // Output: 4
```

## 6. Uji Ketangkasan
**Soal:**
Toko kopi mencatat jumlah penjualan pesanan gelas kopi miliknya selama empat hari operasional secara berurutan: $8, 5, 5, 10$. Tentukan nilai rata-rata hasil penjualan gelas kopi per hari dari toko kopi tersebut beserta nilai perhitungan modus dari transaksinya!

**Penyelesaian:**
1. Urutkan daftar rentang nilai himpunan rekam ukuran data secara urut yaitu $\{5, 5, 8, 10\}$. Temukan angka data yang muncul ukurannya paling banyak pencatatan, yaitu $5$. Ini merupakan nilai kemunculan modus datanya.
2. Hitung ukuran penjumlahan gabungan total nilai hitungan yang ada dari observasi data: $5 + 5 + 8 + 10 = 28$.
3. Gunakan rumus nilai ukuran perhitungan rasio rata-rata komputasi pembagian hasil observasi dengan ukuran rincian hitung jumlah harinya: $28 \div 4 = 7$ porsi per harinya.
