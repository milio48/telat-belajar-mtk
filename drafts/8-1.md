# Bab 1: Persamaan Garis Lurus

## 1. Akar Konsep
Persamaan garis lurus adalah suatu persamaan aljabar linear dengan dua variabel yang apabila digambarkan pada bidang koordinat Kartesius akan membentuk sebuah garis lurus. Persamaan ini menyatakan hubungan proporsional konstan antara dua variabel tersebut, yang diukur melalui kemiringan atau gradien. Konsep persamaan garis lurus ibarat lintasan kereta api yang membentang lurus menghubungkan dua stasiun dengan tingkat kemiringan lintasan yang stabil.

## 2. Pijakan Bumi
* **Arsitek:** Menggunakan konsep kemiringan untuk merancang tingkat kelandaian atap rumah agar air hujan dapat mengalir secara optimal.
* **Insinyur Sipil:** Menerapkan perhitungan gradien untuk menentukan tingkat kemiringan jalan pegunungan yang aman bagi arus transportasi kendaraan.

## 3. Mekanika Dasar
Persamaan umum garis lurus dinyatakan sebagai:
$$ y = mx + c $$
Di mana $m$ adalah gradien (kemiringan) dan $c$ adalah titik potong sumbu Y. Gradien yang melalui dua titik $(x_1, y_1)$ dan $(x_2, y_2)$ dihitung menggunakan rumus:
$$ m = \frac{y_2 - y_1}{x_2 - x_1} $$
Contoh: Diketahui titik P(1, 2) dan titik Q(3, 8).
$m = \frac{8 - 2}{3 - 1} = \frac{6}{2} = 3$.

## 4. Logika Komputasi
```javascript
function hitungNilaiGradien(titikX1, titikY1, titikX2, titikY2) {
  const selisihSumbuY = titikY2 - titikY1;
  const selisihSumbuX = titikX2 - titikX1;
  
  if (selisihSumbuX === 0) {
    return "Garis vertikal, gradien tidak terdefinisi";
  }
  
  const kemiringanGaris = selisihSumbuY / selisihSumbuX;
  return kemiringanGaris;
}

console.log(hitungNilaiGradien(1, 2, 3, 8));
```

## 6. Uji Ketangkasan
Sebuah kolam renang diisi air dengan laju konstan konstan. Volume awal air adalah 100 liter. Setiap jam, jumlah volume bertambah sebanyak 50 liter. Berapakah volume air dalam kolam setelah 4 jam?

1. Tentukan fungsi linear volume: Misalkan fungsi waktu adalah $x$ jam dan volume adalah $y$. Persamaannya menjadi $y = 50x + 100$.
2. Substitusi ukuran waktu ke persamaan: Masukkan $x = 4$ ke dalam rumus. $y = 50(4) + 100$.
3. Hitung hasil perkalian akhir: $y = 200 + 100 = 300$. Volume air kolam tersebut mencapai 300 liter.
