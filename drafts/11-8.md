# Bab 8: Turunan (Diferensial)

## 1. Akar Konsep
Turunan fungsi adalah ukuran laju perubahan sesaat suatu fungsi terhadap variabelnya. Secara geometris, turunan di suatu titik sama dengan kemiringan garis singgung[^1] kurva di titik tersebut. Turunan ibarat speedometer mobil — speedometer tidak mengukur jarak total, melainkan kecepatan sesaat pada detik itu juga.

> **Catatan:** Turunan adalah kebalikan dari integral. Jika turunan memberitahu laju perubahan, integral menghitung total akumulasi perubahan.

## 2. Pijakan Bumi
- **Ekonom Pasar Saham:** Menghitung laju perubahan sesaat indeks saham untuk menentukan apakah tren sedang naik atau turun.
- **Insinyur Otomotif:** Menghitung percepatan sesaat kendaraan dari fungsi kecepatan terhadap waktu.

## 3. Mekanika Dasar
Aturan pangkat turunan:
Jika $f(x) = ax^n$, maka $f'(x) = anx^{n-1}$.

Contoh: Tentukan turunan dari $f(x) = 3x^4$.
$f'(x) = 3 \times 4 \times x^{4-1} = 12x^3$.

Turunan konstanta: $f(x) = c \implies f'(x) = 0$.

## 4. Logika Komputasi
```javascript
function hitungTurunan(koefisien, pangkat) {
  // Aturan pangkat: ax^n → anx^(n-1)
  if (pangkat === 0) return { koefisien: 0, pangkat: 0 };

  const koefisienBaru = koefisien * pangkat;
  const pangkatBaru = pangkat - 1;

  return { koefisien: koefisienBaru, pangkat: pangkatBaru };
}

const hasil = hitungTurunan(5, 3);
console.log(`${hasil.koefisien}x^${hasil.pangkat}`);
// Output: "15x^2"
```

## 6. Uji Ketangkasan

**Soal 1:**
Lintasan gerak sebuah mobil digambarkan oleh fungsi jarak $s(t) = 5t^2 + 2t - 4$ (meter), dengan $t$ dalam detik. Tentukan kecepatan sesaat pada $t = 3$ detik.

**Penyelesaian:**
1. Kecepatan sesaat adalah turunan pertama fungsi jarak: $v(t) = s'(t)$.
2. Hitung turunan: $v(t) = 10t + 2$.
3. Substitusi $t = 3$: $v(3) = 10(3) + 2 = 32$ m/s. Kecepatan sesaat pada detik ke-3 adalah **32 m/s**.

---

**Soal 2:**
Tentukan turunan dari $f(x) = x^3 - 6x^2 + 9x + 1$. Pada nilai $x$ berapa gradien kurva sama dengan nol?

**Penyelesaian:**
1. Hitung turunan: $f'(x) = 3x^2 - 12x + 9$.
2. Setarakan dengan nol: $3x^2 - 12x + 9 = 0$, sederhanakan menjadi $x^2 - 4x + 3 = 0$.
3. Faktorkan: $(x - 1)(x - 3) = 0$, maka $x = 1$ atau $x = 3$. Gradien kurva bernilai nol di **$x = 1$ dan $x = 3$**.

---

[^1]: **Garis Singgung:** Garis lurus yang menyentuh kurva tepat di satu titik dan memiliki kemiringan yang sama dengan kurva di titik tersebut. Kemiringan garis singgung di titik $x = a$ adalah $f'(a)$.
