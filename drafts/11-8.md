# Bab 8: Turunan (Diferensial)

## 1. Akar Konsep
Turunan fungsi adalah proses penentuan tingkat perubahan waktu laju instan sesaat dari suatu nilai parameter perputaran fungsional turunan besaran yang berbanding lurus dari posisi fungsional variabel hitung komputasinya ketika margin rasionya digeser sepersekian yang amat sangat sempit nilainya. Turunan ibarat fungsi dari alat ukur jarum speedometer pada peraga konsol mekanika dasbor mobil. Angka kelajuan tempuh tempuh jarak dihitung mutlak berdasar posisi rasio perpindahan panjang ukuran jarak dari parameter limit satuan perubahan kecil rentang waktu waktu berjalan instannya mutlak.

## 2. Pijakan Bumi
- **Ekonom Pasar Berjangka:** Mengkalkulasi nilai perubahan angka tren margin turunan tingkat laju peningkatan rasio naik atau margin turun merosotnya grafik angka indeks margin keuntungan bursa saham yang mutlak komposit aljabarnya secara sesaat rentang sempit margin.
- **Ahli Rancang Mekanik (Insinyur Otomotif):** Menghitung porsi rasio ukur percepatan koefisien tenaga pacu maksimum rasio limit akselerasi margin mekanika dari rute laju konversi energi batas margin kecepatan gerak gesek rem dinamis pada fungsi fiktif parameter rasio kampas rem rem.

## 3. Mekanika Dasar
Formula aturan dari pangkat dasar modifikasi fungsional parameter variabel mutlak aljabar kalkulus bentuk fiktif turunannya (notasi Leibniz/Newton $d/dx$ atau $f'$):
Jika bentuk fungsi aljabarnya $f(x) = ax^n$, maka fungsi turunannya berwujud manifestasi konversi pangkat mutlaknya $f'(x) = an \cdot x^{n-1}$.

Contoh eksekusi porsi pemetaan turunan limit dari parameter fungsional kurva: Tentukan modifikasi wujud turunan dari kurva $f(x) = 3x^4$.
Mekanika operasinya mutlak pangkat turunan adalah kalikan 4 menyeberang silang ke depan ke unsur 3, lalu bentuk wujud modifikasi sisa unsur rasio eksponen pangkat dikurang letak nilai mutlak satu.
Modifikasi aljabarnya murni menjadi: $f'(x) = 12x^3$.

## 4. Logika Komputasi
```javascript
function hitungTurunanDasar(titikKonstanta, titikPangkat) {
  // Hanya berlaku untuk turunan fungsi ax^n
  if (titikPangkat === 0) return { konstantaBaru: 0, pangkatBaru: 0 };
  
  const hasilKonstantaAkhir = titikKonstanta * titikPangkat;
  const hasilPangkatTurunan = titikPangkat - 1;
  
  return {
    konstantaBaru: hasilKonstantaAkhir,
    pangkatBaru: hasilPangkatTurunan
  };
}

// Simulasi turunan dari fungsi polinomial 5x^3
const modifikasiDerivatifBaru = hitungTurunanDasar(5, 3);
// Format teks log untuk membaca rasionya
console.log(`${modifikasiDerivatifBaru.konstantaBaru}x^${modifikasiDerivatifBaru.pangkatBaru}`); // Output: "15x^2"
```

## 6. Uji Ketangkasan
**Soal:**
Laju pergerakan lintasan melaju fungsional jarak tempuh dari porsi meluncurnya laju mutlak dari lari balapan mobil Formula digambarkan lewat porsi batas wujud manifestasi geraknya pada rasio persamaan $s(t) = 5t^2 + 2t - 4$ (di mana $s(t)$ fungsional adalah dalam ukuran jarak meter fiktif, sedangkan rute koefisien besaran fungsional komposit margin $t$ dihitung berdasar waktu limit s detik). Tentukan ukuran rasio parameter percepatannya tepat kelajuannya dari limit pada saat menunjuk limit margin porsi tepat detik waktu kecepatan di $t = 3$ detik!

**Penyelesaian:**
1. Rangkum konsep mutlak diferensiasinya fungsional: rute komposit laju kelajuan percepatan murni instan akselerasi sesaat tersebut $(v(t))$ didapatkan rasionya lewat pencarian bentuk murni ukuran diferensial bentuk awal hasil fungsi pertama turunan langsung diferensial kalkulusnya $(v(t) = s'(t))$.
2. Terapkan perhitungan pemetaan pangkat mekanika mutlak turunan dasarnya turunan $v(t) = 10t + 2$.
3. Masukkan substitusi angka letak fungsional komputasi batas margin waktu kelajuan saat detik limit mutlak $t = 3$ fiktif rasio. Hitungan lajunya modifikasi fungsional menjadi limit margin kalkulasi kecepatan $v(3) = 10(3) + 2 = 32$. Secara mutlak murni kesimpulannya kelajuan balapan dari lari sesaat mesin pada batas limit 3 detik rasio kelajuan tersebut menyentuh batas murni 32 m/s.
