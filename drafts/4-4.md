# Bab 4: Pengukuran (Sudut, Waktu, Panjang, Berat, Kuantitas)

## 1. Akar Konsep (Definisi & Deskripsi)
Sudut ibarat kemiringan bukaan pintu. Jika tertutup rapat maka 0 derajat. Waktu, panjang, berat sudah kita pelajari; tambahan terbarunya adalah "kuantitas" yaitu mengelompokan benda per lusin (12), atau per gross (144) sebagai hitungan satuan pedagang.

## 2. Pijakan Bumi (Contoh Dunia Nyata)
- **Pedagang Grosir**: Belanja piring tak beli per satuan, melainkan per "lusin" (1 lusin = 12 buah). 
- **Nahkoda Kapal Laut**: Melihat derajat kompas haluan sudut kapal agar tak menabrak pulau/karang.

## 3. Mekanika Dasar (Praktek Perhitungan)
**Mengukur Sudut:**
Kita pakai busur setengah lingkaran.
0 sampai 89 derajat = Sudut lancip.
90 derajat = Sudut Siku-Siku tegak lurus.
Di atas 90 derajat = sudut Tumpul.

**Konversi Kuantitas:**
- 1 Lusin = 12 buah
- 1 Rim = 500 lembar (kertas foto copy)

## 4. Logika Komputasi (Implementasi Kode)
Game simulasi balap butuh sensor belokan stir kendaraan per derajat putaran roda.
```javascript
function cekJenisTikungan(derajatBerbelok) {
  if (derajatBerbelok === 90) return "Siku-Siku Tajam";
  if (derajatBerbelok < 90) return "Belokan Lancip";
  return "Belokan Landai Lebar";
}
console.log(cekJenisTikungan(120)); // Output: "Belokan Landai Lebar"
```

## 5. Navigasi Global (Glosarium Inggris)
- Sudut: *Angle*
- Busur Derajat: *Protractor*
- Siku-siku: *Right Angle*
- Kuantitas Lusin: *Dozen*

## 6. Uji Ketangkasan (Soal & Penyelesaian)
**Soal 1:**
Ibu membeli 3 lusin piring plastik. Berapa buah total piring yang dibeli Ibu?
**Penyelesaian 1:**
$3 \text{ lusin} \times 12 \text{ buah} = 36$ buah piring total.
