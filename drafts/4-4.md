# Bab 4: Pengukuran (Sudut, Waktu, Panjang, Berat, Kuantitas)

## Akar Konsep
Pengukuran adalah proses menentukan nilai besaran fisis (seperti panjang, massa, dan jarak) terhadap standar satuan baku yang terdefinisi dengan jelas ketetapan ukurnya. Praktik pengukuran ibarat menyeimbangkan beban barang timbangan logam ukuran anak timbangan di pasar dengan kuantitas masa massa tumpukan daging cincang sayur mayur buah kepingan buah yang hendak dibelinya.

## Pijakan Bumi
- **Penjahit pakaian pesanan:** Mengukur batas konversi presisi dimensi bentuk lurus panjang pergelangan tepi ukuran perbandingan struktur badan pemesan pelanggan di toko kainnya.
- **Pilot kemudi jet terbang:** Mengkonversi takaran parameter batas waktu batas lintasan parameter rotasi perhitungan skala besaran deret kompas ke letak lokasi landasan laju tujuannya.

## Mekanika Dasar
Rumus menentukan satu putaran penuh nilai derajat busur:
$\text{Sudut Putaran Lurus} = 180^{\circ}$
$\text{Sudut Putaran Penuh} = \text{Sudut Putaran Lurus} \times 2 = 360^{\circ}$

Contoh perhitungan dengan angka 180:
$\text{Sudut Putaran Penuh} = 180^{\circ} \times 2$
$\text{Sudut Putaran Penuh} = 360^{\circ}$

## Logika Komputasi
```javascript
function hitungPutarSudut(batasLingkar, parameterLurus) {
  let takaranUkuranSudut = batasLingkar * parameterLurus;
  return takaranUkuranSudut;
}

console.log(hitungPutarSudut(180, 2));
```
