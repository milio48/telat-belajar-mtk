# Bab 3: Pecahan Lanjutan

## 1. Akar Konsep (Definisi & Deskripsi)
- **Pecahan Senilai**: Ibarat kaca pembesar. Pecahan bisa tampak "bedanya angka". Padahal proporsinya sama persis. Potongan $\frac{1}{2}$ kue sama volumenya dengan $\frac{2}{4}$ kue.
- **Menyederhanakan Pecahan**: Menemukan bentuk paling dasar/simpel yang angka dari atas (pembilang) dan bawah (penyebut) sudah tidak bisa "dibagi oleh angka FPB" secara sama besar mentok lagi.

## 2. Pijakan Bumi (Contoh Dunia Nyata)
- **Penjahit Jas**: Menyatukan kain dengan rasio penukuran presisi. Jika pola awal $\frac{2}{4}$ meter lalu ada pembuangan kain/penghematan diputar balik logikanya maka setara memakai $\frac{1}{2}$ meter (setengah).
- **Resep Adonan**: Chef tahu kalau menu masak 1 loyang perlu 1 liter susu, lalu mau buat porsi mini. Resep yang tadinya seperdelapan ( $\frac{4}{32}$ kg) tepung, sangat rumit, Chef bisa sederhanakan resep ke rasio simpel ( $\frac{1}{8}$ kg ) agar dimengerti juru masak pemula.

## 3. Mekanika Dasar (Praktek Perhitungan)
**Pecahan Senilai & Sederhana:**
Cara mengecilkan. Bagi bagian atas (2) dan bawah (4), masing2 dengan angka sama FPB.
Misal $\frac{2}{4}$. Atas dan bawah sama sama punya bilangan dibagi *2*. Menjadi pecahan $\frac{1}{2}$.
**Penjumlahan Pecahan Penyebut Sama:**
Hanya pembilangnya (yang di posisi atas) yang boleh dijumlah.
$\frac{2}{5} + \frac{1}{5} = \frac{3}{5}$.

*(Karena pembawah / penyebut ibarat menyatakan identitas jenis potonganya, kamu nggak boleh nambah-nambahin identitas loyangnya)*

## 4. Logika Komputasi (Implementasi Kode)
Berikut algoritma yang memakai FPB tadi untuk otomatis mengecilkan fraction ratio form:

```javascript
// Memanggil ulang fungsi cariFPB yang ada di Bab sebelumnya
function cariFPB(a, b) {
  if (b === 0) return a;
  return cariFPB(b, a % b);
}

function sederhanakanPecahan(atas, bawah) {
  const fpbBagi = cariFPB(atas, bawah);
  return `${atas / fpbBagi}/${bawah / fpbBagi}`;
}

console.log(sederhanakanPecahan(4, 32)); // Output: "1/8"
```

## 5. Navigasi Global (Glosarium Inggris)
- Pecahan Senilai: *Equivalent Fractions*
- Menyederhanakan Pecahan: *Simplifying Fractions*
- Pecahan Dasar Tereduksi: *Irreducible Fractions / Lowest Terms*

## 6. Uji Ketangkasan (Soal & Penyelesaian)
**Soal 1:**
Tiga per delapan botol air dituang ditambah lagi dengan dua per delapan air dari botol lainnya. Totalnya?
**Penyelesaian 1:**
Karena penyebut sudah setara, fokus ke atas:
$\frac{3}{8} + \frac{2}{8} = \frac{5}{8}$ botol tercampur.

**Soal 2:**
Dapatkan pecahan senilai teringkas bagi $\frac{6}{12}$ !!
**Penyelesaian 2:**
FPB untuk 6 dan 12 adalah *6*. 
Bagi atas dan pembawah dengan 6.
$\frac{6}{12} = \frac{6 \div 6}{12 \div 6} = \frac{1}{2}$.
Pecahan terkecil adalah $\frac{1}{2}$ (Setengah).
