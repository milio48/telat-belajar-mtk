## 3. Mekanika Dasar
Rumusan perhitungan pembagian formulasi penyelesaian ukuran rata-rata mean:
$$ \bar{x} = \frac{\sum_{i=1}^{n} x_i}{n} $$
Contoh rasio besaran parameter rentang populasi rata jumlah hitung data ukuran sampel: Terdapat susunan seri baris pengamatan kumpulan tiga data observasi ukur bilangan himpunan kuantitatif pencatatan berukuran nominal 4, 6, dan 8.
Agregasi hasil nominal parameter kalkulasi gabung hitungan total penjumlahan semua besaran elemen seluruh komponen ukuran unit kumpulan: 4 + 6 + 8 = 18.
Pembagi jumlah rasio pencacahan pencatatan titik penempatan populasi ruang seluruh ukuran kerumunan parameter elemen batas pengamatan sampel total elemen hitung n = 3.
Rata-rata rata mean himpunan pembagi = $\frac{18}{3} = 6$.

## 4. Logika Komputasi
```javascript
function rataRataGugusData(kumpulanPopulasiSampel) {
  let rasioPenjumlahanAgregasiTotal = 0;
  
  for (let perulanganIndeks = 0; perulanganIndeks < kumpulanPopulasiSampel.length; perulanganIndeks++) {
    rasioPenjumlahanAgregasiTotal += kumpulanPopulasiSampel[perulanganIndeks];
  }
  
  const hasilHasilDistribusiSentral = rasioPenjumlahanAgregasiTotal / kumpulanPopulasiSampel.length;
  return hasilHasilDistribusiSentral;
}

console.log(rataRataGugusData([4, 6, 8]));
```
