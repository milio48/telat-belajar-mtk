Bab 6: Fungsi Komposisi dan Fungsi Invers

Metode Grounded Math — 6 Pilar Pembelajaran

1. Akar Konsep

Fungsi komposisi adalah penggabungan dua fungsi secara berurutan, di mana hasil keluaran fungsi pertama menjadi masukan untuk fungsi kedua. Fungsi invers adalah kebalikan dari suatu fungsi — jika fungsi \(f\) mengubah \(x\) menjadi \(y\), maka \(f^{-1}\) mengubah \(y\) kembali menjadi \(x\). Fungsi komposisi ibarat rantai perakitan di pabrik: pekerja pertama menghaluskan besi, lalu hasilnya diserahkan ke pekerja kedua untuk dicat.

Catatan: Tidak semua fungsi memiliki invers. Syaratnya, fungsi harus bijektif[^1] (satu-satu dan surjektif).

2. Pijakan Bumi

  • Spesialis Valuta Asing: Melakukan konversi bertingkat dan membalikkan kurs (Rupiah → Dollar → Euro, lalu Euro → Dollar → Rupiah saat retur).
  • Teknisi Logistik: Melacak paket melalui kode rute pengiriman berlapis, lalu menggunakan fungsi invers untuk pelacakan balik saat retur.

3. Mekanika Dasar

Komposisi fungsi: \((f \circ g)(x) = f(g(x))\).

Invers fungsi linear: Jika \(f(x) = ax + b\), maka \(f^{-1}(x) = \frac{x - b}{a}\).

Contoh: \(f(x) = 2x + 1\) dan \(g(x) = x - 3\).

\((f \circ g)(x) = f(g(x)) = f(x - 3) = 2(x - 3) + 1 = 2x - 6 + 1 = 2x - 5\).

4. Logika Komputasi

function fungsiF(x) {
  return 2 * x + 1;
}

function fungsiG(x) {
  return x - 3;
}

function komposisiFG(x) {
  const hasilG = fungsiG(x);
  const hasilFG = fungsiF(hasilG);
  return hasilFG;
}

console.log(komposisiFG(10)); // Output: 15

6. Uji Ketangkasan

Soal 1:

Diketahui \(f(x) = 3x - 5\). Tentukan \(f^{-1}(x)\).

Penyelesaian:

  1. Tulis sebagai \(y = 3x - 5\).
  2. Tukar posisi \(x\) dan \(y\): \(x = 3y - 5\).
  3. Selesaikan untuk \(y\): \(3y = x + 5\), maka \(y = \frac{x + 5}{3}\). Jadi, \(f^{-1}(x) = \frac{x + 5}{3}\).

Soal 2:

Diketahui \(f(x) = x + 4\) dan \(g(x) = 2x - 1\). Tentukan \((g \circ f)(3)\).

Penyelesaian:

  1. Hitung \(f(3)\) terlebih dahulu: \(f(3) = 3 + 4 = 7\).
  2. Masukkan hasilnya ke \(g\): \(g(7) = 2(7) - 1 = 14 - 1 = 13\).
  3. Jadi, \((g \circ f)(3) = 13\).

Bijektif: Fungsi yang bersifat satu-satu (setiap output memiliki tepat satu input) dan surjektif (setiap elemen kodomain memiliki pasangan di domain). Contoh fungsi non-bijektif: \(f(x) = x^2\) karena \(f(2) = f(-2) = 4\).
