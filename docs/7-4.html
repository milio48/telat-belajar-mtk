Bab 4: Persamaan dan Pertidaksamaan Linear Satu Variabel

Metode Grounded Math — 6 Pilar Pembelajaran

1. Akar Konsep

Persamaan dan pertidaksamaan linear satu variabel adalah kalimat matematika terbuka yang melibatkan hanya satu macam huruf bernilai variabel yang berderajat pangkat satu. Persamaan menggunakan tanda sama dengan (=\(=\)) yang menyatakan keseimbangan di kedua ruas kalimat, sedangkan pertidaksamaan menggunakan tanda relasi ketidaksamaan (seperti \(<\) atau \(>\)). Penyelesaian sistem ini adalah pencarian satu angka pasti yang akan menjadikan kalimat terbuka tersebut bernilai benar secara logika. Ini ibarat timbangan berat yang harus seimbang di dua sisi kanan dan kiri.

2. Pijakan Bumi

  • Sopir Angkutan Jasa: Pengemudi dapat mengetahui kapasitas bangku batas penumpang yang dilarang melampaui angkutan muatannya.
  • Kasir Toko Jajanan: Kasir toko dapat mencari harga per satuan barang yang tertera pada catatan beli rombongan total ketika struk hilangnya satuan barang.

3. Mekanika Dasar

Penyelesaian Persamaan Linear Satu Variabel dapat dilakukan dengan mengelompokkan variabel di satu ruas:

\[ ax + b = c \]
\[ ax = c - b \]
\[ x = \frac{c - b}{a} \]

Sebagai contoh, jika \( 2x + 3 = 7 \), maka:

\[ 2x = 7 - 3 \]
\[ 2x = 4 \]
\[ x = 2 \]

4. Logika Komputasi

function cariNilaiVariabel(koefisien, konstantaSatu, konstantaDua) {
  let pindahRuas = konstantaDua - konstantaSatu;
  let nilaiVariabel = pindahRuas / koefisien;
  return nilaiVariabel;
}

let nilaiA = 2;
let nilaiB = 3;
let nilaiC = 7;
let hasilBagi = cariNilaiVariabel(nilaiA, nilaiB, nilaiC);
console.log("Nilai variabel adalah: " + hasilBagi);
