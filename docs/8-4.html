Bab 4: Statistika

Metode Grounded Math — 6 Pilar Pembelajaran

1. Akar Konsep

Statistika adalah cabang matematika yang mempelajari cara mengumpulkan, menyusun, menganalisis, dan menafsirkan data untuk mengambil kesimpulan. Tiga ukuran pemusatan data yang paling mendasar adalah mean (rata-rata), median (nilai tengah), dan modus (nilai yang paling sering muncul). Statistika ibarat mengukur suhu rata-rata sebuah kota selama sebulan — kita tidak perlu memantau setiap detik, cukup mengambil sampel data harian lalu menghitung rata-ratanya untuk mendapatkan gambaran umum.

Catatan: Mean cocok untuk data tanpa pencilan[^1], median cocok untuk data yang memiliki pencilan, dan modus cocok untuk data kategorikal.

2. Pijakan Bumi

  • Peneliti Pemasaran: Menghitung rata-rata pengeluaran konsumen per transaksi untuk menentukan strategi harga produk.
  • Insinyur Jaringan: Menganalisis median waktu respons server harian untuk mendeteksi penurunan performa sistem.

3. Mekanika Dasar

Rumus rata-rata (mean):

\[ \bar{x} = \frac{\sum_{i=1}^{n} x_i}{n} \]

Contoh: Terdapat data nilai ujian 3 siswa: 4, 6, dan 8.

  1. Jumlahkan semua data: \(4 + 6 + 8 = 18\).
  2. Hitung banyak data: \(n = 3\).
  3. Rata-rata: \(\bar{x} = \frac{18}{3} = 6\).

Untuk median, urutkan data terlebih dahulu. Jika jumlah data ganjil, median adalah nilai di posisi tengah. Jika genap, median adalah rata-rata dari dua nilai di tengah.

Untuk modus, cari nilai yang paling sering muncul dalam kumpulan data.

4. Logika Komputasi

function hitungMean(kumpulanData) {
  let totalNilai = 0;

  for (let i = 0; i < kumpulanData.length; i++) {
    totalNilai += kumpulanData[i];
  }

  const nilaiRata = totalNilai / kumpulanData.length;
  return nilaiRata;
}

console.log(hitungMean([4, 6, 8])); // Output: 6

6. Uji Ketangkasan

Soal 1:

Seorang manajer warung mencatat jumlah pesanan minuman selama 7 hari: 5, 4, 2, 5, 8, 5, 3. Tentukan modus dari data penjualan tersebut.

Penyelesaian:

  1. Urutkan data dari terkecil ke terbesar: {2, 3, 4, 5, 5, 5, 8}.
  2. Hitung frekuensi kemunculan setiap nilai: angka 2 muncul 1 kali, 3 muncul 1 kali, 4 muncul 1 kali, 5 muncul 3 kali, 8 muncul 1 kali.
  3. Nilai yang paling sering muncul adalah 5. Maka, modus data penjualan adalah 5 pesanan.

Soal 2:

Nilai ulangan 6 siswa adalah: 60, 75, 80, 80, 90, 95. Tentukan median dari data tersebut.

Penyelesaian:

  1. Data sudah terurut: {60, 75, 80, 80, 90, 95}. Jumlah data \(n = 6\) (genap).
  2. Median untuk data genap dihitung dari dua nilai tengah, yaitu data ke-3 dan ke-4: nilai 80 dan 80.
  3. Median \(= \frac{80 + 80}{2} = 80\). Nilai tengah data tersebut adalah 80.

Pencilan (Outlier): Nilai data yang jauh berbeda dari mayoritas data lainnya. Contoh: dalam data {5, 6, 7, 8, 100}, angka 100 adalah pencilan yang dapat menggeser mean secara signifikan.
