Bab 1: Integral

Metode Grounded Math — 6 Pilar Pembelajaran

1. Akar Konsep

Integral adalah konsep kalkulus yang merupakan kebalikan (invers) dari turunan (diferensial). Secara geometris, integral tentu digunakan untuk menghitung luas daerah di bawah kurva fungsi. Integral ibarat menghitung total volume air hujan yang tertampung dalam sebuah ember — turunan memberitahu kecepatan air menetes per detik, sedangkan integral menjumlahkan seluruh tetesan dari awal hingga akhir.

Catatan: Integral tak tentu menghasilkan fungsi baru ditambah konstanta integrasi \(C\)[^1], sedangkan integral tentu menghasilkan nilai numerik pasti.

2. Pijakan Bumi

  • Teknisi Bendungan: Menghitung total volume air yang mengalir melalui bendungan dalam periode waktu tertentu berdasarkan data laju debit air.
  • Ahli Pencitraan Medis: Mengintegrasikan data dari irisan-irisan gambar MRI dua dimensi untuk merekonstruksi bentuk organ tiga dimensi.

3. Mekanika Dasar

Rumus integral tak tentu untuk bentuk \(ax^n\):

\(\int ax^n \, dx = \frac{a}{n+1} x^{n+1} + C\)

Contoh: Hitung \(\int 4x^3 \, dx\).

  1. Tambahkan 1 pada pangkat: \(3 + 1 = 4\).
  2. Bagi koefisien dengan pangkat baru: \(\frac{4}{4} = 1\).
  3. Hasil: \(x^4 + C\).

4. Logika Komputasi

function integralDasar(koefisien, pangkat) {
  // Hanya berlaku untuk integral bentuk ax^n
  const pangkatBaru = pangkat + 1;
  const koefisienBaru = koefisien / pangkatBaru;

  return {
    koefisien: koefisienBaru,
    pangkat: pangkatBaru,
    hasil: `${koefisienBaru}x^${pangkatBaru} + C`
  };
}

// Integral dari 6x^2
const hasil = integralDasar(6, 2);
console.log(hasil.hasil); // Output: "2x^3 + C"

6. Uji Ketangkasan

Soal 1:

Hitunglah hasil dari \(\int (3x^2 - 4x + 5) \, dx\).

Penyelesaian:

  1. Integralkan setiap suku secara terpisah:
  2. \(\int 3x^2 \, dx = \frac{3}{3}x^3 = x^3\).
  3. \(\int -4x \, dx = \frac{-4}{2}x^2 = -2x^2\).
  4. \(\int 5 \, dx = 5x\).
  5. Gabungkan seluruh hasil dan tambahkan konstanta \(C\).
  6. Hasil akhir: \(x^3 - 2x^2 + 5x + C\).

Soal 2:

Hitunglah nilai integral tentu \(\int_{1}^{3} 2x \, dx\).

Penyelesaian:

  1. Cari anti-turunan[^2] dari \(2x\): \(\int 2x \, dx = x^2\).
  2. Evaluasi pada batas atas dan bawah: \([x^2]_{1}^{3} = 3^2 - 1^2 = 9 - 1\).
  3. Hasil: \(\int_{1}^{3} 2x \, dx = 8\).

Konstanta Integrasi (\(C\)): Nilai konstan sembarang yang ditambahkan pada hasil integral tak tentu karena turunan dari konstanta selalu nol, sehingga ada tak hingga banyak fungsi yang memiliki turunan yang sama.

Anti-turunan: Istilah lain untuk integral tak tentu. Jika \(F'(x) = f(x)\), maka \(F(x)\) adalah anti-turunan dari \(f(x)\).
