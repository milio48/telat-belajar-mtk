Bab 3: Sistem Persamaan Linier dan Kuadrat

Metode Grounded Math — 6 Pilar Pembelajaran

1. Akar Konsep

Sistem persamaan adalah kumpulan dua atau lebih persamaan yang saling terkait dan membagikan batasan nilai yang sama pada variabel-variabelnya. Solusi dari sistem ini adalah perpotongan dari semua persamaan tersebut. Mencari solusi sistem persamaan ibarat dua detektif yang mencari lokasi tersangka dari dua jejak laporan yang berbeda; lokasi tersangkanya adalah titik silang interaksi dari kedua jejak tersebut.

2. Pijakan Bumi

  • Ahli Gizi: Menghitung jumlah kombinasi makanan x dan y untuk mencapai batas gizi protein dan karbohidrat yang tepat tanpa melebihi batas kalori harian.
  • Konsultan Bisnis: Menghitung titik impas (break-even point) dengan mempertemukan garis total biaya operasional dengan kurva perkiraan penjualan.

3. Mekanika Dasar

Contoh SPLDV:

1) \(2x + y = 5\)

2) \(x - y = 1\)

Eliminasi dengan menjumlahkan (1) dan (2):

\((2x + x) + (y - y) = 5 + 1\)

\(3x = 6\), maka \(x = 2\)

Substitusikan \(x = 2\) ke (1): \(2(2) + y = 5\), sehingga \(y = 1\).

4. Logika Komputasi

function solusiSpldv(a1, b1, c1, a2, b2, c2) {
  // Matriks determinan untuk aturan Cramer
  const determinan = (a1 * b2) - (a2 * b1);
  
  if (determinan === 0) {
    return "Garis sejajar atau berhimpit";
  }
  
  const nilaiX = ((c1 * b2) - (c2 * b1)) / determinan;
  const nilaiY = ((a1 * c2) - (a2 * c1)) / determinan;
  
  return { titikX: nilaiX, titikY: nilaiY };
}

const titikPotong = solusiSpldv(2, 1, 5, 1, -1, 1);
console.log(titikPotong); // Output: { titikX: 2, titikY: 1 }

6. Uji Ketangkasan

Soal:

Sebuah pabrik produksi merakit meja dan kursi. Waktu yang dicatat untuk merakit 2 meja dan 3 kursi adalah 12 jam. Untuk merakit 3 meja dan 2 kursi memakan waktu 13 jam. Berapa jam waktu yang dibutuhkan untuk merakit 1 meja?

Penyelesaian:

  1. Misalkan waktu merakit meja \(m\) dan kursi \(k\). Buat persamaan: \(2m + 3k = 12\) dan \(3m + 2k = 13\).
  2. Samakan koefisien \(k\) pada kedua persamaan: kalikan persamaan pertama dengan 2 (\(4m + 6k = 24\)), kalikan persamaan kedua dengan 3 (\(9m + 6k = 39\)).
  3. Kurangi persamaan hasil modifikasi: \((9m - 4m) = 39 - 24\), jadi \(5m = 15\), maka \(m = 3\) jam per meja.
