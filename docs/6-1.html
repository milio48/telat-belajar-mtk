Bab 1: Operasi Hitung Bilangan Bulat dan Pecahan (Tingkat Lanjut)

Metode Grounded Math — 6 Pilar Pembelajaran

1. Akar Konsep

Operasi hitung campuran tingkat lanjut adalah penyelesaian perhitungan yang melibatkan berbagai jenis bilangan dan operasi matematika secara bersamaan, mengikuti urutan prioritas baku. Proses ini ibarat merakit sebuah mesin, di mana setiap komponen (penjumlahan, pengurangan, perkalian, pembagian, akar, atau pangkat) harus dipasang sesuai urutan agar mesin dapat beroperasi dengan benar.

2. Pijakan Bumi

  • Akuntan perusahaan: Menghitung total keuntungan dengan menggabungkan bilangan bulat (jumlah barang) dan pecahan desimal (harga satu unit nilai tukar mata uang).
  • Insinyur sipil: Menentukan volume bahan bangunan menggunakan akar pangkat tiga dari kapasitas ruang yang tersedia.

3. Mekanika Dasar

Urutan prioritas operasi hitung (Kaidah KuKaBaTaKu - Kurung, Kali/Bagi, Tambah/Kurang):

  1. Operasi dalam tanda kurung
  2. Pangkat dan akar
  3. Perkalian dan pembagian (dari kiri ke kanan)
  4. Penjumlahan dan pengurangan (dari kiri ke kanan)

Contoh perhitungan campuran pecahan dan bilangan bulat:

\(\frac{1}{2} + 3 \times 0.5 = ?\)

Karena perkalian lebih prioritas, maka:

\(3 \times 0.5 = 1.5\)

Sehingga:

\(0.5 + 1.5 = 2\)

4. Logika Komputasi

function hitungCampuran() {
  const nilaiPecahan = 1 / 2;
  const bilanganBulat = 3;
  const nilaiDesimal = 0.5;
  
  // Perkalian dikerjakan terlebih dahulu sesuai prioritas operasi
  const hasilPerkalian = bilanganBulat * nilaiDesimal;
  const hasilAkhir = nilaiPecahan + hasilPerkalian;
  
  return hasilAkhir;
}

console.log(hitungCampuran()); // Output: 2

6. Uji Ketangkasan

Soal:

Sebuah toko memiliki 2 kuintal beras. Beras tersebut dibagikan kepada 4 panti asuhan sama banyak. Jika setiap panti asuhan sudah memiliki cadangan \(0.25\) kuintal beras, hitunglah total persediaan beras di salah satu panti asuhan sekarang!

Penyelesaian:

  1. Hitung bagian beras yang diterima setiap panti asuhan melalui operasi pembagian: \(2 \div 4 = 0.5\) kuintal.
  2. Tambahkan beras yang diterima dengan cadangan beras yang sudah ada menggunakan operasi penjumlahan: \(0.5 + 0.25\).
  3. Dapatkan hasil akhir persediaan beras: \(0.75\) kuintal.
