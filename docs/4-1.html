Bab 1: Operasi Hitung Bilangan dan Bilangan Bulat

Metode Grounded Math — 6 Pilar Pembelajaran

1. Akar Konsep

Bilangan bulat adalah himpunan bilangan cacah yang digabungkan dengan bilangan negatif, namun himpunan ini tidak termasuk bilangan desimal maupun pecahan. Aturan operasi hitung seperti sifat komutatif, asosiatif, dan distributif dimanfaatkan untuk mempermudah perolehan hasil kalkulasi berantai. Bilangan bulat ibarat urutan nilai bilangan pada anak tangga di sebuah gedung berlantai banyak.

2. Pijakan Bumi

  • Akuntan perusahaan: Menulis perolehan pemasukan sebagai bilangan positif dan tagihan utang sebagai bilangan negatif pada neraca.
  • Pakar meteorologi: Menentukan suhu udara dingin pada musim salju hingga berada dalam tingkat hitungan negatif bilangan celcius.

3. Mekanika Dasar

Rumus sifat distributif perkalian terhadap penjumlahan:

\(a \times (b + c) = (a \times b) + (a \times c)\)

Contoh perhitungan dengan angka dasar:

\(2 \times (3 + 4) = (2 \times 3) + (2 \times 4)\)

\(2 \times 7 = 6 + 8\)

\(14 = 14\)

4. Logika Komputasi

function hitungDistributif(angkaA, angkaB, angkaC) {
  let hasilKiri = angkaA * (angkaB + angkaC);
  let hasilKanan = (angkaA * angkaB) + (angkaA * angkaC);
  return hasilKiri; // Keduanya akan merepresentasikan nilai yang sama
}

6. Uji Ketangkasan

Sebuah kapal selam berada tepat 10 meter di bawah permukaan laut. Kapal selam tersebut kemudian naik 4 meter menuju ke atas permukaan batas perairan. Berapa posisi kapal selam saat ini jika permukaan laut dianggap sebagai titik bernilai nol?

  1. Tuliskan letak koordinat sumbu kedalaman kapal selam di mula titik awal yaitu bernilai -10.
  2. Tambahkan parameter kenaikan tinggi pergerakan sebanyak 4 karena benda bermanuver arah ke atas batas.
  3. Hitung hasil angka penjumlahannya untuk nilai posisi kedalaman kapal selam saat ini (hasil dari penjumlahan -10 dan 4 adalah -6). Posisi kapal selam kini menetap pada radius 6 meter pada area kedalaman minus air.
