Bab 7: Ruang Dimensi Tiga

Metode Grounded Math — 6 Pilar Pembelajaran

1. Akar Konsep

Ruang dimensi tiga adalah kajian geometri yang membahas bangun atau obyek yang memiliki parameter panjang, lebar, serta kedalaman/tinggi. Konsep ini mempelajari letak kedudukan titik, garis, dan jarak antar objek dalam proyeksi spasial. Ruang dimensi tiga ibarat kardus pengiriman barang. Jika kita melubangi salah satu sudut kardus dengan tali dan menariknya lurus ke sudut yang berlawanan arah, garis tali tersebut adalah wujud nyata dari panjang diagonal ruang tiga dimensi.

2. Pijakan Bumi

  • Desainer Interior: Menentukan volume udara total dari sebuah ruangan kosong untuk memasang perangkat sistem AC sentral yang mampu menyejukkan udara dengan kapasitas optimal.
  • Insinyur Pabrikasi: Menghitung jarak potong laser terpendek yang dibutuhkan antar dua lubang cetakan untuk memangkas sisi terluang perakitan alat sasis mobil baja.

3. Mekanika Dasar

Formula jarak titik potong pada sebuah bola dalam bangun kubus dengan panjang rusuk \(s\):

Panjang Diagonal Sisi = \(s\sqrt{2}\)

Panjang Diagonal Ruang = \(s\sqrt{3}\)

Contoh: Sebuah kubus memiliki dimensi sisi rusuk 5 cm. Jarak terpanjang antara dua titik sudut terjauh secara menyilang melewati isi tengah ruang adalah:

\(D_{ruang} = s\sqrt{3} = 5\sqrt{3}\) cm

4. Logika Komputasi

function volumeLimasSegiEmpat(panjangAlas, lebarAlas, tinggiLimas) {
  // Rumus volume limas: (1/3) * Luas Alas * Tinggi
  const luasDasarAlas = panjangAlas * lebarAlas;
  
  const totalKapasitas = (1 / 3) * luasDasarAlas * tinggiLimas;
  
  return totalKapasitas;
}

const sisiPanjang = 4;
const sisiLebar = 4;
const tiangTinggi = 9;

const volumeBangunan = volumeLimasSegiEmpat(sisiPanjang, sisiLebar, tiangTinggi);
console.log(volumeBangunan); // Output: 48

6. Uji Ketangkasan

Soal:

Kamar tidur Joko adalah sebuah kubus dengan spesifikasi sisi berukuran 3 meter. Joko tanpa sengaja menjatuhkan sebuah gantungan dari sudut pojok langit-langit atap kamarnya dan membentur tepat secara bersilangan ke sudut paling ujung terjauh di lantai kamarnya. Semisal lintasan gantungan itu diukur lurus, berapakah meterkah panjang jarak pergeserannya?

Penyelesaian:

  1. Konstruksikan soal ruang cerita ini: ini membahas lintasan terjauh penghubung ujung atas dinding dengan ujung titik dasar di dinding sisi lainnya melewati pusat kosong ruang.
  2. Formulasi diagonal ruang pada bangun matematika balok atau kubus adalah \(s\sqrt{3}\).
  3. Hitung hasil jarak: \(s = 3\), maka jarak lintasan jatuhnya jatuh pada panjang absolut angka \(3\sqrt{3}\) meter.
