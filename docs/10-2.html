Bab 2: Persamaan dan Fungsi Kuadrat

Metode Grounded Math — 6 Pilar Pembelajaran

1. Akar Konsep

Persamaan kuadrat adalah persamaan matematika dengan pangkat tertinggi variabelnya adalah dua, yang bila digambarkan akan membentuk kurva melengkung yang simetris menjauhi satu titik puncak. Fungsi kuadrat ibarat lintasan lemparan bola basket. Ketika bola dilempar, ia naik perlahan, mencapai titik tertinggi, dan melengkung jatuh kembali ke keranjang membentuk kurva parabola.

2. Pijakan Bumi

  • Insinyur Sipil: Menggunakan fungsi kuadrat untuk menghitung lengkungan kabel penyangga pada jembatan gantung untuk menahan beban secara optimal.
  • Ahli Balistik: Menghitung sudut elevasi tembakan dan memprediksi titik jatuhnya proyektil menggunakan persamaan kuadrat.

3. Mekanika Dasar

Bentuk umum persamaan kuadrat: \(ax^2 + bx + c = 0\)

Mencari akar dengan rumus kuadratik (Rumus abc):

\(x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}\)

Contoh: Cari akar dari \(x^2 - 5x + 6 = 0\)

\(a = 1, b = -5, c = 6\)

\(x = \frac{-(-5) \pm \sqrt{(-5)^2 - 4(1)(6)}}{2(1)} = \frac{5 \pm \sqrt{25 - 24}}{2} = \frac{5 \pm 1}{2}\)

Maka \(x_1 = 3\) atau \(x_2 = 2\)

4. Logika Komputasi

function cariAkarKuadrat(a, b, c) {
  const diskriminan = Math.pow(b, 2) - (4 * a * c);
  
  if (diskriminan < 0) {
    return "Akar imajiner";
  }
  
  const akarSatu = (-b + Math.sqrt(diskriminan)) / (2 * a);
  const akarDua = (-b - Math.sqrt(diskriminan)) / (2 * a);
  
  return [akarSatu, akarDua];
}

const akarPersamaan = cariAkarKuadrat(1, -5, 6);
console.log(akarPersamaan); // Output: [3, 2]

6. Uji Ketangkasan

Soal:

Ketinggian lintasan roket air dalam satuan meter setelah \(t\) detik diluncurkan memenuhi persamaan h(t) = \(40t - 5t^2\). Berapa detik waktu yang dibutuhkan roket untuk mencapai titik tertinggi sebelum jatuh kembali ke tanah?

Penyelesaian:

  1. Identifikasi bentuk umum fungsi kuadrat dari \(h(t) = -5t^2 + 40t\), maka \(a = -5, b = 40, c = 0\).
  2. Waktu untuk mencapai titik tertinggi adalah sumbu simetri parabola, yang dihitung dengan rumus \(t = -b / 2a\).
  3. Hitung waktu maksimal: \(t = -40 / (2 \times -5) = -40 / -10 = 4\). Jadi roket mencapai titik tertinggi pada detik ke-4.
