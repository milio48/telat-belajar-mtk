Bab 8: Bilangan Berpangkat dan Bentuk Akar

Metode Grounded Math — 6 Pilar Pembelajaran

1. Akar Konsep

Bilangan berpangkat (eksponen) adalah notasi singkat untuk perkalian berulang suatu bilangan dengan dirinya sendiri. Bentuk akar adalah operasi kebalikan dari perpangkatan, yaitu mencari bilangan dasar yang jika dipangkatkan akan menghasilkan nilai tertentu. Perpangkatan ibarat melipat sebuah kertas secara berulang — setiap lipatan menggandakan ketebalannya secara eksponensial. Bentuk akar ibarat membuka lipatan tersebut untuk mengetahui ketebalan aslinya.

Catatan: Akar kuadrat (\(\sqrt{a}\)) hanya terdefinisi untuk bilangan non-negatif (\(a \geq 0\)) dalam himpunan bilangan real[^1].

2. Pijakan Bumi

  • Biolog Sel: Memodelkan pertumbuhan populasi bakteri yang membelah diri setiap periode waktu tertentu menggunakan fungsi eksponen.
  • Ahli Teknologi Nano: Menggunakan notasi ilmiah berpangkat untuk menuliskan ukuran partikel yang sangat kecil, misalnya \(5 \times 10^{-9}\) meter.

3. Mekanika Dasar

Operasi perpangkatan \(a^n\) berarti:

\[ a^n = \underbrace{a \times a \times a \times \dots \times a}_{n \text{ kali}} \]

Operasi bentuk akar \(\sqrt[n]{a}\):

\[ \sqrt[n]{a} = b \iff b^n = a \]

Contoh perpangkatan: \(3^4 = 3 \times 3 \times 3 \times 3 = 81\).

Contoh bentuk akar: \(\sqrt{25} = 5\), karena \(5^2 = 25\).

4. Logika Komputasi

function hitungPangkat(bilDasar, pangkat) {
  const hasil = Math.pow(bilDasar, pangkat);
  return hasil;
}

function hitungAkarKuadrat(bilAsal) {
  if (bilAsal < 0) {
    return "Tidak terdefinisi untuk bilangan negatif";
  }
  const hasil = Math.sqrt(bilAsal);
  return hasil;
}

console.log(hitungPangkat(3, 4));    // Output: 81
console.log(hitungAkarKuadrat(25));  // Output: 5

6. Uji Ketangkasan

Soal 1:

Sebuah karpet berbentuk persegi memiliki luas \(144\) m². Berapakah panjang sisi karpet tersebut?

Penyelesaian:

  1. Rumus luas persegi: \(L = s^2\), di mana \(s\) adalah panjang sisi.
  2. Substitusi nilai: \(144 = s^2\).
  3. Tarik akar kuadrat: \(s = \sqrt{144} = 12\) meter. Panjang sisi karpet adalah 12 meter.

Soal 2:

Sederhanakan bentuk akar berikut: \(\sqrt{50} + \sqrt{18}\).

Penyelesaian:

  1. Uraikan \(\sqrt{50}\): \(\sqrt{50} = \sqrt{25 \times 2} = 5\sqrt{2}\).
  2. Uraikan \(\sqrt{18}\): \(\sqrt{18} = \sqrt{9 \times 2} = 3\sqrt{2}\).
  3. Jumlahkan suku-suku yang sejenis: \(5\sqrt{2} + 3\sqrt{2} = 8\sqrt{2}\).

Bilangan Real: Himpunan bilangan yang mencakup seluruh bilangan rasional (pecahan) dan irasional (seperti \(\sqrt{2}\) dan \(\pi\)). Disimbolkan dengan \(\mathbb{R}\).
