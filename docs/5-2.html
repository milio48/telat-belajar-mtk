Bab 2: Pengukuran Lanjutan

Metode Grounded Math — 6 Pilar Pembelajaran

1. Akar Konsep

Kecepatan adalah besaran fisis yang menyatakan jarak perpindahan posisi suatu benda dalam satu satuan waktu tertentu. Kecepatan ibarat putaran jarum penunjuk spidometer pada kendaraan mobil yang meningkat saat kita menekan pedal gas.

2. Pijakan Bumi

  • Manajer Logistik: Menghitung estimasi jadwal paket barang tiba dengan memperhatikan jarak antarkota dan rata-rata kecepatan truk.
  • Pengawas Penerbangan: Memantau arah putaran rotasi derajat lintasan pesawat agar bisa mendarat darurat dengan aman di sirkuit bandara.

3. Mekanika Dasar

Rumus dasar kecepatan adalah hasil bagi dari total jarak tempuh dengan total waktu tempuh.

\(v = \frac{s}{t}\)

Jika jarak suatu perjalanan (\(s\)) adalah \(150\) km dan waktu yang dibutuhkan (\(t\)) adalah \(3\) jam, maka kecepatan (\(v\)) dihitung sebagai:

\(v = \frac{150}{3} = 50\) km/jam.

4. Logika Komputasi

function hitungKecepatanLaju(jarakTempuh, waktuTotal) {
    if (waktuTotal <= 0) {
        return 0;
    }
    const nilaiKecepatan = jarakTempuh / waktuTotal;
    
    return nilaiKecepatan;
}

const lajuKendaraan = hitungKecepatanLaju(150, 3);
console.log(lajuKendaraan); // Output: 50

6. Uji Ketangkasan

Soal: Sebuah kereta api komuter melakukan perjalanan lintas kota yang berjarak 300 kilometer. Kereta api ini diatur agar terus melaju dengan stabil pada kecepatan yang konstan 60 km/jam. Jika jadwal awal kereta api tersebut berangkat dari stasiun pada pukul 08:00 pagi, pukul berapakah kereta tersebut mencapai stasiun tujuan akhirnya?

Penyelesaian:

  1. Modifikasi persamaan rumus kecepatan dasar untuk mencari waktu tempuh: \(t = \frac{s}{v}\).
  2. Hitung jumlah waktu perjalanan dengan cara memasukkan besaran angka: \(t = \frac{300}{60} = 5\) jam total waktu di jalan.
  3. Tambahkan durasi durasi tempuh tersebut ke jadwal waktu keberangkatan awal jam: \(08:00 + 5\) jam sama dengan pukul \(13:00\). Kereta tiba tepat pada pukul 13:00 siang.
