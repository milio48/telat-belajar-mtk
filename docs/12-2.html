Bab 2: Program Linear

Metode Grounded Math — 6 Pilar Pembelajaran

1. Akar Konsep

Program linear adalah metode matematika untuk menemukan nilai optimal (maksimum atau minimum) suatu fungsi tujuan yang tunduk pada serangkaian kendala berupa pertidaksamaan linear. Program linear ibarat seorang manajer pabrik yang harus menentukan komposisi produksi terbaik dari beberapa jenis barang dengan keterbatasan bahan baku dan tenaga kerja, agar keuntungan menjadi sebesar-besarnya.

Catatan: Solusi optimal pada program linear selalu terletak di salah satu titik sudut (vertex) dari daerah penyelesaian[^1], bukan di interior area.

2. Pijakan Bumi

  • Manajer Logistik: Menentukan rute distribusi barang antar gudang dan gerai yang paling hemat bahan bakar dengan memperhatikan kapasitas armada.
  • Ahli Gizi Rumah Sakit: Menyusun menu makanan pasien yang memenuhi kebutuhan nutrisi minimum dengan biaya belanja paling rendah.

3. Mekanika Dasar

Komponen program linear:

  • Fungsi tujuan: \(Z = ax + by\) (dimaksimalkan atau diminimalkan).
  • Kendala: Pertidaksamaan linear seperti \(cx + dy \le m\).
  • Syarat non-negatif: \(x \ge 0\), \(y \ge 0\).

Langkah penyelesaian:

  1. Gambar grafik setiap pertidaksamaan pada bidang koordinat.
  2. Tentukan daerah penyelesaian (irisan semua kendala).
  3. Uji setiap titik sudut daerah penyelesaian ke dalam fungsi tujuan.
  4. Pilih titik yang menghasilkan nilai \(Z\) optimal.

4. Logika Komputasi

function cariLabaMaksimal(titikSudut, fungsiZ) {
  let nilaiMaksimal = 0;

  for (let i = 0; i < titikSudut.length; i++) {
    const x = titikSudut[i][0];
    const y = titikSudut[i][1];
    const nilaiZ = fungsiZ(x, y);

    if (nilaiZ > nilaiMaksimal) {
      nilaiMaksimal = nilaiZ;
    }
  }
  return nilaiMaksimal;
}

const titikUji = [[0, 5], [4, 0], [2, 3]];
const fungsiTujuan = (x, y) => 1000 * x + 1500 * y;

console.log(cariLabaMaksimal(titikUji, fungsiTujuan));
// Output: 7500

6. Uji Ketangkasan

Soal 1:

Seorang pengusaha mebel memproduksi kursi dan lemari. Profit per kursi Rp50.000, profit per lemari Rp80.000. Fungsi tujuan: \(Z = 50.000x + 80.000y\). Titik sudut daerah penyelesaian adalah A(10, 0), B(8, 6), dan C(0, 12). Tentukan keuntungan maksimal.

Penyelesaian:

  1. Uji titik A(10, 0): \(Z = 50.000(10) + 80.000(0) = 500.000\).
  2. Uji titik B(8, 6): \(Z = 50.000(8) + 80.000(6) = 400.000 + 480.000 = 880.000\).
  3. Uji titik C(0, 12): \(Z = 50.000(0) + 80.000(12) = 960.000\). Keuntungan maksimal adalah Rp960.000 di titik C(0, 12).

Soal 2:

Sebuah toko roti membuat kue A dan kue B. Setiap kue A membutuhkan 2 jam pembuatan, setiap kue B membutuhkan 3 jam. Total waktu tersedia 12 jam. Kapasitas oven hanya untuk 5 kue per hari. Jika profit kue A = Rp20.000 dan kue B = Rp30.000, berapa kue masing-masing yang harus dibuat agar laba maksimal?

Penyelesaian:

  1. Variabel: \(x\) = jumlah kue A, \(y\) = jumlah kue B. Fungsi tujuan: \(Z = 20.000x + 30.000y\).
  2. Kendala: \(2x + 3y \le 12\); \(x + y \le 5\); \(x, y \ge 0\). Titik sudut: (0, 0), (5, 0), (3, 2), (0, 4).
  3. Uji: Z(5,0) = 100.000; Z(3,2) = 60.000 + 60.000 = 120.000; Z(0,4) = 120.000. Laba maksimal Rp120.000 tercapai di titik (3, 2) atau (0, 4).

Daerah Penyelesaian (Feasible Region): Area pada bidang koordinat yang memenuhi semua kendala pertidaksamaan secara bersamaan. Bentuknya selalu berupa poligon konveks.
