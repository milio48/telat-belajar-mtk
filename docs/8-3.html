Bab 3: Teorema Pythagoras

Metode Grounded Math — 6 Pilar Pembelajaran

1. Akar Konsep

Teorema Pythagoras menyatakan bahwa pada segitiga siku-siku, kuadrat panjang sisi miring (hipotenusa[^1]) sama dengan jumlah kuadrat kedua sisi tegaknya. Teorema ini hanya berlaku untuk segitiga yang memiliki sudut tepat \(90°\). Konsep ini ibarat memastikan sudut siku-siku sebuah tembok dengan mengukur tiga jarak — jika perhitungannya cocok, maka tembok tersebut tegak lurus sempurna.

⚠️ Penting: Teorema Pythagoras hanya berlaku untuk segitiga siku-siku. Untuk segitiga sembarang, digunakan aturan cosinus.

2. Pijakan Bumi

  • Tukang Konstruksi: Memastikan sudut fondasi bangunan benar-benar \(90°\) dengan menggunakan perbandingan sisi 3-4-5 (tripel Pythagoras).
  • Petugas Pemadam Kebakaran: Menghitung panjang tangga minimum yang dibutuhkan untuk mencapai jendela di ketinggian tertentu dengan jarak tertentu dari dinding.

3. Mekanika Dasar

Rumus Teorema Pythagoras:

\[ c^2 = a^2 + b^2 \]

Di mana \(c\) = hipotenusa (sisi miring), \(a\) dan \(b\) = kedua sisi tegak.

Contoh: Sisi tegak \(a = 6\) cm dan \(b = 8\) cm.

\(c^2 = 6^2 + 8^2\)

\(c^2 = 36 + 64 = 100\)

\(c = \sqrt{100} = 10\) cm.

4. Logika Komputasi

function hitungHipotenusa(sisiA, sisiB) {
  const kuadratA = sisiA * sisiA;
  const kuadratB = sisiB * sisiB;

  const hipotenusa = Math.sqrt(kuadratA + kuadratB);
  return hipotenusa;
}

console.log(hitungHipotenusa(6, 8)); // Output: 10

6. Uji Ketangkasan

Soal 1:

Tim pendaki berjalan 12 km ke arah utara, lalu berbelok ke arah timur sejauh 5 km. Berapakah jarak terdekat (garis lurus) dari titik awal ke titik akhir?

Penyelesaian:

  1. Lintasan membentuk segitiga siku-siku dengan sisi \(a = 12\) km (utara) dan \(b = 5\) km (timur).
  2. Gunakan rumus Pythagoras: \(c^2 = 12^2 + 5^2 = 144 + 25 = 169\).
  3. Jarak terdekat: \(c = \sqrt{169} = 13\) km.

Soal 2:

Sebuah tangga bersandar pada dinding setinggi 12 meter. Jika kaki tangga berjarak 9 meter dari dinding, berapa panjang tangga?

Penyelesaian:

  1. Tinggi dinding \(a = 12\) m, jarak kaki tangga \(b = 9\) m.
  2. Panjang tangga (hipotenusa): \(c^2 = 12^2 + 9^2 = 144 + 81 = 225\).
  3. Panjang tangga: \(c = \sqrt{225} = 15\) meter.

Hipotenusa: Sisi terpanjang pada segitiga siku-siku yang terletak berhadapan dengan sudut siku-siku (\(90°\)). Berasal dari bahasa Yunani "hypoteinousa" yang berarti "membentang di bawah."
