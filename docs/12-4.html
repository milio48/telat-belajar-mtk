Bab 4: Barisan, Deret, dan Notasi Sigma

Metode Grounded Math — 6 Pilar Pembelajaran

1. Akar Konsep

Barisan adalah susunan bilangan yang disusun menurut aturan tertentu. Deret adalah hasil penjumlahan suku-suku berturut-turut dari sebuah barisan. Notasi Sigma (\(\Sigma\)) adalah cara ringkas untuk menuliskan penjumlahan deret. Barisan aritmetika ibarat anak tangga yang jarak antar pijakannya selalu sama — setiap langkah naik dengan ketinggian yang tetap.

Catatan: Barisan aritmetika memiliki selisih (beda) tetap antar suku, sedangkan barisan geometri[^1] memiliki rasio (perbandingan) tetap antar suku.

2. Pijakan Bumi

  • Ahli Demografi: Memprediksi pertumbuhan populasi kota menggunakan model barisan geometri berdasarkan laju pertumbuhan tahunan.
  • Analis Keuangan: Menghitung total setoran deposito berbunga tetap dalam beberapa periode menggunakan deret aritmetika.

3. Mekanika Dasar

Suku ke-\(n\) barisan aritmetika:

\(U_n = a + (n - 1)b\)

Di mana \(a\) = suku pertama dan \(b\) = beda (selisih antar suku).

Notasi Sigma:

\(\sum_{i=1}^{4} i^2 = 1^2 + 2^2 + 3^2 + 4^2 = 1 + 4 + 9 + 16 = 30\)

Contoh: Suku ke-5 dari barisan dengan \(a = 2\) dan \(b = 3\):

\(U_5 = 2 + (5 - 1) \times 3 = 2 + 12 = 14\).

4. Logika Komputasi

function hitungSigma(batasAwal, batasAkhir) {
  let totalJumlah = 0;

  for (let i = batasAwal; i <= batasAkhir; i++) {
    totalJumlah += Math.pow(i, 2);
  }

  return totalJumlah;
}

console.log(hitungSigma(1, 4)); // Output: 30

6. Uji Ketangkasan

Soal 1:

Diketahui barisan aritmetika: 5, 9, 13, 17, ... Hitunglah suku ke-15 dari barisan tersebut.

Penyelesaian:

  1. Identifikasi: \(a = 5\), dan beda \(b = 9 - 5 = 4\).
  2. Gunakan rumus: \(U_{15} = a + (15 - 1)b = 5 + 14 \times 4\).
  3. Hitung: \(U_{15} = 5 + 56 = 61\). Suku ke-15 adalah 61.

Soal 2:

Hitunglah jumlah deret aritmetika 10 suku pertama dari barisan 3, 7, 11, 15, ...

Penyelesaian:

  1. Identifikasi: \(a = 3\), \(b = 4\), \(n = 10\). Hitung \(U_{10} = 3 + 9 \times 4 = 39\).
  2. Gunakan rumus jumlah deret: \(S_n = \frac{n}{2}(a + U_n) = \frac{10}{2}(3 + 39)\).
  3. Hitung: \(S_{10} = 5 \times 42 = 210\). Jumlah 10 suku pertama adalah 210.

Barisan Geometri: Barisan yang setiap sukunya diperoleh dengan mengalikan suku sebelumnya dengan rasio tetap \(r\). Contoh: 2, 6, 18, 54 (rasio \(r = 3\)). Rumus suku ke-\(n\): \(U_n = a \cdot r^{n-1}\).
