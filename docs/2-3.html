Bab 3: Pengukuran (Panjang, Berat, dan Waktu)

Metode Grounded Math — 6 Pilar Pembelajaran

1. Akar Konsep

Pengukuran adalah tindakan menemukan angka pada besaran dimensi fisik seperti panjang, berat, dan waktu lewat instrumen standar. Pengukuran ibarat pembacaan skala posisi jarum indikator berat pada alat ukur neraca pegas.

2. Pijakan Bumi

  • Penjahit Baju: Mengukur meter pita pada bidang gulungan kain untuk membentuk pola ukuran yang sesuai model potongan kemeja pria.
  • Pedagang Beras: Menganalisa bacaan jarum sebuah timbangan analog demi memastikan jumlah berat beras terjaga akurasinya.

3. Mekanika Dasar

Konversi standar baku metrik panjang linear yaitu mengubah rentang format meter menuju ekuivalen format sentimeternya:

\(1 \text{ m} = 100 \text{ cm}\)

Contoh langkah penjabaran ekuivalensi pipa besi sepanjang 2 meter:

\(2 \text{ m} = 2 \times 100 \text{ cm} = 200 \text{ cm}\)

4. Logika Komputasi

function ubahMeterKeCenti(ukuranMeter) {
  let koefisienPengali = 100;
  let hasilSentimeter = ukuranMeter * koefisienPengali;
  return hasilSentimeter;
}

6. Uji Ketangkasan

Soal: Dua utas tali diukur masing-masing. Tali pertama 1 meter dan tali kedua panjangnya 50 sentimeter. Apabila keduanya dikaitkan memanjang, berapa total ukuran sentimeter kedua tali tersebut?

Penyelesaian:

  1. Lakukan konversi terlebih dahulu pada panjang tali bagian pertama dari bentuk meter ke sentimeter (\(1 \times 100 = 100 \text{ cm}\)).
  2. Jumlahkan nilai parameter utamanya pada hitungan unit serupa, yaitu sentimeter ke rentang sentimeter sisanya.
  3. Tambahkan 100 sentimeter dari sambungan mula dengan 50 sentimeter hasil panjang tambahannya. Secara keseluruhan panjang talinya adalah 150 sentimeter angka absolut.
